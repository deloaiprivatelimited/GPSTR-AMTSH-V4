# ಅಧ್ಯಾಯ - 7
# ಪರಿಮಾಣಗಳ ಹೋಲಿಕೆ

## 7.1 ಶೇಕಡಾ ಕ್ರಮದಿಂದ ಪರಿಮಾಣಗಳ ಹೋಲಿಕೆಯ ಮತ್ತೊಂದು ಕ್ರಮ

--- FIGURE ---
Description: ಎರಡು ಹುಡುಗಿಯರ ವರದಿ ಕಾರ್ಡ್‌ಗಳನ್ನು ಹೋಲಿಸಲಾಗುತ್ತಿದೆ. ಅನಿತಳ ವರದಿ ಮತ್ತು ರೀಟಾಳ ವರದಿ.

ಶೇಕಡಾ : 80

ರೀಟಾಳ ವರದಿ

--- END FIGURE ---

ತನಗೆ 320 ಅಂಕಗಳು ಬಂದಿದ್ದು, ರೀಟಾಳಿಗೆ 300 ಅಂಕಗಳು ಬಂದಿರುವುದರಿಂದ ತನ್ನ ಫಲಿತಾಂಶವೇ ಹೆಚ್ಚು ಎಂದು ಅನಿತ ಹೇಳುತ್ತಾಳೆ. ಅವಳ ಮಾತನ್ನು ನೀವು ಒಪ್ಪುತ್ತೀರಾ? ಅವಳ ಫಲಿತಾಂಶವೇ ಹೆಚ್ಚು ಎಂದು ಭಾವಿಸುತ್ತೀರಾ?

ಅವರಿಬ್ಬರ ಪರೀಕ್ಷೆಯ ಗರಿಷ್ಠ ಅಂಕಗಳು ಬೇರೆ ಬೇರೆಯಾಗಿರುವುದರಿಂದ ಅವರು ಪಡೆದಿರುವ ಅಂಕಗಳ ಮೇಲೆ ಯಾರ ಫಲಿತಾಂಶ ಉತ್ತಮವಾಗಿದೆಯೆಂದು ತೀರ್ಮಾನಿಸಲು ಸಾಧ್ಯವಿಲ್ಲವೆಂದು ಮಾನಸಿಯು ಅವರಿಗೆ ಹೇಳಿದಳು.

ಪ್ರಗತಿ ಪತ್ರದಲ್ಲಿರುವ ಶೇಕಡಾವಾರು ಅಂಕಗಳನ್ನು ನೀವೇಕೆ ಗಮನಿಸಬಾರದು ಎಂದು ಹೇಳಿದಳು.

ಅನಿತಾಳ ಶೇಕಡಾವಾರು ಅಂಕಗಳು 80 ಹಾಗೂ ರೀಟಾಳ ಶೇಕಡಾವಾರು ಅಂಕಗಳು 83.3 ಆಗಿರುವುದರಿಂದ ರೀಟಾಳ ಸಾಧನೆಯೇ ಉತ್ತಮವಾಗಿದೆಯೆಂದು ಹೇಳಬಹುದು.

ನೀವು ಇದನ್ನು ಒಪ್ಪುತ್ತೀರಾ?

ಛೇದ 100 ಇರುವ ಭಿನ್ನರಾಶಿಯ ಅಂಶವೇ ಶೇಕಡಾ ಆಗಿದೆ. ಇದನ್ನು ಫಲಿತಾಂಶಗಳನ್ನು ಹೋಲಿಸಲು ಬಳಸಲಾಗುತ್ತದೆ. ಈಗ ಇದರ ಬಗ್ಗೆ ವಿವರವಾಗಿ ಅರ್ಥೈಸಿಕೊಳ್ಳೋಣ.

### 7.1.1 ಶೇಕಡಾವಾರು - ಅರ್ಥ

--- DEFINITION ---
Term: ಶೇಕಡಾ (percent)
Definition: ಶೇಕಡಾ (percent) ಇದು ಲ್ಯಾಟಿನ್ನಿನ per centum (ಪ್ರತಿಶತದಿಂದ) ಪಡೆದದ್ದಾಗಿದ್ದು ಅರ್ಥ 'ಪ್ರತಿ ನೂರಕ್ಕೆ' ಎಂದಾಗಿದೆ.
--- END DEFINITION ---

ಶೇಕಡಾವನ್ನು % ಚಿಹ್ನೆಯಿಂದ ಸೂಚಿಸಲಾಗುತ್ತದೆ. ಇದರ ಅರ್ಥ ನೂರನೆಯ ಎಂದೂ ಕೂಡ ಆಗಿದೆ. 1% ಎಂದರೆ ನೂರಕ್ಕೆ ಒಂದು ಅಥವಾ ನೂರನೆಯ ಒಂದು ಎಂದರ್ಥ.

ಅದನ್ನು ಹೀಗೆ ಬರೆಯಬಹುದು : $1\% = \frac{1}{100} = 0.01$

ಇದನ್ನು ವಿವರವಾಗಿ ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಮುಂದಿನ ಉದಾಹರಣೆಯನ್ನು ಗಮನಿಸಿ.

ರೀನಾಳು ಬೇರೆ ಬೇರೆ ಬಣ್ಣಗಳ 100 ಹಾಸುಗಳಿಂದ (ಟೈಲ್ಸ್) ಕೂಡಿದ ಒಂದು ಕೊಠಡಿಯನ್ನು ನೋಡಿದಳು. ಹಳದಿ, ಹಸಿರು, ಕೆಂಪು ಮತ್ತು ನೀಲಿ ನೆಲಹಾಸುಗಳನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ಎಣಿಸಿ ಮುಂದಿನ ಕೋಷ್ಠಕದ ಕೆಲವು ಭಾಗವನ್ನು ಭರ್ತಿ ಮಾಡಿದಳು. ಕೋಷ್ಠಕವನ್ನು ಭರ್ತಿ ಮಾಡಲು ಅವಳಿಗೆ ಸಹಾಯ ಮಾಡುವಿರಾ?

| :--- | :--- | :--- | :--- | :--- | :--- |

2. ಒಂದು ಪೂ ಅಂಗಡಿಯಲ್ಲಿರುವ ಬೇರೆ ಬೇರೆ ಅಳತೆಯ ಹೂಗಳ ಸಂಖ್ಯೆಯನ್ನು ಮುಂದೆ ನೀಡಲಾಗಿದೆ.

ಈ ದತ್ತಾಂಶಗಳಿಗೆ ಹಿಂದೆ ರಚಿಸಿರುವಂತೆ ಒಂದು ಕೋಷ್ಟಕವನ್ನು ರಚಿಸಿ. ಪ್ರತಿ ಅಳತೆಯ ಪೂಗಳ ಶೇಕಡಾವಾರು ಕಂಡು ಹಿಡಿಯಿರಿ.

### ಮೊತ್ತ 100 ಇಲ್ಲದಿದ್ದಾಗ ಶೇಕಡಾವಾರು

ಇದುವರೆಗಿನ ಎಲ್ಲಾ ಉದಾಹರಣೆಗಳಲ್ಲಿ ವಸ್ತುಗಳ ಒಟ್ಟು ಸಂಖ್ಯೆ 100 ಆಗಿತ್ತು ಉದಾಹರಣೆಗೆ ರೀನಾ ನೋಡಿದ ಟೈಲ್ಸ್ ಗಳು 100, ಮಕ್ಕಳ ಸಂಖ್ಯೆ 100 ಹಾಗೂ ಪೂ ಅಂಗಡಿಯಲ್ಲಿ 100 ಜೊತೆ ಹೂಗಳಿದ್ದವು. ವಸ್ತುಗಳ ಒಟ್ಟು ಸಂಖ್ಯೆ 100 ಕ್ಕಿಂತ ಕಡಿಮೆಯಿದ್ದಾಗ ಶೇಕಡಾವಾರು ಲೆಕ್ಕಾಚಾರ ಮಾಡುವುದು ಹೇಗೆ? ಇಂತಹ ಪ್ರಕರಣಗಳಲ್ಲಿ ಭಿನ್ನರಾಶಿಯನ್ನು ಛೇದ 100 ಆಗುವಂತೆ ಸಮಾನ ಭಿನ್ನರಾಶಿಗೆ ಪರಿವರ್ತಿಸಬೇಕು. ಮುಂದಿನ ಉದಾಹರಣೆಯನ್ನು ಗಮನಿಸಿ. ನಿಮ್ಮ ಬಳಿ ಇರುವ ಒಂದು ಸರದಲ್ಲಿ (necklace) ಎರಡು ಬಣ್ಣದ 20 ಮಣಿಗಳಿವೆ.

| ಬಣ್ಣ | ಮಣಿಗಳ ಸಂಖ್ಯೆ | ಭಿನ್ನರಾಶಿ | 100 ರ ಛೇದ | ಶೇಕಡವಾರು |
| :--- | :--- | :--- | :--- | :--- |
| ಕೆಂಪು | 8 | $\frac{8}{20}$ | $\frac{8}{20} \times \frac{100}{100} = \frac{40}{100}$ | 40% |
| ನೀಲಿ | 12 | $\frac{12}{20}$ | $\frac{12}{20} \times \frac{100}{100} = \frac{60}{100}$ | 60% |
| ಒಟ್ಟು | 20 | | | |

ಅನ್ವರ್ ಕೆಂಪು ಮಣಿಗಳ ಶೇಕಡಾವಾರು ಕಂಡುಹಿಡಿದ ರೀತಿ ಹೀಗಿದೆ. ಒಟ್ಟು 20 ಮಣಿಗಳಲ್ಲಿ ಕೆಂಪು ಮಣಿಗಳ ಸಂಖ್ಯೆ 8. ಹಾಗಾಗಿ, 100 ಕ್ಕೆ ಕೆಂಪು ಮಣಿಗಳ ಸಂಖ್ಯೆ
$\frac{8}{20} \times 100 = 40$ (ನೂರಕ್ಕೆ) = 40%

ಆಶಾ ಕಂಡುಹಿಡಿದ ರೀತಿ ಹೀಗಿದೆ.
$$ \frac{8}{20} = \frac{8 \times 5}{20 \times 5} = \frac{40}{100} = 40\% $$

ವಸ್ತುಗಳ ಒಟ್ಟು ಸಂಖ್ಯೆ 100 ಕ್ಕಿಂತ ಕಡಿಮೆಯಿದ್ದಾಗ ಈ ಮೂರು ವಿಧಾನಗಳ ಮೂಲಕ ಶೇಕಡಾವಾರು ಕಂಡು ಹಿಡಿಯಬಹುದು. ಮೇಲಿನ ಕೋಷ್ಟಕದಲ್ಲಿ ತೋರಿಸಿರುವಂತೆ ಭಿನ್ನರಾಶಿಯನ್ನು $\frac{100}{100}$ ರಿಂದ ಗುಣಿಸಬೇಕು. ಹಾಗಾಗಿ ಇದರಿಂದ ಭಿನ್ನರಾಶಿಯ ಬೆಲೆ ಬದಲಾಗುವುದಿಲ್ಲ 100 ಮಾತ್ರ ಛೇದದಲ್ಲಿ ಉಳಿಯುತ್ತದೆ.

ಅನ್ವರ್ ಏಕಮಾನ ಪದ್ಧತಿ ಉಪಯೋಗಿಸಿದ್ದಾನೆ. ಆಶಾ ಛೇದದಲ್ಲಿ 100ನ್ನು ಪಡೆಯಲು ಭಿನ್ನರಾಶಿಯನ್ನು $\frac{5}{5}$ ರಿಂದ ಗುಣಿಸಿದ್ದಾಳೆ. ನೀವು ನಿಮಗೆ ಸೂಕ್ತವಾದ ಯಾವುದಾದರೂ ವಿಧಾನವನ್ನು ಉಪಯೋಗಿಸಬಹುದು. ಅಥವಾ ನಿಮ್ಮದೇ ಯಾವುದಾದರೂ ವಿಧಾನ ಅನುಸರಿಸಬಹುದು.

ಅನ್ವರ್ ಉಪಯೋಗಿಸಿದ ವಿಧಾನವನ್ನು ಎಲ್ಲಾ ಅನುಪಾತಗಳಿಗೂ ಅನ್ವಯಿಸಬಹುದು. ಆಶಾ ಉಪಯೋಗಿಸಿದ ವಿಧಾನ ಎಲ್ಲಾ ಅನುಪಾತಗಳಿಗೂ ಅನ್ವಯವಾಗುತ್ತದೆಯೇ? ಛೇದವನ್ನು ಒಂದು ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಯಿಂದ ಗುಣಿಸಿ 100 ಬರುವಂತಿದ್ದಲ್ಲಿ ಮಾತ್ರ ಆಶಾಳ ವಿಧಾನವನ್ನು ಅನುಸರಿಸಬಹುದೆಂದು ಅನ್ವರ್ ಹೇಳುತ್ತಾನೆ. ಛೇದ 20 ಆಗಿದ್ದರಿಂದ ಅದಕ್ಕೆ 5ನ್ನು ಗುಣಿಸಿ 100 ಆಶಾ ಪಡೆದಿದ್ದಾಳೆ. ಛೇದ 6 ಆಗಿದ್ದರೆ ಅವಳು ಆ ವಿಧಾನವನ್ನು ಉಪಯೋಗಿಸಲಾಗುತ್ತಿರಲಿಲ್ಲ. ಇದನ್ನು ನೀವು ಒಪ್ಪುವಿರಾ?

**ಇವುಗಳನ್ನು ಪ್ರಯತ್ನಿಸಿ.**

1. ಬೇರೆ ಬೇರೆ ಬಣ್ಣದ 10 ಬಿಲ್ಲೆಗಳನ್ನು ಸಂಗ್ರಹಿಸಲಾಗಿದೆ

| ಬಣ್ಣ | ಸಂಖ್ಯೆ | ಭಿನ್ನರಾಶಿ | ನೂರರ ಛೇದ | ಶೇಕಡಾವಾರು |
| :--- | :--- | :--- | :--- | :--- |
| ಹಸಿರು | | | | |
| ನೀಲಿ | | | | |
| ಕೆಂಪು | | | | |
| ಒಟ್ಟು | | | | |

ಕೋಷ್ಟಕವನ್ನು ಭರ್ತಿಮಾಡಿ ಪ್ರತೀ ಬಣ್ಣದ ಬಿಲ್ಲೆಯ ಶೇಕಡಾ ಕಂಡು ಹಿಡಿಯಿರಿ.

2. ಮಾಲಾ ಬಳಿ 20 ಚಿನ್ನದ ಹಾಗೂ 10 ಬೆಳ್ಳಿಯ ಬಳೆಗಳ ಸಂಗ್ರಹವಿದೆ. ಪ್ರತಿ ವಿಧದ ಬಳೆಗಳ ಶೇಕಡಾವಾರು ಎಷ್ಟು? ಮೇಲಿನ ಉದಾಹರಣೆಯಲ್ಲಿರುವಂತೆ ಕೋಷ್ಟಕವನ್ನು ರಚಿಸಬಲ್ಲಿರಾ?

**ಆಲೋಚಿಸಿ, ಚರ್ಚಿಸಿ ಮತ್ತು ಬರೆಯಿರಿ.**

1. ಮುಂದೆ ನೀಡಲಾಗಿರುವ ಉದಾಹರಣೆಗಳನ್ನು ಗಮನಿಸಿ. ಪ್ರತಿಯೊಂದರಲ್ಲೂ ಹೋಲಿಕೆಗೆ ಸೂಕ್ತವಾದುದು ಯಾವುದೆಂದು ಚರ್ಚಿಸಿ.

### 7.1.2 ಭಿನ್ನರಾಶಿಯನ್ನು ಶೇಕಡಾ ಕ್ರಮಕ್ಕೆ ಬದಲಾಯಿಸುವುದು

ಭಿನ್ನರಾಶಿಯಲ್ಲಿ ಛೇದಗಳು ಬೇರೆ ಬೇರೆಯಾಗಿರಬಹುದು. ಭಿನ್ನರಾಶಿಗಳನ್ನು ಹೋಲಿಸಬೇಕಾದರೆ ನಮಗೆ ಸಾಮಾನ್ಯ ಛೇದದ ಅವಶ್ಯಕತೆಯಿದೆ ಹಾಗೂ ಛೇದ 100 ಆಗಿದ್ದಾಗ ಹೋಲಿಕೆಯು ಇನ್ನೂ ಅನುಕೂಲಕರವಾಗಿರುತ್ತದೆ ಎಂಬುದನ್ನು ನಾವು ಗಮನಿಸಿದ್ದೇವೆ. ಇಲ್ಲಿ ಭಿನ್ನರಾಶಿಯನ್ನು ಶೇಕಡಾ ಕ್ರಮಕ್ಕೆ ಪರಿವರ್ತಿಸುತ್ತಿದ್ದೇವೆ. ವಿವಿಧ ಭಿನ್ನರಾಶಿಗಳನ್ನು ಶೇಕಡಾ ಕ್ರಮಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದಕ್ಕೆ ಪ್ರಯತ್ನಿಸೋಣ.

--- EXAMPLE START ---
Source: Class 8, Example 1 (page 5)
Problem: $\frac{1}{3}$ ಇನ್ನು ಶೇಕಡಾಕ್ರಮದಲ್ಲಿ ಬರೆಯಿರಿ.
Solution:
$$ \frac{1}{3} = \frac{1}{3} \times 100\% = \frac{100}{3}\% = 33\frac{1}{3}\% $$
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 8, Example 2 (page 5)
Problem: ಒಂದು ತರಗತಿಯಲ್ಲಿರುವ 25 ವಿದ್ಯಾರ್ಥಿಗಳಲ್ಲಿ ಬಾಲಕಿಯರ ಸಂಖ್ಯೆ 15. ಬಾಲಕಿಯರ ಶೇಕಡಾ ಪ್ರಮಾಣ ಎಷ್ಟು?
Solution:
25 ವಿದ್ಯಾರ್ಥಿಗಳಲ್ಲಿ ಬಾಲಕಿಯರು 15 ಆದ್ದರಿಂದ, ಶೇಕಡಾ ಬಾಲಕಿಯರು
$$ = \frac{15}{25} \times 100 = 60 $$
ತರಗತಿಯಲ್ಲಿ 60% ಬಾಲಕಿಯರಿದ್ದಾರೆ.
--- EXAMPLE END ---

Solution:

(ii) ವಸ್ತುವೊಂದರ ಬೆಲೆ 50% ರಷ್ಟು ಹೆಚ್ಚಾಗಬಹುದೇ?
ವಸ್ತುವೊಂದರ ಬೆಲೆ 100% ರಷ್ಟು ಹೆಚ್ಚಾಗಬಹುದೇ?
ವಸ್ತುವೊಂದರ ಬೆಲೆ 150% ರಷ್ಟು ಹೆಚ್ಚಾಗಬಹುದೇ?

### 7.1.3 ದಶಮಾಂಶಗಳನ್ನು ಶೇಕಡ ರೂಪಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದು.

ಭಿನ್ನರಾಶಿಯನ್ನು ಶೇಕಡಾ ಕ್ರಮಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದನ್ನು ನಾವು ತಿಳಿದಿದ್ದೇವೆ. ಈಗ ದಶಮಾಂಶವನ್ನು ಶೇಕಡ ರೂಪಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದು ಹೇಗೆ ಎಂಬುದನ್ನು ತಿಳಿಯೋಣ.

--- EXAMPLE START ---
Source: Class 8, Example 4 (page 6)
Problem: ಮುಂದೆ ನೀಡಲಾಗಿರುವ ದಶಮಾಂಶಗಳನ್ನು ಶೇಕಡಾಗೆ ಪರಿವರ್ತಿಸಿ.
(a) 0.75
(b) 0.09
(c) 0.2
Solution:
(a) $0.75 = 0.75 \times 100\% = \frac{75}{100} \times 100\% = 75\%$
(b) $0.09 = \frac{9}{100} = 9\%$
(c) $0.2 = \frac{2}{10} = \frac{20}{100} = 20\%$
--- EXAMPLE END ---

### 7.1.4 ಶೇಕಡಾ ಬೆಲೆಯನ್ನು ಭಿನ್ನರಾಶಿಗೆ ಅಥವಾ ದಶಮಾಂಶಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದು

ಇದುವರೆಗೂ ನಾವು ಭಿನ್ನರಾಶಿಯನ್ನು ಹಾಗೂ ದಶಮಾಂಶವನ್ನು ಶೇಕಡಾಗೆ ಪರಿವರ್ತಿಸಿದ್ದೇವೆ. ಇದರ ವಿಲೋಮ ಕ್ರಿಯೆಯನ್ನೂ ಸಹ ನಾವು ಮಾಡಬಹುದು. ಅಂದರೆ ದತ್ತ ಶೇಕಡಾವನ್ನು ಭಿನ್ನರಾಶಿಗೆ ಹಾಗೂ ದಶಮಾಂಶಕ್ಕೆ ಪರಿವರ್ತಿಸಬಹುದು.

ಮುಂದಿನ ಕೋಷ್ಟಕವನ್ನು ಗಮನಿಸಿ, ಪೂರ್ಣಗೊಳಿಸಿ:

| ಶೇಕಡಾ | 1% | 10% | 25% | 50% | 90% | 125% | 250% |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| ಭಿನ್ನರಾಶಿ | $\frac{1}{100}$ | $\frac{10}{100} = \frac{1}{10}$ | | | | | |
| ದಶಮಾಂಶ | 0.01 | 0.10 | | | | | |

**ಭಾಗಾಂಶಗಳ ಮೊತ್ತ ಯಾವಾಗಲೂ ಪೂರ್ಣವನ್ನು ನೀಡುತ್ತದೆ**

ಬಣ್ಣದ ನೆಲಹಾಸುಗಳು (ಟೈಲ್ಸ್), ಮಕ್ಕಳ ಎತ್ತರಗಳು ಹಾಗೂ ಗಾಳಿಯಲ್ಲಿನ ಅನಿಲಗಳ ಉದಾಹರಣೆಗಳಲ್ಲಿನ ಶೇಕಡಾ ಬೆಲೆಗಳನ್ನು ಕೂಡಿದಾಗ ಮೊತ್ತ 100 ಆಗುತ್ತದೆ ಎಂಬುದನ್ನು ನೋಡಿದ್ದೇವೆ. ಪೂರ್ಣಾಂಕದಲ್ಲಿನ ಭಾಗಾಂಶಗಳನ್ನು ಕೂಡಿದಾಗ ಮೊತ್ತವು ಪೂರ್ಣಾಂಕ ಅಥವಾ 100% ಆಗುತ್ತದೆ. ಆದ್ದರಿಂದ ನಮಗೆ ಒಂದು ಭಾಗಾಂಶವನ್ನು ನೀಡಿದಾಗ ಇನ್ನೊಂದು ಭಾಗಾಂಶವನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು. ತರಗತಿಯ 30% ವಿದ್ಯಾರ್ಥಿಗಳು ಬಾಲಕಿಯರು, ಉಳಿದವರು ಬಾಲಕರು ಎಂದಿಟ್ಟುಕೊಳ್ಳಿ.

ಇದರ ಅರ್ಥ ತರಗತಿಯಲ್ಲಿರುವ 100 ವಿದ್ಯಾರ್ಥಿಗಳಲ್ಲಿ 30 ವಿದ್ಯಾರ್ಥಿಗಳು ಬಾಲಕಿಯರು, ಉಳಿದವರು ಬಾಲಕರು.

ಬಾಲಕರ ಪ್ರಮಾಣ ನಿಸ್ಸಂಶಯವಾಗಿ (100-30)% = 70%

### 7.1.5 ಅಂದಾಜಿಸುವಲ್ಲಿ ತಮಾಷೆ

ಒಂದು ಸ್ಥಳದ ಭಾಗಗಳನ್ನು ಅಂದಾಜಿಸಲು ಶೇಕಡಾ ನಮಗೆ ಸಹಾಯ ಮಾಡುತ್ತದೆ.

--- EXAMPLE START ---
Source: Class 8, Example 5 (page 8)
Problem: ಪಕ್ಕದ ಚಿತ್ರದಲ್ಲಿ ಶೇಕಡಾ ಎಷ್ಟು ಭಾಗ ಛಾಯೀಕೃತವಾಗಿದೆ?
Solution:
ಮೊದಲು ಚಿತ್ರದಲ್ಲಿ ಎಷ್ಟು ಭಾಗ ಛಾಯೀಕೃತವಾಗಿದೆ ಎಂದು ಕಂಡುಹಿಡಿದು ಅದರಿಂದ ಛಾಯೀಕೃತ ಭಾಗದ ಶೇಕಡಾ ಬೆಲೆ ಕಂಡುಹಿಡಿಯಬಹುದು.
ಚಿತ್ರದಲ್ಲಿ ಅರ್ಧ ಭಾಗ ಛಾಯೀಕೃತವಾಗಿರುವುದನ್ನು ಗಮನಿಸಬಹುದು.
$$ \frac{1}{2} = \frac{1}{2} \times 100 = 50\% $$
ಆದ್ದರಿಂದ ಚಿತ್ರದ 50% ಭಾಗ ಛಾಯೀಕೃತವಾಗಿದೆ.
--- EXAMPLE END ---

## 7.2 ಶೇಕಡಾ ಉಪಯೋಗಗಳು

### 7.2.1 ಅರ್ಥೈಸಿಕೊಳ್ಳುವುದು.

ಶೇಕಡಾ ಹೋಲಿಕೆಗಳಲ್ಲಿ ಹೇಗೆ ಉಪಯುಕ್ತವಾಗಿದೆ ಎಂಬುದನ್ನು ತಿಳಿದೆವು. ಭಿನ್ನರಾಶಿಗಳನ್ನು ಹಾಗೂ ದಶಮಾಂಶಗಳನ್ನು ಶೇಕಡಾಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದನ್ನೂ ನಾವು ಕಲಿತೆವು. ಈಗ ಶೇಕಡಾವನ್ನು ನಿತ್ಯಜೀವನದಲ್ಲಿ ಹೇಗೆ ಉಪಯೋಗಿಸಬಹುದೆಂಬುದನ್ನು ಕಲಿಯೋಣ. ಮೊದಲು ಮುಂದಿನ ಹೇಳಿಕೆಗಳನ್ನು ಅರ್ಥೈಸುತ್ತಾ ಪ್ರಾರಂಭಿಸೋಣ:

- ರವಿ ತನ್ನ ಆದಾಯದ 5%ರಷ್ಟು ಉಳಿಸಿದನು.
- ಮೀರಾಳ ಉಡುಪುಗಳ 20% ಭಾಗವು ನೀಲಿಯ ಬಣ್ಣದ್ದಾಗಿದೆ.
- ರೇಖಾ ತಾನು ಮಾರುವ ಪ್ರತೀ ಪುಸ್ತಕಕ್ಕೆ 10% ಪಡೆಯುತ್ತಾಳೆ.

5% ಎಂದರೆ 100 ರಲ್ಲಿ 5 ಭಾಗಗಳು ಅಥವಾ ಇದನ್ನು ಹೀಗೆ ಬರೆಯಬಹುದು $\frac{5}{100}$. ಇದರರ್ಥ ರವಿಯು ಗಳಿಸಿದ ಪ್ರತೀ ₹100 ರಲ್ಲಿ ₹5 ನ್ನು ಉಳಿಸುತ್ತಾನೆ. ಇದೇ ರೀತಿ ಮೇಲಿನ ಇನ್ನುಳಿದ ಹೇಳಿಕೆಗಳನ್ನು ಅರ್ಥೈಸಿ.

### 7.2.2 ಶೇಕಡಾವಾರನ್ನು - 'ಎಷ್ಟು' ಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದು

ಮುಂದಿನ ಉದಾಹರಣೆಗಳನ್ನು ಪರಿಗಣಿಸಿ.

--- EXAMPLE START ---
Source: Class 8, Example 6 (page 9)
Problem: ಒಂದು ಸಮೀಕ್ಷೆಯ ಪ್ರಕಾರ 40 ಮಕ್ಕಳಲ್ಲಿ 25% ಮಕ್ಕಳು ಫುಟ್‌ಬಾಲ್ ಆಡುವುದನ್ನು ಇಷ್ಟಪಡುತ್ತಾರೆ. ಫುಟ್ ಬಾಲ್ ಆಡುವುದನ್ನು ಇಷ್ಟಪಡುವ ಮಕ್ಕಳ ಸಂಖ್ಯೆ ಎಷ್ಟು ?
Solution:
ಇಲ್ಲಿ ಒಟ್ಟು ಮಕ್ಕಳ ಸಂಖ್ಯೆ 40. ಇದರಲ್ಲಿ 25% ಮಕ್ಕಳು ಫುಟ್‌ಬಾಲ್ ಆಡುವುದನ್ನು ಇಷ್ಟಪಡುತ್ತಾರೆ. ಮೀನಾ ಮತ್ತು ಅರುಣ್ ಮಕ್ಕಳ ಸಂಖ್ಯೆಯನ್ನು ಕಂಡು ಹಿಡಿಯಲು ಮುಂದಿನ ವಿಧಾನಗಳನ್ನು ಅನುಸರಿಸುತ್ತಾರೆ. ನೀವು ಯಾವುದಾದರೂ ಒಂದು ವಿಧಾನವನ್ನು ಆಯ್ಕೆ ಮಾಡಬಹುದು.

**ಅರುಣ್ ಅನುಸರಿಸಿದ ವಿಧಾನ**
100 ಮಕ್ಕಳಲ್ಲಿ 25 ಮಕ್ಕಳು ಫುಟಬಾಲ್ ಇಷ್ಟಪಡುತ್ತಾರೆ.
ಆದ್ದರಿಂದ 40 ಮಕ್ಕಳಲ್ಲಿ ಫುಟ್‌ಬಾಲ್ ಇಷ್ಟಪಡುವವರ ಸಂಖ್ಯೆ
$$ \frac{25}{100} \times 40 = 10 $$

**ಮೀನಾ ಅನುಸರಿಸಿದ ವಿಧಾನ**
$$ 40 \text{ ರ } 25\% = \frac{25}{100} \times 40 = 10 $$
ಆದ್ದರಿಂದ 40 ಮಕ್ಕಳಲ್ಲಿ 10 ಮಕ್ಕಳು ಫುಟ್‌ಬಾಲ್ ಇಷ್ಟಪಡುತ್ತಾರೆ.
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 8, Example 7 (page 9)
Problem: ರಾಹುಲ್ ಖರೀದಿಸಿದ ಸೈಟರ್ ಮೇಲೆ 25% ರಿಯಾಯಿತಿ ನೀಡಿದ್ದರಿಂದ ರಾಹುಲ್‌ಗೆ ₹ 200 ಉಳಿಯಿತು. ರಿಯಾಯಿತಿ ನೀಡುವ ಮೊದಲು ಸೈಟರ್‌ನ ಬೆಲೆ ಎಷ್ಟು?
Solution:
ಸೈಟರ್ ಬೆಲೆಯ ಮೇಲೆ 25% ರಿಯಾಯಿತಿ ನೀಡಿದ್ದರಿಂದ ರಾಹುಲ್‌ಗೆ ₹ 200 ಉಳಿಯಿತು. ಅಂದರೆ ಸೈಟರ್‌ನ ಮೂಲ ಬೆಲೆಯ 25% ರಾಹುಲ್‌ಗೆ ಉಳಿದ ಹಣವಾಗಿದೆ. ಮೋಹನ ಮತ್ತು ಅಬ್ದುಲ್ ಸೈಟರ್‌ನ ಮೂಲ ಬೆಲೆಯನ್ನು ಹೇಗೆ ಕಂಡು ಹಿಡಿದರು ಎಂಬುದನ್ನು ನೋಡೋಣ.

**ಮೋಹನ್‌ ಪರಿಹಾರ**
ಮೂಲ ಬೆಲೆಯ 25% = ₹ 200
ಮೂಲ ಬೆಲೆಯು 'p' (₹ ಗಳಲ್ಲಿ) ಆಗಿರಲಿ.
ಆದ್ದರಿಂದ p ಯ 25% = 200
ಅಥವಾ $\frac{25}{100} \times p = 200$
ಅಥವಾ $\frac{1}{4} p = 200$ ಅಥವಾ $p = 200 \times 4$
ಆದ್ದರಿಂದ, $p = 800$.

**ಅಬ್ದುಲ್‌ನ ಪರಿಹಾರ**
ಪ್ರತಿ ₹ 100 ಕ್ಕೆ ₹ 25 ಉಳಿತಾಯವಾಗಿದೆ.
₹ 200 ಉಳಿತಾಯವಾದರೆ
$= \frac{100}{25} \times 200 = ₹ 800$
ಆದ್ದರಿಂದ ಸೈಟರಿನ ಮೂಲಬೆಲೆ ₹ 800 ಎಂದು ಇಬ್ಬರೂ ಕಂಡು ಹಿಡಿದರು.
--- EXAMPLE END ---

### 7.2.3 ಅನುಪಾತವನ್ನು ಶೇಕಡಾಗೆ ಪರಿವರ್ತಿಸುವುದು

ಕೆಲವೊಮ್ಮೆ ಭಾಗಗಳನ್ನು ಅನುಪಾತಗಳ ರೂಪದಲ್ಲಿ ನೀಡಲಾಗಿರುತ್ತದೆ ಮತ್ತು ಅವುಗಳನ್ನು ಶೇಕಡಾಗೆ ಪರಿವರ್ತಿಸಬೇಕಾಗುತ್ತದೆ. ಮುಂದಿನ ಉದಾಹರಣೆಯನ್ನು ಪರಿಗಣಿಸಿ.

--- EXAMPLE START ---
Source: Class 8, Example 8 (page 11)
Problem: ರೀನಾಳ ತಾಯಿ ಇಡ್ಲಿ ತಯಾರಿಸಲು ಎರಡು ಭಾಗ ಅಕ್ಕಿ ಹಾಗೂ ಒಂದು ಭಾಗ ಉದ್ದಿನಬೇಳೆ ಬೇಕಾಗುತ್ತದೆ ಎಂದು ಅವಳಿಗೆ ಹೇಳಿದರು. ಇಡ್ಲಿಯ ಮಿಶ್ರಣದಲ್ಲಿ ಶೇಕಡಾ ಎಷ್ಟು ಅಕ್ಕಿ ಮತ್ತು ಶೇಕಡಾ ಎಷ್ಟು ಉದ್ದಿನಬೇಳೆಯಿದೆ?
Solution:
ಅನುಪಾತದಲ್ಲಿ ಇದನ್ನು ಹೀಗೆ ಬರೆಯುತ್ತೇವೆ. ಅಕ್ಕಿ : ಉದ್ದಿನಬೇಳೆ = 2 : 1
ಈಗ, ಎಲ್ಲಾ ಭಾಗಗಳ ಒಟ್ಟು $2+1 = 3$ ಎಂದರೆ $\frac{2}{3}$ ಭಾಗ ಅಕ್ಕಿ, ಹಾಗೂ $\frac{1}{3}$ ಭಾಗ ಉದ್ದಿನಬೇಳೆ.
ಅಕ್ಕಿಯ ಶೇಕಡಾ ಪ್ರಮಾಣ, $\frac{2}{3} \times 100\% = \frac{200}{3} = 66\frac{2}{3}\%$
ಉದ್ದಿನ ಬೇಳೆಯ ಶೇಕಡಾ ಪ್ರಮಾಣ, $\frac{1}{3} \times 100\% = \frac{100}{3} = 33\frac{1}{3}\%$
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 8, Example 9 (page 12)
Problem: ₹ 250ನ್ನು ರವಿ, ರಾಜು ಮತ್ತು ರಾಯ್ ಇವರುಗಳಿಗೆ - ರವಿಗೆ 2 ಭಾಗ, ರಾಜುಗೆ 3 ಭಾಗ ಮತ್ತು ರಾಯ್‌ಗೆ 5 ಭಾಗ ಬರುವಂತೆ ಹಂಚಬೇಕಾಗಿದೆ. ಪ್ರತಿಯೊಬ್ಬರು ಪಡೆಯುವ ಹಣ ಎಷ್ಟು? ಶೇಕಡಾ ಕ್ರಮದಲ್ಲಿ ಎಷ್ಟಾಗುತ್ತದೆ?
Solution:
ಮೂರು ಬಾಲಕರಿಗೆ ದೊರೆಯುವ ಹಣದ ಭಾಗವನ್ನು ಅನುಪಾತದಲ್ಲಿ 2:3:5 ಎಂದು ಬರೆಯಬಹುದು. ಒಟ್ಟು ಭಾಗಗಳು $2 + 3 + 5 = 10$

**ಪ್ರತಿಯೊಬ್ಬರೂ ಪಡೆಯುವ ಹಣ**
ರವಿ: $\frac{2}{10} \times 250 = ₹ 50$
ರಾಜು: $\frac{3}{10} \times 250 = ₹ 75$
ರಾಯ್: $\frac{5}{10} \times 250 = ₹ 125$

**ಪ್ರತಿಯೊಬ್ಬರ ಹಣದ ಶೇಕಡಾ**
ರವಿ: $\frac{2}{10} \times 100\% = 20\%$
ರಾಜು: $\frac{3}{10} \times 100\% = 30\%$
ರಾಯ್: $\frac{5}{10} \times 100\% = 50\%$

### 7.2.4 ಪ್ರತಿ ಶತಕ್ಕೆ ಏರಿಕೆ ಅಥವಾ ಇಳಿಕೆ

ಒಂದು ನಿರ್ದಿಷ್ಟ ಪರಿಮಾಣದಲ್ಲಿ ಶೇಕಡಾ ಎಷ್ಟು ಹೆಚ್ಚಾಗಿದೆ ಅಥವಾ ಕಡಿಮೆಯಾಗಿದೆ ಎಂದು ತಿಳಿಯಬೇಕಾದ ಸಂದರ್ಭಗಳು ಇರುತ್ತವೆ. ಉದಾಹರಣೆಗೆ ಒಂದು ರಾಜ್ಯದ ಜನಸಂಖ್ಯೆಯು 5,50,000 ರಿಂದ 6,05,000 ಕ್ಕೆ ಏರಿಕೆಯಾಗಿದ್ದರೆ, ಏರಿಕೆಯಾದ ಜನಸಂಖ್ಯೆಯ ಪರಿಮಾಣ 10% ಎಂದರೆ ಸುಲಭವಾಗಿ ಅರ್ಥವಾಗುತ್ತದೆ.

ಪ್ರಾರಂಭಿಕ ಮೊತ್ತಕ್ಕೆ ಹೆಚ್ಚಾದ ಅಥವಾ ಕಡಿಮೆಯಾದ ಪರಿಮಾಣವನ್ನು ಶೇಕಡಾಗೆ ಹೇಗೆ ಪರಿವರ್ತಿಸಬಹುದು? ಮುಂದಿನ ಉದಾಹರಣೆಯನ್ನು ಪರಿಗಣಿಸಿ.

--- EXAMPLE START ---
Source: Class 8, Example 10 (page 12)
Problem: ಒಂದು ಶಾಲಾ ತಂಡವು ಪ್ರಸ್ತುತ ವರ್ಷ 6 ಪಂದ್ಯಗಳನ್ನು ಗೆದ್ದಿದೆ. ಕಳೆದ ವರ್ಷ 4 ಪಂದ್ಯಗಳನ್ನು ಗೆದ್ದಿತ್ತು. ಏರಿಕೆಯ ಪ್ರಮಾಣ ಎಷ್ಟು?
Solution:
ಏರಿಕೆಯಾದ ಗೆಲುವುಗಳ ಸಂಖ್ಯೆ = $6 - 4 = 2$
ಶೇಕಡಾ ಏರಿಕೆ = $\frac{\text{ಒಟ್ಟು ಬದಲಾವಣೆ}}{\text{ಮೂಲ ಮೊತ್ತ}} \times 100$
= $\frac{\text{ಏರಿಕೆಯಾದ ಗೆಲುವುಗಳ ಸಂಖ್ಯೆ}}{\text{ಒಟ್ಟು ಮೂಲ ಗೆಲವುಗಳು}} \times 100 = \frac{2}{4} \times 100 = 50$
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 8, Example 11 (page 13)
Problem: ಒಂದು ದೇಶದಲ್ಲಿ ಅನಕ್ಷರಸ್ಥರ ಸಂಖ್ಯೆ 10 ವರ್ಷಗಳಲ್ಲಿ 150 ಲಕ್ಷದಿಂದ 100 ಲಕ್ಷಕ್ಕೆ ಇಳಿಯಿತು. ಶೇಕಡಾ ಇಳಿಕೆ ಎಷ್ಟು?

ಮೂಲ ಮೊತ್ತ = ಆರಂಭಿಕ ಅನಕ್ಷರಸ್ಥರ ಸಂಖ್ಯೆ = 150 ಲಕ್ಷಗಳು.
ಒಟ್ಟು ಬದಲಾವಣೆ = ಅನಕ್ಷರಸ್ಥರ ಸಂಖ್ಯೆಯಲ್ಲಿನ ಇಳಿಕೆ = $150 - 100 = 50$ ಲಕ್ಷಗಳು.
ಆದ್ದರಿಂದ, ಶೇಕಡಾ ಇಳಿಕೆ
= $\frac{\text{ಒಟ್ಟು ಬದಲಾವಣೆ}}{\text{ಮೂಲ ಮೊತ್ತ}} \times 100 = \frac{50}{150} \times 100 = 33\frac{1}{3}$

## 7.3. ಒಂದು ವಸ್ತುವಿಗೆ ಸಂಬಂಧಿಸಿದ ಬೆಲೆಗಳು ಅಥವಾ ಕೊಂಡುಕೊಳ್ಳುವುದು ಮತ್ತು ಮಾರುವುದು

ಒಂದು ವಸ್ತುವನ್ನು ಖರೀದಿಸಿದ ಬೆಲೆಯನ್ನು ಕೊಂಡ ಬೆಲೆ ಎನ್ನುತ್ತೇವೆ. ಅದನ್ನು ಸಂಕ್ಷಿಪ್ತವಾಗಿ CP ಎಂದು ಬರೆಯಲಾಗುತ್ತದೆ.
ಒಂದು ವಸ್ತುವನ್ನು ನೀವು ಮಾರುವ ಬೆಲೆಗೆ ಮಾರಾಟ ಬೆಲೆ ಎನ್ನುತ್ತೇವೆ. ಅಥವಾ ಸಂಕ್ಷಿಪ್ತವಾಗಿ SP ಎಂದು ಬರೆಯಲಾಗುತ್ತದೆ.
ಒಂದು ವಸ್ತುವನ್ನು ಕೊಂಡ ಬೆಲೆಗಿಂತ ಕಡಿಮೆ ಬೆಲೆಗೆ, ಅದೇ ಬೆಲೆಗೆ ಅಥವಾ ಹೆಚ್ಚಿನ ಬೆಲೆಗೆ ನೀವು ಮಾರಬಹುದು. ಇವುಗಳಲ್ಲಿ ಯಾವುದು ಉತ್ತಮ? ಎಂದು ನೀವು ಹೇಳುವಿರಿ. ಮಾರಾಟವು ಲಾಭದಾಯಕವಾಗಿದೆಯೇ ಅಥವಾ ಇಲ್ಲವೇ ಎಂಬುದನ್ನು ಕೊಂಡ ಬೆಲೆ ಮತ್ತು ಮಾರಿದ ಬೆಲೆಯ ಮೇಲೆ ನಿರ್ಧರಿಸಬಹುದು. ಕೊಂಡ ಬೆಲೆಯು ಮಾರಿದ ಬೆಲೆಗಿಂತ ಕಡಿಮೆಯಿದ್ದರೆ, ನೀವು ಲಾಭ ಗಳಿಸುತ್ತೀರಿ.
ಲಾಭ = ಮಾರಿದ ಬೆಲೆ - ಕೊಂಡ ಬೆಲೆ
ಕೊಂಡಬೆಲೆ = ಮಾರಿದ ಬೆಲೆ ಆಗಿದ್ದರೆ, ಲಾಭವೂ ಇಲ್ಲದ ನಷ್ಟವೂ ಇಲ್ಲದ ಸನ್ನಿವೇಶ.
ಕೊಂಡ ಬೆಲೆಯು ಮಾರಿದ ಬೆಲೆಗಿಂತ ಹೆಚ್ಚಿದ್ದರೆ ನಿಮಗೆ ನಷ್ಟವಾಗುತ್ತದೆ. ನಷ್ಟ = ಕೊಂಡ ಬೆಲೆ - ಮಾರಿದ ಬೆಲೆ.

ವಸ್ತುಗಳಿಗೆ ಸಂಬಂಧಿಸಿದ ಬೆಲೆಗಳಿಂದ ಹೇಳಿಕೆಗಳನ್ನು ಅರ್ಥ ಮಾಡಿಕೊಳ್ಳಲು ಪ್ರಯತ್ನಿಸೋಣ.
*   ಒಂದು ಆಟಿಕೆಯನ್ನು ₹ 72 ಕ್ಕೆ ಕೊಂಡು ₹ 80ಕ್ಕೆ ಮಾರಲಾಯಿತು.
*   ಒಂದು T - ಷರ್ಟ್‌ನ್ನು ₹ 120 ಕ್ಕೆ ಕೊಂಡು ₹ 100ಕ್ಕೆ ಮಾರಲಾಯಿತು.
*   ಒಂದು ಸೈಕಲ್‌ನ್ನು ₹800 ಕ್ಕೆ ಕೊಂಡು ₹ 940ಕ್ಕೆ ಮಾರಲಾಯಿತು

ಮೊದಲ ಹೇಳಿಕೆಯನ್ನು ಪರಿಗಣಿಸೋಣ.
ವಸ್ತುವಿನ ಖರೀದಿಸಿದ ಬೆಲೆಯು (ಕೊಂಡ ಬೆಲೆ) ₹ 72 ಮತ್ತು ಮಾರಿದ ಬೆಲೆಯು ₹ 80. ಅಂದರೆ ಮಾರಿದ ಬೆಲೆಯು ಕೊಂಡ ಬೆಲೆಗಿಂತ ಹೆಚ್ಚು. ಹಾಗಾಗಿ ಗಳಿಸಿದ ಲಾಭ = ಮಾರಿದ ಬೆಲೆ-ಕೊಂಡ ಬೆಲೆ = ₹ 80 - ₹ 72 = ₹ 8.
ಇನ್ನುಳಿದ ಎರಡು ಹೇಳಿಕೆಗಳನ್ನು ಇದೇ ರೀತಿ ಅರ್ಥೈಸಲು ಪ್ರಯತ್ನಿಸೋಣ.

### 7.3.1 ಶೇಕಡಾ ಲಾಭ ಅಥವಾ ಶೇಕಡಾ ನಷ್ಟ

ಲಾಭ ಮತ್ತು ನಷ್ಟವನ್ನು ಶೇಕಡಾಗೆ (ಪ್ರತಿಶತಕ್ಕೆ) ಪರಿವರ್ತಿಸಬಹುದು. ಇದನ್ನು ಯಾವಾಗಲೂ ಕೊಂಡ ಬೆಲೆಯ ಮೇಲೆ ಲೆಕ್ಕಾಚಾರ ಮಾಡಲಾಗುತ್ತದೆ. ಮೇಲಿನ ಉದಾಹರಣೆಗಳಿಗೆ ಶೇಕಡಾ ಲಾಭ ಅಥವಾ ಶೇಕಡಾ ನಷ್ಟ ಕಂಡು ಹಿಡಿಯಬಹುದು.

ಆಟಿಕೆಗೆ ಸಂಬಂಧಿಸಿದ ಉದಾಹರಣೆಯನ್ನು ಪರಿಗಣಿಸೋಣ. ಇಲ್ಲಿ ಕೊಂಡ ಬೆಲೆ = ₹ 72, ಮಾರಿದ ಬೆಲೆ = ₹ 80, ಲಾಭ = ₹ 8. ಶೇಕಡಾ ಲಾಭವನ್ನು ಕಂಡು ಹಿಡಿಯಲು ನೇಹಾ ಮತ್ತು ಶೇಖರ್ ಅನುಸರಿಸಿದ ವಿಧಾನಗಳು ಮುಂದಿನಂತಿವೆ :

**ನೇಹಾ ಅನುಸರಿಸಿದ ವಿಧಾನ**
ಶೇಕಡಾ ಲಾಭ = $\frac{\text{ಲಾಭ}}{\text{ಕೊಂಡಬೆಲೆ}} \times 100$
= $\frac{8}{72} \times 100$
= $\frac{1}{9} \times 100 = 11\frac{1}{9}$

**ಶೇಖರ್ ಅನುಸರಿಸಿದ ವಿಧಾನ**
₹ 72 ಕ್ಕೆ ಲಾಭ ₹ 8 ಆದರೆ.
₹ 100 ಕ್ಕೆ ಲಾಭ, ಲಾಭ = $\frac{8}{72} \times 100$
= $11\frac{1}{9}$ ಆದ್ದರಿಂದ ಶೇಕಡಾ ಲಾಭ = $11\frac{1}{9}$

ಆದ್ದರಿಂದ ಲಾಭ ₹ 8 ಮತ್ತು ಶೇಕಡಾ ಲಾಭ $11\frac{1}{9}$

--- NOTE ---
ಲಾಭವಾಗಲಿ ಅಥವಾ ನಷ್ಟವಾಗಲಿ ಶೇಕಡಾವು ಯಾವಾಗಲೂ ಕೊಂಡು ಬೆಲೆಗೆ ಲೆಕ್ಕಾಚಾರ ಮಾಡಲಾಗುತ್ತದೆ.
--- END NOTE ---

ಇದೇ ರೀತಿ ಎರಡನೇ ಪ್ರಕರಣದಲ್ಲಿ ಶೇಕಡಾ ನಷ್ಟವನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು. ಇಲ್ಲಿ
ಕೊಂಡ ಬೆಲೆ = ₹ 120, ಮಾರಿದ ಬೆಲೆ = ₹ 100.
ಆದ್ದರಿಂದ ನಷ್ಟ = ₹ 120 - ₹ 100 = ₹ 20

ಶೇಕಡಾ ನಷ್ಟ = $\frac{\text{ನಷ್ಟ}}{\text{ಕೊಂಡಬೆಲೆ}} \times 100$
= $\frac{20}{120} \times 100$
= $\frac{50}{3} = 16\frac{2}{3}$

₹ 120 ಕ್ಕೆ ₹ 20 ನಷ್ಟ ಆದರೆ,
₹ 100 ಕ್ಕೆ ನಷ್ಟ = $\frac{20}{120} \times 100 = \frac{50}{3} = 16\frac{2}{3}$
ಆದ್ದರಿಂದ, ಶೇಕಡಾ ನಷ್ಟ $16\frac{2}{3}$

ಅಂತಿಮ ಪ್ರಕರಣವನ್ನು ಪ್ರಯತ್ನಿಸಿ.

ಒಂದು ವಸ್ತುವಿಗೆ ಸಂಬಂಧಿಸಿದ ಕೊಂಡ ಬೆಲೆ, ಮಾರಿದ ಬೆಲೆ, ಲಾಭ ಅಥವಾ ನಷ್ಟ ಅಥವಾ ಶೇಕಡಾ ಲಾಭ ಅಥವಾ ನಷ್ಟದ ಮೂರು ಪರಿಮಾಣಗಳಲ್ಲಿ ಯಾವುದಾದರೂ ಎರಡು ಪರಿಮಾಣಗಳನ್ನು ನೀಡಿದಾಗ ಇನ್ನೊಂದು ಪರಿಮಾಣವನ್ನು ಕಂಡು ಹಿಡಿಯಬಹುದು.

--- EXAMPLE START ---
Source: Class 8, Example 12 (page 15)
Problem: ಒಂದು ಹೂದಾನಿಯ ಕೊಂಡ ಬೆಲೆ ₹120. ಅಂಗಡಿಯವನು ಅದನ್ನು 10% ನಷ್ಟಕ್ಕೆ ಮಾರಿದರೆ, ಮಾರಿದ ಬೆಲೆಯನ್ನು ಕಂಡು ಹಿಡಿಯಿರಿ.
Solution:
ಕೊಂಡ ಬೆಲೆ = ₹ 120 ಹಾಗೂ ಶೇಕಡಾ ನಷ್ಟ = 10 ಎಂದು ನೀಡಲಾಗಿದ್ದು ನಾವು ಮಾರಿದ ಬೆಲೆಯನ್ನು ಕಂಡು ಹಿಡಿಯಬೇಕಾಗಿದೆ.

**ಮೋಹನ್ ಈ ರೀತಿ ಮಾಡುತ್ತಾನೆ.**
10%. ನಷ್ಟವೆಂದರೆ ಕೊಂಡ ಬೆಲೆ ₹ 100 ಇದ್ದಾಗ ನಷ್ಟ = ₹ 10
ಆದ್ದರಿಂದ ಮಾರಿದ ಬೆಲೆ ₹ (100-10) = ₹ 90
ಕೊಂಡ ಬೆಲೆ ₹100 ಇದ್ದಾಗ ಮಾರಿದ ಬೆಲೆ ₹90.
ಆದ್ದರಿಂದ ಕೊಂಡ ಬೆಲೆ ₹ 120 ಇದ್ದಾಗ,
ಮಾರಿದ ಬೆಲೆಯು = $\frac{90}{100} \times 120 = ₹ 108$

**ಆನಂದಿ ಈ ರೀತಿ ಮಾಡುತ್ತಾಳೆ.**
ಕೊಂಡ ಬೆಲೆಯಲ್ಲಿ 10% ನಷ್ಟವಾಗಿದೆ.
= 120 ರ 10%
= $\frac{10}{100} \times 120 = ₹ 12$
ಆದ್ದರಿಂದ, ಮಾರಿದ ಬೆಲೆ = ಕೊಂಡ ಬೆಲೆ - ನಷ್ಟ
= ₹ 120 - ₹ 12 = ₹ 108.
ಹೀಗೆ ಎರಡೂ ವಿಧಾನಗಳಲ್ಲಿ ಮಾರಿದ ಬೆಲೆಯು ₹ 108 ಆಗಿದೆ.
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 8, Example 13 (page 16)
Problem: ಆಟಿಕೆ ಕಾರಿನ ಮಾರಾಟ ಬೆಲೆ ₹ 540. ಅಂಗಡಿಯವನು 20% ರಷ್ಟು ಲಾಭಗಳಿಸಿದರೆ, ಆಟಿಕೆಯ ಕೊಂಡ ಬೆಲೆ ಎಷ್ಟು ?
Solution:
ಮಾರಿದ ಬೆಲೆ = ₹ 540 ಹಾಗೂ ಲಾಭ = 20% ಆಗಿದ್ದು, ನಾವು ಕೊಂಡ ಬೆಲೆ ಕಂಡು ಹಿಡಿಯಬೇಕಾಗಿದೆ.

**ಅಮೀನ ಅನುಸರಿಸಿದ ವಿಧಾನ**
20% ಲಾಭವೆಂದರೆ ಕೊಂಡ ಬೆಲೆ ₹ 100 ಇದ್ದಾಗ ಲಾಭ ₹ 20
ಆದ್ದರಿಂದ, ಮಾರಿದ ಬೆಲೆ = 100+20 = 120
ಯಾವಾಗ ಮಾರಿದ ಬೆಲೆಯು ₹120 ಆಗಿರುತ್ತದೆಯೋ, ಕೊಂಡ ಬೆಲೆಯು ₹ 100 ಆಗಿರುತ್ತದೆ.
ಆದ್ದರಿಂದ, ಮಾರಿದ ಬೆಲೆಯು ₹ 540 ಆಗಿದ್ದಾಗ ಕೊಂಡ ಬೆಲೆಯು
= $\frac{100}{120} \times 540 = ₹ 450$

**ಅರುಣ್‌ ಅನುಸರಿಸಿದ ವಿಧಾನ**
ಲಾಭ = ಕೊಂಡ ಬೆಲೆಯ 20% ಮತ್ತು
ಮಾರಿದ ಬೆಲೆ = ಕೊಂಡಬೆಲೆ+ಲಾಭ
ಆದ್ದರಿಂದ,
540 = ಕೊಂಡಬೆಲೆ + ಕೊಂಡ ಬೆಲೆಯ 20%
= ಕೊಂಡಬೆಲೆ + $\frac{20}{100} \times$ ಕೊಂಡಬೆಲೆ
= $(1 + \frac{1}{5})$ ಕೊಂಡಬೆಲೆ
= $\frac{6}{5} \times$ ಕೊಂಡಬೆಲೆ
ಆದ್ದರಿಂದ, ಕೊಂಡಬೆಲೆ = $540 \times \frac{5}{6}$
ಕೊಂಡ ಬೆಲೆ = ₹ 450
ಹೀಗೆ, ಎರಡೂ ವಿಧಾನಗಳಲ್ಲಿ ಕೊಂಡ ಬೆಲೆಯು ₹ 450 ಆಗಿದೆ.
--- EXAMPLE END ---

## 7.4 ತೆಗೆದುಕೊಂಡ ಹಣದ ಮೇಲಿನ ಶುಲ್ಕ ಅಥವಾ ಸರಳ ಬಡ್ಡಿ

ಸೋಹಿನಿ ಒಂದು ಹೊಸ ಸ್ಕೂಟರನ್ನು ಕೊಂಡುಕೊಳ್ಳುವುದಾಗಿ ಹೇಳಿದಳು. ಮೋಹನ್ ಸ್ಕೂಟರನ್ನು ಕೊಳ್ಳಲು ಹಣವಿದೆಯೇ ಎಂದು ಕೇಳಿದನು. ಆಗ ಸೋಹಿನಿ ನಮ್ಮ ತಂದೆ ಬ್ಯಾಂಕ್‌ನಿಂದ ಸಾಲ ಪಡೆಯುವುದಾಗಿ ಹೇಳಿದಳು. ಹೀಗೆ ತೆಗೆದುಕೊಂಡ ಹಣವನ್ನು ತೆಗೆದುಕೊಂಡ ಮೊತ್ತ ಅಥವಾ 'ಅಸಲು' ಎನ್ನುತ್ತೇವೆ.

ಹೀಗೆ ತೆಗೆದುಕೊಂಡ ಹಣವನ್ನು ಸ್ವಲ್ಪ ಸಮಯದವರೆಗೆ ಬ್ಯಾಂಕಿಗೆ ಹಿಂದಿರುಗಿಸುವವರೆಗೆ ಉಪಯೋಗಿಸಬಹುದು. ಹೀಗೆ ಸ್ವಲ್ಪ ಕಾಲ ಇಟ್ಟು ಕೊಂಡ ಹಣವನ್ನು ಬ್ಯಾಂಕಿಗೆ ಹಿಂದಿರುಗಿಸುವಾಗ ಸ್ವಲ್ಪ ಹೆಚ್ಚಿನ ಹಣವನ್ನು ನೀಡಬೇಕಾಗುತ್ತದೆ. ಹೆಚ್ಚಾಗಿ ಕೊಟ್ಟ ಹಣವನ್ನು 'ಬಡ್ಡಿ' ಎನ್ನುತ್ತೇವೆ.

ಸಾಲ ಪಡೆದ ಹಣ ಮತ್ತು ಬಡ್ಡಿಯನ್ನು ಸೇರಿಸಿ ವರ್ಷದ ಕೊನೆಯಲ್ಲಿ ನೀವು ಪಾವತಿಸಬೇಕಾಗಿರುವ ಮೊತ್ತವನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು.
ಮೊತ್ತ = ಅಸಲು + ಬಡ್ಡಿ

ಬಡ್ಡಿಯನ್ನು ಸಾಮಾನ್ಯವಾಗಿ ಒಂದು ವರ್ಷದ ಅವಧಿಗೆ ಪ್ರತಿಶತಕ್ಕೆ ಲೆಕ್ಕ ಹಾಕಲಾಗುತ್ತದೆ. ಉದಾಹರಣೆಗೆ ಬಡ್ಡಿ 10% ಇದ್ದರೆ, ಅದನ್ನು ಸಾಲಿಯಾನ 10% ಅಥವಾ ವಾರ್ಷಿಕ 10% ಎಂದು ಬರೆಯುತ್ತೇವೆ.

ಸಾಲಿಯಾನ 10% ಎಂದರೆ ತೆಗೆದುಕೊಂಡ ಪ್ರತಿ ₹ 100 ಕ್ಕೆ ಒಂದು ವರ್ಷಕ್ಕೆ ₹ 10 ಬಡ್ಡಿಯಾಗುತ್ತದೆ. ಈಗ ಉದಾಹರಣೆಯ ಮೂಲಕ ವಿವರವಾಗಿ ತಿಳಿಯೋಣ.

--- EXAMPLE START ---
Source: Class 8, Example 14 (page 17)
Problem: ಅನಿತ ₹ 5,000 ನ್ನು ಸಾಲಿಯಾನ 15% ಬಡ್ಡಿಯ ದರದಲ್ಲಿ ಸಾಲ ಪಡೆಯುತ್ತಾಳೆ. ವರ್ಷದ ಅಂತ್ಯದಲ್ಲಿ ಅವಳು ಪಾವತಿಸಬೇಕಾದ ಬಡ್ಡಿಯೆಷ್ಟು?
Solution:
ತೆಗೆದುಕೊಂಡ ಹಣ = ₹ 5,000, ಬಡ್ಡಿಯ ದರ = ಸಾಲಿಯಾನ 15% ಎಂದರೆ ತೆಗೆದುಕೊಂಡ ಪ್ರತಿ ₹ 100 ಕ್ಕೆ ಒಂದು ವರ್ಷದ ಅವಧಿಗೆ ₹ 15 ಬಡ್ಡಿ ಪಾವತಿಸಬೇಕು. ಅವಳು ₹ 5,000 ತೆಗೆದುಕೊಂಡಿರುವುದರಿಂದ ವರ್ಷದ ಕೊನೆಯಲ್ಲಿ ಪಾವತಿಸಬೇಕಾದ ಒಟ್ಟು ಬಡ್ಡಿ = $\frac{15}{100} \times 5000$
= ₹ 750
ಆದ್ದರಿಂದ, ವರ್ಷದ ಕೊನೆಯಲ್ಲಿ ಅವಳು ಪಾವತಿಸಬೇಕಾದ ಮೊತ್ತ
₹ 5,000+ ₹ 750 = ₹ 5,750
ವರ್ಷದ ಅವಧಿಗೆ ಬಡ್ಡಿಯನ್ನು ಲೆಕ್ಕ ಮಾಡಬಹುದಾದ ಒಂದು ಸಾಮಾನ್ಯ ರೂಪವನ್ನು ನಾವು ಬರೆಯಬಹುದು.
ತೆಗೆದುಕೊಂಡ ಹಣ ಅಥವಾ ಅಸಲು 'P' ಹಾಗೂ ವಾರ್ಷಿಕ ಬಡ್ಡಿಯ ದರ ಪ್ರತಿ ನೂರಕ್ಕೆ 'R' ಆಗಿರಲಿ.
ತೆಗೆದುಕೊಂಡ ಪ್ರತಿ ₹ 100 ಕ್ಕೆ ಪಾವತಿಸಬೇಕಾದ ಬಡ್ಡಿ ₹ R.
ಆದ್ದರಿಂದ ತೆಗೆದುಕೊಂಡ ₹ P ಗೆ ಒಂದು ವರ್ಷದ ಅಂತ್ಯಕ್ಕೆ ಪಾವತಿಸಬೇಕಾದ ಬಡ್ಡಿಯು,
$$ \frac{R \times P}{100} = \frac{P \times R}{100} $$
--- EXAMPLE END ---

### 7.4.1 ಹೆಚ್ಚು ವರ್ಷಗಳಿಗೆ ಬಡ್ಡಿಯನ್ನು ಲೆಕ್ಕಿಸುವುದು.

ಹಣವನ್ನು ಒಂದು ವರ್ಷಕ್ಕಿಂತ ಹೆಚ್ಚು ಅವಧಿಗೆ ತೆಗೆದುಕೊಂಡರೆ, ತೆಗೆದುಕೊಂಡ ಅಷ್ಟು ವರ್ಷಗಳಿಗೆ ಬಡ್ಡಿಯನ್ನು ತೆರಬೇಕಾಗುತ್ತದೆ. ಮೇಲಿನ ಉದಾಹರಣೆಯಲ್ಲಿ ಅನಿತ ಅದೇ ಬಡ್ಡಿ ದರದಲ್ಲಿ ಸಾಲವನ್ನು ಎರಡು ವರ್ಷಗಳಿಗೆ ಪಡೆದಿದ್ದರೆ ತಾನು ಪಾವತಿಸಿದ ಬಡ್ಡಿಯ ಎರಡರಷ್ಟು ಬಡ್ಡಿಯನ್ನು ತೆರಬೇಕಾಗುತ್ತಿತ್ತು. ಅಂದರೆ, ಮೊದಲನೆ ವರ್ಷಕ್ಕೆ ₹ 750 ಹಾಗೂ ಎರಡನೇ ವರ್ಷಕ್ಕೆ ₹ 750 ಬಡ್ಡಿಯಾಗುತ್ತಿತ್ತು. ಈ ರೀತಿಯಾಗಿ ನಿಗಧಿತ ಅಸಲಿಗೆ ಲೆಕ್ಕಿಸುವ ಬಡ್ಡಿಯನ್ನು 'ಸರಳ ಬಡ್ಡಿ' ಎನ್ನುತ್ತೇವೆ. ಸಾಲ ತೆಗೆದುಕೊಂಡ ಅವಧಿಯು ಹೆಚ್ಚಿದಂತೆ ಬಡ್ಡಿಯು ಹೆಚ್ಚುತ್ತದೆ. ₹ 100 ನ್ನು 3 ವರ್ಷಗಳಿಗೆ 18% ಬಡ್ಡಿಯ ದರದಲ್ಲಿ ತೆಗೆದುಕೊಂಡಿದ್ದರೆ, 3 ವರ್ಷಗಳ ಅಂತ್ಯದಲ್ಲಿ ಪಾವತಿಸಬೇಕಾದ ಬಡ್ಡಿಯು $18+ 18+ 18 = 3 \times 18 = ₹ 54$.

ಒಂದು ವರ್ಷದ ಅವಧಿಗೆ ಸರಳ ಬಡ್ಡಿಯನ್ನು ಲೆಕ್ಕಿಸುವ ಸಾಮಾನ್ಯ ರೂಪವನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು.
ಅಸಲು ₹ P ಗೆ R% ಬಡ್ಡಿಯ ದರದಲ್ಲಿ ಒಂದು ವರ್ಷದ ಅವಧಿಗೆ ಪಾವತಿಸಬೇಕಾದ ಬಡ್ಡಿಯು $\frac{R \times P}{100}$ ಎಂದು ತಿಳಿದಿದೆ. ಆದ್ದರಿಂದ, T ವರ್ಷಗಳಿಗೆ ಪಾವತಿಸಬೇಕಾದ ಬಡ್ಡಿಯು (I)
$$ \frac{T \times R \times P}{100} = \frac{P \times R \times T}{100} \text{ ಅಥವಾ } \frac{PRT}{100} $$
T ವರ್ಷಗಳ ಅಂತ್ಯದಲ್ಲಿ ಪಾವತಿಸಬೇಕಾದ ಮೊತ್ತ $A = P + I$

ಒಂದು ವಸ್ತುವಿಗೆ ಸಂಬಂಧಿಸಿದ ಎರಡು ಬೆಲೆಗಳನ್ನು ನೀಡಿದಾಗ, ಹೇಗೆ ಮೂರನೇ ಪರಿಮಾಣ ಕಂಡು ಹಿಡಿಯುತ್ತಿದ್ದೇವೆಯೋ ಅದೇ ರೀತಿ, $I = \frac{P \times T \times R}{100}$ ಸಂಬಂಧದ ನಾಲ್ಕು ಅಂಶಗಳಲ್ಲಿ ಯಾವುದಾದರೂ ಮೂರು ಅಂಶಗಳನ್ನು ನೀಡಿದಾಗ, ಉಳಿದ ಅಂಶವನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು.

--- EXAMPLE START ---
Source: Class 8, Example 15 (page 19)
Problem: ₹4,500 ಮೊಬಲಿಗಿಗೆ 2 ವರ್ಷಗಳ ಅವಧಿಗೆ ಮನೋಹರ್ ₹750 ಬಡ್ಡಿ ಪಾವತಿಸಿದರೆ, ಬಡ್ಡಿಯ ದರವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ ?
Solution:
**ಪರಿಹಾರ - 1**
$I = \frac{P \times T \times R}{100}$
ಆದ್ದರಿಂದ, $750 = \frac{4500 \times 2 \times R}{100}$
ಅಥವಾ $\frac{750}{45 \times 2} = R$
ಆದ್ದರಿಂದ, ಬಡ್ಡಿಯ ದರ $= 8\frac{1}{3}\%$

**ಪರಿಹಾರ - 2**
2 ವರ್ಷದ ಅವಧಿಗೆ ಪಾವತಿಸಿದ ಬಡ್ಡಿ ₹ 750
ಆದ್ದರಿಂದ, ಒಂದು ವರ್ಷಕ್ಕೆ ಪಾವತಿಸಿದ ಬಡ್ಡಿ ₹ $\frac{750}{2} = ₹ 375$
₹ 4,500 ಕ್ಕೆ ಪಾವತಿಸಿದ ಬಡ್ಡಿ ₹ 375
ಆದ್ದರಿಂದ, ₹ 100 ಕ್ಕೆ ಪಾವತಿಸಿದ ಬಡ್ಡಿಯ ದರ
$= \frac{375 \times 100}{4500} = 8\frac{1}{3}\%$
--- EXAMPLE END ---

### ಇಲ್ಲಿಯವರೆಗೆ ಚರ್ಚಿಸಿರುವ ಅಂಶಗಳು

1. ಪರಿಮಾಣಗಳನ್ನು ಹೋಲಿಸುವ ಒಂದು ಕ್ರಮ ಶೇಕಡಾ. ಛೇದ 100 ಇರುವ ಭಿನ್ನರಾಶಿಗಳಲ್ಲಿನ ಅಂಶವೇ ಶೇಕಡಾ. ಶೇಕಡಾ ಎಂದರೆ 100ಕ್ಕೆ ಎಂದರ್ಥ. ಉದಾಹರಣೆಗೆ 82% ಅಂಕ ಎಂದರೆ 100ಕ್ಕೆ 82 ಎಂದರ್ಥ.
2. ಭಿನ್ನರಾಶಿಯನ್ನು ಶೇಕಡಾಕ್ಕೆ ಹಾಗೂ ಶೇಕಡಾವನ್ನು ಭಿನ್ನರಾಶಿಗೆ ಪರಿವರ್ತಿಸಬಹುದು.
ಉದಾಹರಣೆಗೆ, $\frac{1}{4} = \frac{1}{4} \times 100\%$ ಹಾಗೂ $75\% = \frac{75}{100} = \frac{3}{4}$
3. ದಶಮಾಂಶಗಳನ್ನು ಶೇಕಡಾಕ್ಕೆ ಹಾಗೂ ಶೇಕಡಾವನ್ನು ದಶಮಾಂಶಗಳಿಗೆ ಪರಿವರ್ತಿಸಬಹುದು.
ಉದಾಹರಣೆಗೆ , $0.25 = 0.25 \times 100\% = 25\%$
4. ದಿನನಿತ್ಯದ ಜೀವನದಲ್ಲಿ ಶೇಕಡಾದ ಬಳಕೆ ವ್ಯಾಪಕವಾಗಿದೆ.
(a) ಒಟ್ಟು ಪ್ರಮಾಣಕ್ಕೆ ಶೇಕಡಾ ದರವನ್ನು ನೀಡಿದಾಗ ಸರಿಯಾದ ಮೌಲ್ಯವನ್ನು ಕಂಡು ಹಿಡಿಯುವುದನ್ನು ಕಲಿತಿದ್ದೇವೆ.
(b) ಪರಿಮಾಣದ ಭಾಗಗಳನ್ನು ಅನುಪಾತಗಳಲ್ಲಿ ಕೊಟ್ಟಾಗ, ಅವುಗಳನ್ನು ಶೇಕಡಾಗೆ ಪರಿವರ್ತಿಸುವ ವಿಧಾನವನ್ನು ಕಲಿತಿದ್ದೇವೆ.
(c) ಪರಿಮಾಣದಲ್ಲಿನ ಏರಿಕೆ ಅಥವಾ ಇಳಿಕೆಯನ್ನು ಶೇಕಡಾದಲ್ಲಿ ವ್ಯಕ್ತಪಡಿಸಬಹುದು.
(d) ಒಂದು ವ್ಯವಹಾರದಲ್ಲಿನ ಲಾಭ ಅಥವಾ ನಷ್ಟವನ್ನು ಶೇಕಡಾದಲ್ಲಿ ವ್ಯಕ್ತಪಡಿಸಬಹುದು.
(e) ಸಾಲದ ಹಣಕ್ಕೆ ಬಡ್ಡಿಯನ್ನು ಲೆಕ್ಕ ಹಾಕುವಾಗ, ಬಡ್ಡಿಯ ದರ ಪ್ರತಿನೂರಕ್ಕೆ ನೀಡಲಾಗಿರುತ್ತದೆ. ಉದಾಹರಣೆಗೆ, ₹ 800ನ್ನು ಮೂರು ವರ್ಷಗಳಿಗೆ 12% ಬಡ್ಡಿ ದರದಲ್ಲಿ ಪಡೆಯಲಾಗಿದೆ.

# ಅಧ್ಯಾಯ 7 ಪರಿಮಾಣಗಳ ಹೋಲಿಕೆ

## 7.1 ಅನುಪಾತ ಮತ್ತು ಶೇಕಡಾ ಕ್ರಮಗಳ ಪುನರಾವಲೋಕನ

ಅನುಪಾತ ಎಂದರೆ ಎರಡು ಪರಿಮಾಣಗಳ ಹೋಲಿಕೆ ಎಂಬುದು ನಮಗೆ ತಿಳಿದಿದೆ.
ಒಂದು ಬುಟ್ಟಿಯಲ್ಲಿ ಎರಡು ವಿಧದ ಹಣ್ಣುಗಳಿವೆ: ಅವು 20 ಸೇಬು ಮತ್ತು 5 ಕಿತ್ತಳೆ ಹಣ್ಣುಗಳೆಂದು ತಿಳಿಯೋಣ. ಅಗ ಕಿತ್ತಳೆ ಹಣ್ಣುಗಳ ಸಂಖ್ಯೆಗೂ ಸೇಬು ಹಣ್ಣುಗಳ ಸಂಖ್ಯೆಗೂ ಇರುವ ಅನುಪಾತವು 5:20.
$\frac{5}{20} = \frac{1}{4}$ ಎಂಬ ಭಿನ್ನರಾಶಿಯನ್ನು ಬಳಸಿಕೊಂಡೂ ಇದನ್ನು ಹೋಲಿಸಬಹುದು.
ಕಿತ್ತಳೆ ಹಣ್ಣುಗಳ ಸಂಖ್ಯೆಯು ಸೇಬು ಹಣ್ಣುಗಳ ಸಂಖ್ಯೆಯ $\frac{1}{4}$ ರಷ್ಟಾಗಿರುವುದು. ಅನುಪಾತ ರೂಪದಲ್ಲಿ ಇದು 1:4. ಇದನ್ನು "1 ಅನುಪಾತ 4" (1 is to 4) ಎಂದು ಓದಬೇಕು.
ಅಥವಾ
ಸೇಬು ಹಣ್ಣುಗಳ ಸಂಖ್ಯೆಗೂ, ಕಿತ್ತಳೆ ಹಣ್ಣುಗಳ ಸಂಖ್ಯೆಗೆ ಇರುವ ಅನುಪಾತ = $\frac{20}{5} = \frac{4}{1}$
ಇದರ ಅರ್ಥ. ಸೇಬು ಹಣ್ಣುಗಳ ಸಂಖ್ಯೆಯು ಕಿತ್ತಳೆ ಹಣ್ಣುಗಳ ಸಂಖ್ಯೆಯ 4 ರಷ್ಟಾಗಿರುವುದು. ಈ ಹೋಲಿಕೆಯನ್ನು ಶೇಕಡಾ ಕ್ರಮದಿಂದಲೂ ಮಾಡಬಹುದು.

25 ಹಣ್ಣುಗಳಲ್ಲಿ 5 ಕಿತ್ತಳೆ ಹಣ್ಣುಗಳಿವೆ.
ಹೀಗಾಗಿ ಕಿತ್ತಳೆ ಹಣ್ಣುಗಳ ಶೇಕಡಾವು
$\frac{5}{25} \times \frac{4}{4} = \frac{20}{100} = 20\%$
[ಛೇದವನ್ನು 100ಕ್ಕೆ ಪರಿವರ್ತಿಸಿದೆ.]

ಅಥವಾ
**ಏಕಾಂಶ ಪದ್ಧತಿಯಿಂದ**
25 ಹಣ್ಣುಗಳಲ್ಲಿ, ಕಿತ್ತಳೆ ಹಣ್ಣುಗಳ ಸಂಖ್ಯೆ
$\frac{5}{25} \times 100 = 20$

ಬುಟ್ಟಿಯಲ್ಲಿ ಕೇವಲ ಸೇಬು ಹಾಗೂ ಕಿತ್ತಳೆ ಹಣ್ಣುಗಳಿವೆ.
ಹೀಗಾಗಿ, ಸೇಬಿನ ಶೇಕಡಾ + ಕಿತ್ತಳೆಯ ಶೇಕಡಾ = 100%.
ಅಥವಾ ಸೇಬಿನ ಶೇಕಡಾ + 20% = 100%
ಅಥವಾ ಸೇಬಿನ ಶೇಕಡಾ = 100% - 20% = 80%
ಆದ್ದರಿಂದ ಬುಟ್ಟಿಯು 20% ಕಿತ್ತಳೆ ಹಾಗೂ 80% ಸೇಬನ್ನು ಹೊಂದಿದೆ.

--- EXAMPLE START ---
Source: Class 8, Example 1 (page 23)
Problem: ಒಂದು ಶಾಲೆಯ VII ನೇ ತರಗತಿಗೆ ಕಿರುಪ್ರವಾಸವನ್ನು ಏರ್ಪಡಿಸಲಾಗಿದೆ. ಒಟ್ಟು ವಿದ್ಯಾರ್ಥಿಗಳ ಸಂಖ್ಯೆಯ 60% ರಷ್ಟು ಹುಡುಗಿಯರಾಗಿದ್ದು ಅವರ ಸಂಖ್ಯೆ 18. ಕಿರುಪ್ರವಾಸದ ತಾಣವು ಶಾಲೆಯಿಂದ 55 ಕಿ.ಮೀ. ದೂರದಲ್ಲಿದ್ದು, ಸಾರಿಗೆ ಸಂಸ್ಥೆಯು ಪ್ರತಿ ಕಿ.ಮೀ.ಗೆ ₹12 ದರದಂತೆ ಶುಲ್ಕ ತೆಗೆದುಕೊಳ್ಳುತ್ತದೆ. ಒಟ್ಟು ಉಪಹಾರದ ಖರ್ಚು ₹4280.
ನೀವು ಕೆಳಗಿನವುಗಳನ್ನು ಹೇಳಬಲ್ಲಿರಾ?
1. ತರಗತಿಯಲ್ಲಿನ ಹುಡುಗಿಯರ ಸಂಖ್ಯೆಗೂ ಹುಡುಗರ ಸಂಖ್ಯೆಗೂ ಇರುವ ಅನುಪಾತ.
2. ತರಗತಿಯ ವಿದ್ಯಾರ್ಥಿಗಳೊಂದಿಗೆ ಇಬ್ಬರು ಶಿಕ್ಷಕರು ಸಹ ಹೋಗುತ್ತಿದ್ದರೆ, ಆಗ ಪ್ರತಿಯೊಬ್ಬರಿಗೆ ತಗುಲುವ ಖರ್ಚು.
3. ಅವರ ಮೊದಲ ನಿಲುಗಡೆಯು ಶಾಲೆಯಿಂದ 22 ಕಿ.ಮೀ ದೂರವಿದ್ದರೆ ಇದು ಒಟ್ಟು ದೂರವಾದ 55 ಕಿ.ಮೀ ನ ಶೇಕಡಾವಾರು ಎಷ್ಟಾಗುತ್ತದೆ? ಕ್ರಮಿಸಬೇಕಾದ ಉಳಿದ ದೂರವು ಒಟ್ಟು ದೂರದ ಶೇಕಡಾವಾರು ಎಷ್ಟಾಗುತ್ತದೆ?
Solution:
1. **ಹುಡುಗಿಯರಿಗೂ ಹುಡುಗರಿಗೂ ಇರುವ ಅನುಪಾತವನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.**
ಆಶೀಮಾ ಮತ್ತು ಜಾನ್ ಈ ಕೆಳಗಿನ ಉತ್ತರಗಳನ್ನು ಪಡೆದಿರುತ್ತಾರೆ. ಅವರಿಗೆ ಹುಡುಗರ ಸಂಖ್ಯೆ ಮತ್ತು ವಿದ್ಯಾರ್ಥಿಗಳ ಸಂಖ್ಯೆಗಳನ್ನು ತಿಳಿಯುವ ಅಗತ್ಯವಿದೆ.

**ಆಶೀಮಾಳು ಹೀಗೆ ಮಾಡಿದ್ದಾಳೆ:**
ಒಟ್ಟು ವಿದ್ಯಾರ್ಥಿಗಳ ಸಂಖ್ಯೆ x ಎಂದಿರಲಿ.
x ನ 60% ಎಂದರೆ ಹುಡುಗಿಯರು.
ಆದ್ದರಿಂದ, x ನ 60% = 18
$\frac{60}{100} \times x = 18$ ಅಥವಾ
$x = 18 \times \frac{100}{60} = 30$
ಒಟ್ಟು ವಿದ್ಯಾರ್ಥಿಗಳ ಸಂಖ್ಯೆ = 30
ಹೀಗಾಗಿ ಹುಡುಗರ ಸಂಖ್ಯೆ = 30 - 18 = 12.
ಆದ್ದರಿಂದ ಹುಡುಗಿಯರ ಸಂಖ್ಯೆಗೂ ಹುಡುಗರ ಸಂಖ್ಯೆಗೂ ಇರುವ ಅನುಪಾತವು 18:12 ಅಥವಾ $\frac{18}{12} = \frac{3}{2}$
$\frac{3}{2}$ ನ್ನು 3:2 ಎಂದು ಬರೆಯಬಹುದು ಮತ್ತು ಇದನ್ನು 3 ಅನುಪಾತ 2 (3 is to 2)ಎಂದು ಓದಬೇಕು.

**ಜಾನ್‌ನು ಏಕಾಂಶ ಪದ್ಧತಿಯನ್ನು ಬಳಸಿದ್ದಾನೆ.**
100 ವಿದ್ಯಾರ್ಥಿಗಳಲ್ಲಿ 60 ಹುಡುಗಿಯರು ಇರುವರು. ಅಲ್ಲಿಗೆ 100 ವಿದ್ಯಾರ್ಥಿಗಳಲ್ಲಿ $\frac{100}{60}$ ಒಬ್ಬ ಹುಡುಗಿ ಇರುವಳು. ಹೀಗಾಗಿ, ಎಷ್ಟು ಜನ ವಿದ್ಯಾರ್ಥಿಗಳಲ್ಲಿ 18 ಹುಡುಗಿಯರು ಇರುವರು?
ಒಟ್ಟು ವಿದ್ಯಾರ್ಥಿಗಳ ಸಂಖ್ಯೆ = $\frac{100}{60} \times 18 = 30$

2. **ಪ್ರತಿಯೊಬ್ಬನ ಖರ್ಚು ಕಂಡುಹಿಡಿಯುವುದು:**
ಸಾರಿಗೆ ಖರ್ಚು = ಎರಡೂ ಮಾರ್ಗದ ದೂರ × ದರ
= (55 × 2) × 12 = 110 × 12 = ₹1320
ಒಟ್ಟು ವೆಚ್ಚ = ಉಪಹಾರದ ಖರ್ಚು + ಸಾರಿಗೆ ಖರ್ಚು
= ₹4280 + ₹1320 = ₹5600
ಒಟ್ಟು ಜನರ ಸಂಖ್ಯೆ = 18 ಹುಡುಗಿಯರು + 12 ಹುಡುಗರು + 2ಶಿಕ್ಷಕರು = 32 ಜನರು.
ಆಶೀಮಾ ಮತ್ತು ಜಾನ್‌ರು ಏಕಾಂಶ ಪದ್ಧತಿಯಿಂದ ಪ್ರತಿಯೊಬ್ಬನ ಖರ್ಚನ್ನು ಕಂಡುಹಿಡಿಯುವರು.
32ಜನರಿಗೆ, ಮಾಡಲಾದ ಖರ್ಚು ₹5600 ಆಗಿದೆ.
ಒಬ್ಬನಿಗೆ ಮಾಡಲಾದ ಖರ್ಚು = $\frac{5600}{32} = ₹ 175$

3. **ಮೊದಲನೇ ನಿಲುಗಡೆ ಸ್ಥಳವಿರುವ ದೂರ = 22 ಕಿ.ಮೀ ದೂರದ ಶೇಕಡಾ ಕಂಡುಹಿಡಿಯುವಿಕೆ:**

**ಆಶೀಮಾ ಈ ಪದ್ಧತಿಯನ್ನು ಬಳಸಿದ್ದಾಳೆ:**
$\frac{22}{55} = \frac{22}{55} \times \frac{100}{100} = 40\%$
[ಅವಳು ಅನುಪಾತವನ್ನು $\frac{100}{100} = 1$ ಗುಣಿಸಿ ಶೇಕಡಾಕ್ಕೆ ಪರಿವರ್ತಿಸಿರುತ್ತಾಳೆ.]

**ಜಾನ್ ಏಕಾಂಶ ಪದ್ಧತಿಯನ್ನು ಬಳಸಿರುತ್ತಾನೆ:**
55ಕಿ.ಮೀಗಳಲ್ಲಿ 22ಕಿ.ಮೀ ಪ್ರಯಾಣವಾಗಿದೆ.
1 ಕಿ.ಮೀನಲ್ಲಿ $\frac{22}{55}$ಕಿ.ಮೀ ಪ್ರಯಾಣವಾಗಿದೆ.
100 ಕಿ.ಮೀಗಳಲ್ಲಿ $\frac{22}{55} \times 100$ಕಿ.ಮೀ ಪ್ರಯಾಣವಾಗಿದೆ. ಅಂದರೆ ಒಟ್ಟು ದೂರದಲ್ಲಿ 40% ಪ್ರಯಾಣವಾಗಿದೆ.

ಇಬ್ಬರೂ ಸಹ, ಶಾಲೆಯಿಂದ ಮೊದಲನೇ ನಿಲುಗಡೆ ಸ್ಥಳವಿರುವ ದೂರವು ಒಟ್ಟು ದೂರದ 40%ದಷ್ಟಿದೆ ಎಂಬ ಉತ್ತರವನ್ನು ಪಡೆದಿರುತ್ತಾರೆ. ಆದ್ದರಿಂದ ಇನ್ನೂ ಕ್ರಮಿಸಬೇಕಾದ ದೂರದ ಶೇಕಡಾ ಪ್ರಮಾಣ = 100% - 40% = 60%.

## 7.2 ರಿಯಾಯಿತಿ ಕಂಡುಹಿಡಿಯುವುದು:

ವಸ್ತುವಿನ ನಮೂದಿತ ಬೆಲೆಯ (MP)ಮೇಲೆ ನೀಡಲಾಗುವ ಕಡಿತವನ್ನು ರಿಯಾಯಿತಿ ಎನ್ನುವರು.
ಸಾಮಾನ್ಯವಾಗಿ ಗ್ರಾಹಕರನ್ನು ಆಕರ್ಷಿಸಲು ಅಥವಾ ಮಾರಾಟವನ್ನು ಉತ್ತೇಜಿಸಲು ರಿಯಾಯಿತಿಯನ್ನು ನೀಡಲಾಗುತ್ತದೆ. ನೀವು
ನಮೂದಿತ ಬೆಲೆಯಿಂದ ಮಾರಾಟ ಬೆಲೆ (SP)ಯನ್ನು ಕಳೆಯುವುದರ ಮೂಲಕ ರಿಯಾಯಿತಿಯನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು.
ಹೀಗೆ, ರಿಯಾಯಿತಿ = ನಮೂದಿತ ಬೆಲೆ - ಮಾರಾಟ ಬೆಲೆ

--- EXAMPLE START ---
Source: Class 8, Example 2 (page 26)
Problem: ₹840 ನಮೂದಿತವಾಗಿದ್ದ ವಸ್ತುವನ್ನು ₹714 ಗಳಿಗೆ ಮಾರಾಟ ಮಾಡಲಾಯಿತು. ನೀಡಿದ ರಿಯಾಯಿತಿ ಎಷ್ಟು? ಮತ್ತು ಅದರ ಶೇಕಡಾ ಪ್ರಮಾಣ ಎಷ್ಟು?
Solution:
ರಿಯಾಯಿತಿ = ನಮೂದಿತ ಬೆಲೆ – ಮಾರಾಟ ಬೆಲೆ
= ₹840 - ₹714
= ₹126
ರಿಯಾಯಿತಿಯನ್ನು ನಮೂದಿತ ಬೆಲೆಯಲ್ಲಿ ನೀಡುವುದರಿಂದ, ನಾವು ನಮೂದಿತ ಬೆಲೆಯನ್ನು ಆಧಾರವನ್ನಾಗಿ ಬಳಸಬೇಕು.
ನಮೂದಿತ ಬೆಲೆ ₹840 ರ ಮೇಲೆ ರಿಯಾಯಿತಿ ₹126
ನಮೂದಿತ ಬೆಲೆ ₹100 ಆದಾಗ ರಿಯಾಯಿತಿ ಎಷ್ಟಾಗಬಹುದು?
ರಿಯಾಯಿತಿ = $\frac{126}{840} \times 100 = 15\%$
ಶೇಕಡಾ ರಿಯಾಯಿತಿಯನ್ನು ಕೊಟ್ಟಾಗಲೂ ಸಹ ನೀವು ರಿಯಾಯಿತಿಯನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು.
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 8, Example 3 (page 26)
Problem: ಒಂದು ಫ್ರಾಕ್‌ನ ನಮೂದಿತ ಬೆಲೆ ₹220 ಅದರ ಮಾರಾಟದ ಮೇಲೆ 20% ರಿಯಾಯಿತಿಯನ್ನು ಘೋಷಿಸಿದೆ. ಅದರ ಮೇಲಿನ ರಿಯಾಯಿತಿ ಮೊಬಲಗು ಮತ್ತು ಮಾರಾಟ ಬೆಲೆ ಎಷ್ಟು?
Solution:
ನಮೂದಿತ ಬೆಲೆಯ ಮೇಲೆ 20%ರಷ್ಟು ರಿಯಾಯಿತಿ ಎಂದರೆ ₹100 ಗಳ ಮೇಲೆ ₹20 ರಿಯಾಯಿತಿ ಎಂದರ್ಥ.
ಏಕಾಂಶ ಪದ್ಧತಿಯಿಂದ, ₹1 ರ ಮೇಲಿನ ರಿಯಾಯಿತಿಯು $\frac{20}{100}$ ಆಗುವುದು.
₹220 ರ ಮೇಲಿನ ರಿಯಾಯಿತಿ = $\frac{20}{100} \times ₹220 = ₹44$
ಮಾರಾಟ ಬೆಲೆ = (₹220 – ₹44) ಅಥವಾ ₹176
ರೆಹಾನಾಳು ಮಾರಾಟ ಬೆಲೆಯನ್ನು ಈ ರೀತಿ ಕಂಡುಹಿಡಿದಳು.
20% ರಿಯಾಯಿತಿ ಎಂದರೆ ನಮೂದಿತ ಬೆಲೆ 100 ರೂಪಾಯಿಗೆ ರಿಯಾಯಿತಿ ₹20 ಎಂದರ್ಥ.
ಆದ್ದರಿಂದ ಮಾರಾಟ ಬೆಲೆಯು ₹80
ಏಕಾಂಶ ಪದ್ಧತಿಯನ್ನು ಬಳಸಿದಾಗ,
ನಮೂದಿತ ಬೆಲೆ ₹100 ಆದಾಗ ಮಾರಾಟ ಬೆಲೆಯು 80ರೂ;
ನಮೂದಿತ ಬೆಲೆ ₹1 ಆದಾಗ ಮಾರಾಟ ಬೆಲೆಯು ₹ $\frac{80}{100}$ ಆಗುವುದು. ಹಾಗಾಗಿ ನಮೂದಿತ ಬೆಲೆ ₹220 ಆದಾಗ ಮಾರಾಟ ಬೆಲೆ = $\frac{80}{100} \times 220 = ₹176$

### 7.2.1 ಶೇಕಡಾದರಗಳಲ್ಲಿ ಅಂದಾಜಿಸುವಿಕೆ:

ಅಂಗಡಿಯೊಂದರಲ್ಲಿ ನಿಮ್ಮ ಬೆಲೆ ಪಟ್ಟಿಯು (bill) ₹577.80 ಹಾಗೂ ಅಂಗಡಿಯವನು 15% ರಿಯಾಯಿತಿಯನ್ನು ನೀಡುತ್ತಿದ್ದಾನೆ. ಸಂದಾಯಮಾಡಬೇಕಾದ ಮೊಬಲಗನ್ನು ನೀವು ಹೇಗೆ ಅಂದಾಜಿಸುವಿರಿ?
(i) ಬೆಲೆ ಪಟ್ಟಿಯನ್ನು ₹577.80 ರ ಹತ್ತರ ಸನ್ನಿಹಿತ ಬೆಲೆಗೆ ಅಂದರೆ, ₹580 ಗೆ, ಸಮಗೊಳಿಸಿ.
(ii) ಇದರ 10%ನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ ಅಂದರೆ, ₹ $\frac{10}{100} \times 580 = ₹58$
(iii) ಇದರ ಅರ್ಧವನ್ನು ತೆಗೆದುಕೊಳ್ಳಿ ಅಂದರೆ, ₹ $\frac{1}{2} \times 58 = ₹29$
(iv) (ii) ಮತ್ತು (iii) ರಲ್ಲಿನ ಮೊತ್ತವನ್ನು ಕೂಡಿಸಿ ₹87 ನ್ನು ಪಡೆಯಿರಿ.
ಆದ್ದರಿಂದ ನೀವೀಗ ನಿಮ್ಮ ಬೆಲೆ ಪಟ್ಟಿಯನ್ನು ಈ ಮೂಲಕ ₹87 ಅಥವಾ ₹85 ಕಡಿಮೆ ಮಾಡಿದರೆ ಅದು ಸುಮಾರು ₹495 ಆಗುತ್ತದೆ.

## 7.3 ಮಾರಾಟ ತೆರಿಗೆ/ಮೌಲ್ಯವರ್ಧಿತ ತೆರಿಗೆ (ವ್ಯಾಟ್) / ಸರಕು ಮತ್ತು ಸೇವಾತೆರಿಗೆ (ಜಿಎಸ್‌ಟಿ)

ಈ ಮಾರಾಟ ತೆರಿಗೆಯನ್ನು ವಸ್ತುವಿನ ಮಾರಾಟದ ಮೇಲೆ ಸರ್ಕಾರವು ವಿಧಿಸುತ್ತದೆ. ಇದನ್ನು ಅಂಗಡಿಯವನು ಗ್ರಾಹಕನಿಂದ ಸಂಗ್ರಹಿಸಿ ಅದನ್ನು ಸರ್ಕಾರಕ್ಕೆ ನೀಡುವನು. ಆದ್ದರಿಂದ ಇದು ಯಾವಾಗಲೂ ವಸ್ತುವಿನ ಮಾರಾಟದ ಬೆಲೆಯ ಮೇಲೆ ಆಧರಿತವಾಗಿದೆ ಮತ್ತು ಬಿಲ್ಲಿನ ಮೌಲ್ಯಕ್ಕೆ ಸೇರಿಸಲಾಗುತ್ತದೆ. ಆದರೆ ಇಂದಿನ ದಿನಗಳಲ್ಲಿ ಬೆಲೆಯಲ್ಲಿ ಮೌಲ್ಯವರ್ಧಿತ ತೆರಿಗೆಯು ಸೇರಿರುತ್ತದೆ.
2017ನೇ ಇಸವಿ ಜುಲೈ 1 ರಿಂದ ಭಾರತ ಸರ್ಕಾರವು ಸರಕು ಮತ್ತು ಸೇವಾ ತೆರಿಗೆ (GST)ಯನ್ನು ಜಾರಿಗೆ ತಂದಿತು. ಇದನ್ನು ಸರಕುಗಳ ಸರಬರಾಜು ಅಥವಾ ಸೇವೆ ಅಥವಾ ಇವೆರಡರ ಮೇಲೂ ವಿಧಿಸಲಾಗುತ್ತದೆ.

--- EXAMPLE START ---
Source: Class 8, Example 4 (page 28)
Problem: (ಮಾರಾಟ ತೆರಿಗೆಯನ್ನು ಕಂಡುಹಿಡಿಯುವುದು) ಅಂಗಡಿಯೊಂದರಲ್ಲಿ ಒಂದು ಜೊತೆ ಉರುಳು ಮೆಟ್ಟಿನ (Roller skates) ಬೆಲೆ ₹450. 5% ಮಾರಾಟ ತೆರಿಗೆಯನ್ನು ವಿಧಿಸಿದರೆ, ಬಿಲ್ಲಿನ ಮೊತ್ತವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
Solution:
₹100 ಕ್ಕೆ ಪಾವತಿಸುವ ತೆರಿಗೆಯು ₹5 ಆಗಿದೆ.
₹450 ಕ್ಕೆ ಪಾವತಿಸುವ ತೆರಿಗೆ
= ₹ $\frac{5}{100} \times 450 = ₹22.50$
ಬಿಲ್ಲಿನ ಮೊತ್ತ = ವಸ್ತುವಿನ ಅಸಲು + ಮಾರಾಟ ತೆರಿಗೆ
= ₹450 + ₹22.50 = ₹472.50
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 8, Example 5 (page 28)
Problem: (ಮೌಲ್ಯವರ್ಧಿತ ತೆರಿಗೆ (ವ್ಯಾಟ್)) ವಹೀದಾಳು ಗಾಳಿಯಿಂದ ತಂಪುಗೊಳಿಸುವ ಯಂತ್ರ (air cooler)ವೊಂದನ್ನು 10% ತೆರಿಗೆಯನ್ನು ಒಳಗೊಂಡಂತೆ ₹3300 ಗೆ ಖರೀದಿಸಿದಳು. ವ್ಯಾಟ್‌ನ್ನು ಸೇರಿಸುವುದಕ್ಕಿಂತ ಮೊದಲು ಇದ್ದ ಅದರ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
Solution:
ಬೆಲೆಯು ವ್ಯಾಟ್ ಅಂದರೆ ಮೌಲ್ಯವರ್ಧಿತ ತೆರಿಗೆಯನ್ನು ಒಳಗೊಂಡಿದೆ. ಹೀಗಾಗಿ, 10% ವ್ಯಾಟ್ ಎಂದರೆ ₹100 ಎಂಬುದು ವ್ಯಾಟ್ ರಹಿತ ಬೆಲೆಯಾದರೆ ₹110 ಎಂಬುದು ವ್ಯಾಟ್ ಸಹಿತ ಬೆಲೆ ಎಂದರ್ಥ.
ಈಗ ವ್ಯಾಟ್ ಸಹಿತ ಬೆಲೆಯು ₹110 ಆದಾಗ ಮೂಲ ಬೆಲೆಯು ₹100 ಆಗಿರುವುದು. ಆದ್ದರಿಂದ ತೆರಿಗೆ ಸಹಿತ ಬೆಲೆಯು ₹3300 ಆದಾಗ ಮೂಲ ಬೆಲೆ = ₹ $\frac{100}{110} \times 3300 = ₹3000$

Problem: ಸಲೀಂನು ಒಂದು ವಸ್ತುವನ್ನು ₹784ಕ್ಕೆ ಕೊಂಡನು. ಇದರೊಳಗೆ 12%ರಷ್ಟು GST ಸೇರಿದೆ ಹಾಗಾದರೆ ಜಿಎಸ್‌ಟಿ ಸೇರಿಸುವುದಕ್ಕೆ ಮೊದಲು ವಸ್ತುವಿನ ಬೆಲೆ ಎಷ್ಟು?

ಮಾರಾಟ ಬೆಲೆ ₹112 ಆದಾಗ ಮೂಲ ಬೆಲೆ = ₹100
ಮಾರಾಟ ಬೆಲೆ ₹ 784 ಆದರೆ ಮೂಲ ಬೆಲೆ = ₹ $\frac{100}{112} \times 784 = ₹700$

## 7.4 ಚಕ್ರಬಡ್ಡಿ

ಬ್ಯಾಂಕ್ ಅಥವಾ ಅಂಚೆ ಕಛೇರಿಗಳಂತಹ ಸಂಸ್ಥೆಗಳು ತಮ್ಮಲ್ಲಿ ಠೇವಣಿಯಾಗಿರಿಸಿದ ಹಣಕ್ಕೆ ಪಾವತಿಸುವ ಹೆಚ್ಚುವರಿ ಹಣವನ್ನು ಬಡ್ಡಿ ಎನ್ನುವರು. ಸಾಲವಾಗಿ ಹಣವನ್ನು ತೆಗೆದುಕೊಂಡಾಗಲೂ ಸಹ ಜನರು ಬಡ್ಡಿಯನ್ನು ಪಾವತಿಸಬೇಕಾಗುತ್ತದೆ. ನಾವೀಗಾಗಲೇ ಸರಳ ಬಡ್ಡಿಯನ್ನು ಲೆಕ್ಕ ಮಾಡುವುದನ್ನು ತಿಳಿದಿದ್ದೇವೆ.

--- EXAMPLE START ---
Source: Class 8, Example 7 (page 30)
Problem: ₹10,000 ಮೊತ್ತವನ್ನು ವಾರ್ಷಿಕ 15% ದರದಲ್ಲಿ 2 ವರ್ಷಗಳ ಅವಧಿಗೆ ಸಾಲವಾಗಿ ಪಡೆಯಲಾಗಿದೆ. 2 ವರ್ಷಗಳ ಕೊನೆಯಲ್ಲಿ ಇದರ ಮೇಲಿನ ಸರಳಬಡ್ಡಿ ಮತ್ತು ಒಟ್ಟು ಮೊತ್ತವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
Solution:
₹100 ಕ್ಕೆ 1 ವರ್ಷಕ್ಕೆ ವಿಧಿಸುವ ಬಡ್ಡಿಯು ರೂ.15
ಹೀಗೆ, ₹10,000ಕ್ಕೆ ವಿಧಿಸುವ ಬಡ್ಡಿ = ₹ $\frac{15}{100} \times 10000 = 1500$
2 ವರ್ಷದ ಬಡ್ಡಿ = 1500 × ₹2 = ₹3000
2 ವರ್ಷಗಳ ಕೊನೆಯಲ್ಲಿ ಪಾವತಿಸಬೇಕಾಗುವ ಮೊತ್ತ = ಅಸಲು + ಬಡ್ಡಿ
= ₹10,000 + ₹3000 = ₹13,000
--- EXAMPLE END ---

### ಚಕ್ರಬಡ್ಡಿಯನ್ನು ಲೆಕ್ಕಿಸುವುದು:

ಸಾಮಾನ್ಯವಾಗಿ, ಪಾವತಿಸುವ ಅಥವಾ ವಿಧಿಸುವ ಬಡ್ಡಿ ಸರಳ ಬಡ್ಡಿಯಲ್ಲ. ಈ ಬಡ್ಡಿಯನ್ನು ಹಿಂದಿನ ವರ್ಷದ ಮೊತ್ತಕ್ಕೆ ಲೆಕ್ಕಿಸಲಾಗುತ್ತದೆ. ಇದನ್ನು ಚಕ್ರಬಡ್ಡಿ (C.I) ಎನ್ನುವರು.
ನಾವೀಗ ಉದಾಹರಣೆಯೊಂದನ್ನು ತೆಗೆದುಕೊಳ್ಳೋಣ ಮತ್ತು ವರ್ಷದಿಂದ ವರ್ಷಕ್ಕೆ ದೊರಕುವ ಬಡ್ಡಿಯನ್ನು ಕಂಡುಹಿಡಿಯೋಣ. ಪ್ರತಿ ವರ್ಷವೂ ನಮ್ಮ ಹಣ ಅಥವಾ ಅಸಲು ಬದಲಾಗುವುದು.

ವಾರ್ಷಿಕವಾಗಿ ಲೆಕ್ಕ ಮಾಡುವಂತೆ 8% ಬಡ್ಡಿದರದಲ್ಲಿ ಹೀನಾಳು 2 ವರ್ಷಗಳಿಗೆ ₹20,000 ಗಳ ಸಾಲವನ್ನು ಪಡೆದಿದ್ದಾಳೆ. ಪ್ರತಿವರ್ಷವೂ ಬಡ್ಡಿ ಲೆಕ್ಕಮಾಡಿದರೆ 2 ವರ್ಷಗಳ ಕೊನೆಯಲ್ಲಿ ಅವಳು ಕೊಡಬೇಕಾದ ಚಕ್ರಬಡ್ಡಿ (CI) ಮತ್ತು ಒಟ್ಟು ಮೊತ್ತವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
ಆಗ ಅಸ್ಲಾಂನು ವರ್ಷದಿಂದ ವರ್ಷಕ್ಕೆ ದೊರಕುವ ಬಡ್ಡಿಯನ್ನು ಕಂಡುಹಿಡಿಯಬೇಕೆಂಬುದು ಇದರ ಅರ್ಥವೇ ಎಂದು ಶಿಕ್ಷಕರಲ್ಲಿ ಕೇಳಿದನು. ಅದಕ್ಕೆ ಶಿಕ್ಷಕರು ಹೌದು ಎಂದು ಉತ್ತರಿಸಿ ಅವನಿಗೆ ಕೆಳಗಿನ ಹಂತಗಳನ್ನು ಅನುಸರಿಸಲು ತಿಳಿಸಿದರು.

1. **ಒಂದು ವರ್ಷದ ಸರಳಬಡ್ಡಿ (SI)ಯನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.**
ಮೊದಲನೇ ವರ್ಷದ ಅಸಲು P₁ ಎಂದಿರಲಿ. ಇಲ್ಲಿ P₁ = ₹20,000.
SI₁ = ವಾರ್ಷಿಕ 8% ದರದಲ್ಲಿ ಮೊದಲನೇ ವರ್ಷದ SI₁ = ₹ $\frac{20000 \times 8}{100} = ₹1600$

2. **ನಂತರ ಪಾವತಿಸಬೇಕಾಗುವ ಅಥವಾ ಸ್ವೀಕರಿಸಬೇಕಾಗುವ ಮೊತ್ತವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ. ಇದು ಮುಂದಿನ ವರ್ಷದ ಅಸಲಾಗುವುದು.**
ಮೊದಲನೇ ವರ್ಷದ ಕೊನೆಯಲ್ಲಿ ಪಾವತಿಸಬೇಕಾಗುವ ಮೊತ್ತ = P₁ + SI₁
= ₹20,000 + ₹1600
= ₹21,600 = P₂(2ನೇ ವರ್ಷದ ಅಸಲು)

3. **ಈ ಮೊತ್ತಕ್ಕೆ ಇನ್ನೊಂದು ವರ್ಷದ ಅವಧಿಗೆ ಪುನಃ ಬಡ್ಡಿಯನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.**
SI₂ = ವಾರ್ಷಿಕ 8% ದರದಲ್ಲಿ 2ನೇ ವರ್ಷದ SI₂ = ₹$21,600 \times \frac{8}{100} = ₹1728$

4. **ಎರಡನೇ ವರ್ಷದ ಕೊನೆಯಲ್ಲಿ ಪಾವತಿಸಬೇಕಾಗುವ ಅಥವಾ ಸ್ವೀಕರಿಸಬೇಕಾಗುವ ಮೊತ್ತವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.**
ಎರಡನೇ ವರ್ಷದ ಕೊನೆಯಲ್ಲಿ ಪಾವತಿಸಬೇಕಾಗುವ ಮೊತ್ತ = P₂ + SI₂
= ₹21,600 + ₹1728
= ₹23,328
ನೀಡಲಾಗುವ ಒಟ್ಟು ಬಡ್ಡಿ = ₹1600 + ₹1728 = ₹3328

ರೀಟಾಳು ಸರಳಬಡ್ಡಿಯ ಲೆಕ್ಕದಲ್ಲಿ ಈ ಮೊತ್ತವು ಬೇರೆಯಾಗಿರುವುದೇ? ಎಂದು ಕೇಳಿದಳು. ಅದಕ್ಕೆ ಶಿಕ್ಷಕರು ಅವಳಿಗೆ 2 ವರ್ಷಗಳ ಅವಧಿಗೆ ಬಡ್ಡಿಯನ್ನು ಕಂಡುಹಿಡಿಯಲು ಮತ್ತು ಅವಳೇ ಸ್ವತಃ ಪರೀಕ್ಷಿಸಿಕೊಳ್ಳುವಂತೆ ಹೇಳಿದರು.
2 ವರ್ಷಗಳ ಅವಧಿಗೆ ಸರಳಬಡ್ಡಿ = ₹ $\frac{20000 \times 8 \times 2}{100} = ₹3200$.
ಚಕ್ರಬಡ್ಡಿಯನ್ನು ಬಳಸಿದಾಗ ಹೀನಾಳು ₹128 ಹೆಚ್ಚಿಗೆ ಪಾವತಿಸಬೇಕಾಗುವುದು ಎಂದು ರೀಟಾಳು ಹೇಳಿದಳು.
ನಾವೀಗ ಸರಳಬಡ್ಡಿ ಮತ್ತು ಚಕ್ರಬಡ್ಡಿಗಳ ನಡುವಿನ ವ್ಯತ್ಯಾಸವನ್ನು ನೋಡೋಣ. ನಾವು ₹100ರಿಂದ ಪ್ರಾರಂಭಿಸೋಣ.

| | ಸರಳಬಡ್ಡಿ | ಚಕ್ರಬಡ್ಡಿ |
| :--- | :--- | :--- |
| **ಮೊದಲನೇ ವರ್ಷ** | | |
| ಅಸಲು | ₹100.00 | ₹100.00 |
| 10% ಬಡ್ಡಿ | ₹10.00 | ₹10.00 |
| ವರ್ಷದ ಕೊನೆಯಲ್ಲಿನ ಮೊತ್ತ | ₹110.00 | ₹110.00 |
| **ಎರಡನೇ ವರ್ಷ** | | |
| ಅಸಲು | ₹100.00 | ₹110.00 |
| 10% ಬಡ್ಡಿ | ₹10.00 | ₹11.00 |
| ವರ್ಷದ ಕೊನೆಯಲ್ಲಿನ ಮೊತ್ತ | ₹(110+10) = ₹120.00 | ₹121.00 |
| **ಮೂರನೇ ವರ್ಷ** | | |

3 ವರ್ಷದಲ್ಲಿ ಗಮನಿಸಿ.
ಸರಳಬಡ್ಡಿಯ ಪ್ರಕಾರ ಗಳಿಸಿದ ಬಡ್ಡಿ = ₹(130 - 100) = ₹30 ಆದರೆ,
ಚಕ್ರಬಡ್ಡಿಯ ಪ್ರಕಾರಗಳಿಸಿದ ಬಡ್ಡಿ = ₹(133.10 - 100) = ₹33.10
ಅಸಲು, ಸರಳಬಡ್ಡಿಯ ಪ್ರಕಾರ ಯಾವಾಗಲೂ ಒಂದೇ ಆಗಿರುವುದು ಮತ್ತು ಚಕ್ರಬಡ್ಡಿಯ ಪ್ರಕಾರ ವರ್ಷದಿಂದ ವರ್ಷಕ್ಕೆ ಬದಲಾಗುವುದನ್ನು ಗಮನಿಸಿ.

## 7.5 ಚಕ್ರಬಡ್ಡಿಯ ಸೂತ್ರವನ್ನು ಕಂಡುಹಿಡಿಯುವುದು

ಝುಬೇದಾಳು ಶಿಕ್ಷಕರಲ್ಲಿ "ಚಕ್ರಬಡ್ಡಿಯನ್ನು ಕಂಡುಹಿಡಿಯಲು ಇದಕ್ಕೂ ಸುಲಭವಾದ ವಿಧಾನವಿದೆಯೇ?" ಎಂದು ಕೇಳಿದಳು. ಅದಕ್ಕೆ ಶಿಕ್ಷಕರು "ಚಕ್ರಬಡ್ಡಿಯನ್ನು ಕಂಡುಹಿಡಿಯಲು ಸುಲಭವಾದ ವಿಧಾನವಿದೆ. ಅದನ್ನು ಕಂಡುಹಿಡಿಯಲು ಪ್ರಯತ್ನಿಸೋಣ" ಎಂದು ಹೇಳಿದರು.
P₁ ಮೊತ್ತದ ಮೇಲೆ ವಾರ್ಷಿಕ R% ಬಡ್ಡಿದರದಲ್ಲಿ ಪ್ರತಿವರ್ಷ ಚಕ್ರಬಡ್ಡಿಯನ್ನು ಲೆಕ್ಕ ಮಾಡಲಾಗಿದೆ ಎಂದು ತಿಳಿಯೋಣ.
P₁ = ₹5000 ಮತ್ತು R = 5% ಎಂದಿರಲಿ. ಈಗ ಮೇಲೆ ತಿಳಿಸಿದ ಹಂತಗಳ ಪ್ರಕಾರ,

1. $SI_1 = ₹ \frac{5000 \times 5 \times 1}{100}$ ಅಥವಾ $SI_1 = \frac{P_1 \times R \times 1}{100}$
ಆದ್ದರಿಂದ, $A_1 = ₹5000 + \frac{5000 \times 5 \times 1}{100}$ ಅಥವಾ $A_1 = P_1 + SI_1 = P_1 + \frac{P_1 R}{100}$
$= ₹ 5000 \left[1 + \frac{5}{100}\right] = P_2$ ಅಥವಾ $= P_1 \left[1 + \frac{R}{100}\right] = P_2$

2. $SI_2 = ₹ 5000 \left[1 + \frac{5}{100}\right] \times \frac{5 \times 1}{100}$ ಅಥವಾ $SI_2 = \frac{P_2 \times R \times 1}{100}$
$= \frac{5000 \times 5}{100} \left[1 + \frac{5}{100}\right]$
$= P_1 \left[1 + \frac{R}{100}\right] \times \frac{R}{100} = \frac{P_1 R}{100} \left[1 + \frac{R}{100}\right]$
$A_2 = P_2 + SI_2$
$= ₹ 5000 \left[1 + \frac{5}{100}\right] + ₹ \frac{5000 \times 5}{100} \left[1 + \frac{5}{100}\right]$
$= ₹ 5000 \left[1 + \frac{5}{100}\right] \left[1 + \frac{5}{100}\right]$
$= ₹ 5000 \left[1 + \frac{5}{100}\right]^2 = P_3$
$A_2 = P_1 \left[1 + \frac{R}{100}\right] + P_1 \frac{R}{100} \left[1 + \frac{R}{100}\right]$
$= P_1 \left[1 + \frac{R}{100}\right] \left[1 + \frac{R}{100}\right]$
$= P_1 \left[1 + \frac{R}{100}\right]^2 = P_3$

ಇದೇ ರೀತಿಯಲ್ಲಿ ಮುಂದುವರಿದಾಗ 'n' ವರ್ಷಗಳ ಕೊನೆಯಲ್ಲಿನ ಮೊತ್ತವು
$A_n = P \left[1 + \frac{R}{100}\right]^n$
ಅಥವಾ ನಾವು $A = P \left[1 + \frac{R}{100}\right]^n$ ಎಂದು ಹೇಳಬಹುದು.
ಆಗ ಝುಬೇದಾಳು, ಇದನ್ನು ಬಳಸಿ ನಾವು n ವರ್ಷಗಳ ಕೊನೆಯಲ್ಲಿ ಪಾವತಿಸಬೇಕಾಗುವ ಮೊತ್ತವನ್ನು ಕಂಡುಹಿಡಿಯುವ ಸೂತ್ರವನ್ನು ಮಾತ್ರ ಪಡೆದಿದ್ದೇವೆ, ವಿನಃ ಚಕ್ರಬಡ್ಡಿಯ ಸೂತ್ರವನ್ನಲ್ಲ ಎನ್ನುತ್ತಾಳೆ. ಕೂಡಲೆ ಅರುಣಾಳು ನಮಗೆ CI = A - P ಎಂಬುದು ತಿಳಿದಿದೆ. ಹೀಗಾಗಿ ನಾವು ಸುಲಭವಾಗಿ ಚಕ್ರಬಡ್ಡಿಯನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು ಎನ್ನುತ್ತಾಳೆ.

--- EXAMPLE START ---
Source: Class 8, Example 8 (page 34)
Problem: ವಾರ್ಷಿಕ 10% ಬಡ್ಡಿದರದಲ್ಲಿ ಪ್ರತಿವರ್ಷ ಚಕ್ರಬಡ್ಡಿ ಲೆಕ್ಕ ಮಾಡಿದರೆ ₹12,600 ಗಳಿಗೆ 2 ವರ್ಷಗಳಿಗೆ ಆಗುವ ಚಕ್ರಬಡ್ಡಿಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
Solution:
ಸೂತ್ರ: $A = P \left[1 + \frac{R}{100}\right]^n$
ಇಲ್ಲಿ ಅಸಲು P = ₹12,600, ದರ (R) = 10, ವರ್ಷಗಳ ಸಂಖ್ಯೆ (n) = 2
ಆದ್ದರಿಂದ $A = 12600 \left[1 + \frac{10}{100}\right]^2$
$= ₹12600 \left[\frac{11}{10}\right]^2$
$= ₹12600 \times \frac{11}{10} \times \frac{11}{10} = ₹15246$
CI = A - P = ₹15246 - ₹12600 = ₹2646
--- EXAMPLE END ---

## 7.6 ಚಕ್ರಬಡ್ಡಿಯ ಸೂತ್ರದ ಅನ್ವಯಗಳು:

ಚಕ್ರಬಡ್ಡಿಯ ಮೊತ್ತವನ್ನು ಕಂಡುಹಿಡಿಯುವ ಸೂತ್ರವನ್ನು ನಾವು ಬಳಸಬಹುದಾದ ಕೆಲವು ಸಂದರ್ಭಗಳಿವೆ. ಅವುಗಳಲ್ಲಿ ಕೆಲವು ಹೀಗಿವೆ.
(i) ಜನಸಂಖ್ಯೆಯಲ್ಲಿನ ಏರಿಕೆ (ಅಥವಾ ಇಳಿಕೆ).
(ii) ಬ್ಯಾಕ್ಟಿರಿಯಾಗಳ ಬೆಳವಣಿಗೆಯ ವೇಗ ತಿಳಿದಿದ್ದರೆ ಅವುಗಳ ಬೆಳವಣಿಗೆ.
(iii) ಒಂದು ವಸ್ತುವಿನ ಬೆಲೆ ಬೇರೆ ಬೇರೆ ಕಾಲಾವಧಿಯಲ್ಲಿ ಹೆಚ್ಚು-ಕಡಿಮೆಯಾದರೆ ಅದರ ಮೌಲ್ಯ.

--- EXAMPLE START ---
Source: Class 8, Example 9 (page 34)
Problem: ಒಂದು ನಗರದ ಜನಸಂಖ್ಯೆಯು 1997ರಲ್ಲಿ 20,000 ಆಗಿತ್ತು. ಅದು ವಾರ್ಷಿಕ 5% ವೇಗದಲ್ಲಿ ಹೆಚ್ಚಾಯಿತು. 2000ದ ಕೊನೆಯಲ್ಲಿ ಜನಸಂಖ್ಯೆಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
Solution:
ಪ್ರತಿ ವರ್ಷವೂ ಜನಸಂಖ್ಯೆಯಲ್ಲಿ 5% ಏರಿಕೆ ಇದೆ. ಹೀಗೆ, ಆದ್ದರಿಂದ ಪ್ರತಿವರ್ಷವೂ ಜನಸಂಖ್ಯೆಯು ಬದಲಾಗುತ್ತಿದೆ. ಆದ್ದರಿಂದ ಅದು ಚಕ್ರಬಡ್ಡಿಯ ರೂಪದಲ್ಲೇ ಏರುತ್ತಿದೆ ಎಂದು ಹೇಳಬಹುದು.
1998ರ ಪ್ರಾರಂಭದಲ್ಲಿ ಜನಸಂಖ್ಯೆ = 20,000
(ನಾವು ಇದನ್ನು 1ನೇ ವರ್ಷದ ಅಸಲು ಎಂದು ಪರಿಗಣಿಸೋಣ).
5% ಹೆಚ್ಚಳ = $\frac{5}{100} \times 20,000 = 1000$
1999ರ ಪ್ರಾರಂಭದಲ್ಲಿ ಜನಸಂಖ್ಯೆ = 20,000 + 1000 = 21,000
5% ಹೆಚ್ಚಳ = $\frac{5}{100} \times 21,000 = 1050$
2000ರ ಪ್ರಾರಂಭದಲ್ಲಿ ಜನಸಂಖ್ಯೆ = 21,000 + 1050 = 22,050
5% ಹೆಚ್ಚಳ = $\frac{5}{100} \times 22,050 = 1102.5$
2000ದ ಕೊನೆಯಲ್ಲಿ ಜನಸಂಖ್ಯೆ = 22050 + 1102.5 = 23152.5
ಅಥವಾ,
2000ದ ಕೊನೆಯಲ್ಲಿ ಜನಸಂಖ್ಯೆ = $20000 \left[1 + \frac{5}{100}\right]^3$
$= 20000 \times \frac{21}{20} \times \frac{21}{20} \times \frac{21}{20}$
$= 23152.5$
ಹೀಗೆ, ಅಂದಾಜಿಸಿದ ಜನಸಂಖ್ಯೆ = 23153
ಆಗ ಅರುಣಾಳು ಜನಸಂಖ್ಯೆ ಕಡಿಮೆಯಾಗುತ್ತಿದ್ದರೆ ಏನು ಮಾಡುವುದು ಎಂದು ಕೇಳುತ್ತಾಳೆ. ಅದಕ್ಕೆ ಶಿಕ್ಷಕರು ಕೆಳಗಿನ ಉದಾಹರಣೆಯನ್ನು ತೆಗೆದುಕೊಂಡರು.
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 8, Example 10 (page 35)
Problem: ಟಿವಿಯೊಂದನ್ನು ₹21000 ಕ್ಕೆ ಖರೀದಿಸಲಾಯಿತು. ಒಂದು ವರ್ಷದ ನಂತರ ಟಿವಿಯ ಮೌಲ್ಯವು 5% ದರದಲ್ಲಿ ಸವಕಳಿಗೊಂಡರೆ (ಸವಕಳಿ ಎಂದರೆ ವಸ್ತುವಿನ ಬಳಕೆ ಮತ್ತು ಸಮಯದ ಕಾರಣದಿಂದಾಗಿ ಅವರ ಮೌಲ್ಯದಲ್ಲಾಗುವ ಇಳಿಕೆ). ಒಂದು ವರ್ಷದ ಕೊನೆಯಲ್ಲಿ ಟಿವಿಯ ಮೌಲ್ಯವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
Solution:
ಅಸಲು = ₹21,000
ಇಳಿಕೆ = ₹21000 ಕ್ಕೆ ಪ್ರತಿವರ್ಷ 5% ದರದಲ್ಲಿ
= ₹ $\frac{21000 \times 5 \times 1}{100} = ₹1050$
ಒಂದು ವರ್ಷದ ಕೊನೆಯಲ್ಲಿನ ಮೌಲ್ಯ = ₹21000 – ₹1050 = ₹19,950 ಪರ್ಯಾಯವಾಗಿ, ನಾವೀಗ ಇದನ್ನು ನೇರವಾಗಿ ಈ ಕೆಳಗಿನಂತೆ ಪಡೆಯಬಹುದು:
ಒಂದು ವರ್ಷದ ಕೊನೆಯಲ್ಲಿನ ಮೌಲ್ಯ = ₹ $21000 \left[1 - \frac{5}{100}\right]$
$= ₹21000 \times \frac{19}{20} = ₹19,950$
--- EXAMPLE END ---

### ಸಾರಾಂಶ

1. ರಿಯಾಯಿತಿ ಎಂಬುದು ನಮೂದಿತ ಬೆಲೆಯ ಮೇಲಿನ ಕಡಿತವಾಗಿದೆ.
ರಿಯಾಯಿತಿ = ನಮೂದಿತ ಬೆಲೆ - ಮಾರಾಟ ಬೆಲೆ.
2. ರಿಯಾಯಿತಿಯನ್ನು ರಿಯಾಯಿತಿ ದರವನ್ನು ಕೊಟ್ಟಾಗ ಕಂಡುಹಿಡಿಯಬಹುದು.
ರಿಯಾಯಿತಿ = ನಮೂದಿತ ಬೆಲೆಯ ಮೇಲಿನ ಶೇಕಡಾ ರಿಯಾಯಿತಿ.
3. ವಸ್ತುವನ್ನು ಕೊಂಡ ನಂತರ ಮಾಡಲಾಗುವ ಹೆಚ್ಚಿನ ಖರ್ಚನ್ನು ಅಸಲು ಬೆಲೆ (CP)ಗೆ ಸೇರಿಸಲಾಗಿರುತ್ತದೆ ಮತ್ತು ಇದನ್ನು ಹೆಚ್ಚುವರಿ ಖರ್ಚು ಎನ್ನುವರು.
ಅಸಲು ಬೆಲೆ (CP) = ಕೊಂಡಬೆಲೆ + ಹೆಚ್ಚುವರಿ ಖರ್ಚು
4. ಮಾರಾಟ ತೆರಿಗೆಯನ್ನು ವಸ್ತುವಿನ ಮಾರಾಟದ ಮೇಲೆ ಸರ್ಕಾರವು ವಿಧಿಸುತ್ತದೆ ಮತ್ತು ಬಿಲ್‌ನ ಮೊತ್ತಕ್ಕೆ ಸೇರಿಸಲಾಗುತ್ತದೆ.
ಮಾರಾಟ ತೆರಿಗೆ = ಬಿಲ್‌ನ ಮೊತ್ತದ ಮೇಲಿನ ಶೇಕಡಾ ತೆರಿಗೆ
5. GST ಎಂದರೆ ಸರಕು ಮತ್ತು ಸೇವಾ ತೆರಿಗೆ, ಇದನ್ನು ಸರಕುಗಳ ಸರಬರಾಜು ಅಥವಾ ಸೇವೆ ಅಥವಾ ಇವೆರಡರ ಮೇಲೂ ವಿಧಿಸಲಾಗುತ್ತದೆ.
6. ಚಕ್ರಬಡ್ಡಿಯು ಹಿಂದಿನ ವರ್ಷದ ಮೊತ್ತದ ಮೇಲೆ ಲೆಕ್ಕ ಮಾಡಿದ ಬಡ್ಡಿಯಾಗಿರುವುದು. (A = P + I)
