[ಮೂಲ_ಅವಧಾರಣೆ_1]
concept_id: ALG-EXP-2_concept_1
concept_name: ಬೀಜೋಕ್ತಿಗಳ ಮೂಲಭೂತ ಅಂಶಗಳು ಮತ್ತು ವಿಧಗಳು (Basics and Types of Algebraic Expressions)
domain: ಬೀಜಗಣಿತ (Algebra)

  [ಸಾರಾಂಶ]
  ಬೀಜೋಕ್ತಿಗಳು ಸ್ಥಿರಾಂಕಗಳು ಮತ್ತು ಚರಾಕ್ಷರಗಳ ಸಂಯೋಜನೆಯಾಗಿವೆ. ಅವುಗಳನ್ನು ಪದಗಳಾಗಿ ವಿಂಗಡಿಸಬಹುದು, ಮತ್ತು ಪ್ರತಿ ಪದವು ಅಪವರ್ತನಗಳನ್ನು ಹೊಂದಿರುತ್ತದೆ. ಪದಗಳ ಸಂಖ್ಯೆಯನ್ನು ಆಧರಿಸಿ ಬೀಜೋಕ್ತಿಗಳನ್ನು ಏಕಪದೋಕ್ತಿ, ದ್ವಿಪದೋಕ್ತಿ, ತ್ರಿಪದೋಕ್ತಿ ಮತ್ತು ಬಹುಪದೋಕ್ತಿಗಳಾಗಿ ವರ್ಗೀಕರಿಸಲಾಗುತ್ತದೆ. ಒಂದೇ ರೀತಿಯ ಚರಾಕ್ಷರ ಭಾಗಗಳನ್ನು ಹೊಂದಿರುವ ಪದಗಳನ್ನು ಸಜಾತಿ ಪದಗಳು ಎನ್ನುತ್ತಾರೆ.

  [ವ್ಯಾಖ್ಯೆ]
  1.  **ಬೀಜೋಕ್ತಿ (Algebraic Expression):** ಚರಾಕ್ಷರಗಳು (variables) ಮತ್ತು ಸ್ಥಿರಾಂಕಗಳನ್ನು (constants) ಸಂಕಲನ, ವ್ಯವಕಲನ, ಗುಣಾಕಾರ, ಭಾಗಾಕಾರದಂತಹ ಗಣಿತದ ಕ್ರಿಯೆಗಳನ್ನು ಬಳಸಿ ಸಂಯೋಜಿಸಿದಾಗ ಬೀಜೋಕ್ತಿ ಉಂಟಾಗುತ್ತದೆ. ಉದಾ: $7x^2 - 4x + 5$.
  2.  **ಪದ (Term):** ಬೀಜೋಕ್ತಿಯಲ್ಲಿ ಸಂಕಲನ ಅಥವಾ ವ್ಯವಕಲನ ಚಿಹ್ನೆಗಳಿಂದ ಬೇರ್ಪಡಿಸಿದ ಭಾಗಗಳನ್ನು ಪದಗಳೆಂದು ಕರೆಯುತ್ತಾರೆ. ಉದಾ: $7x^2$, $-4x$ ಮತ್ತು $5$ ಇವು $7x^2 - 4x + 5$ ರ ಪದಗಳು.
  3.  **ಅಪವರ್ತನ (Factor):** ಒಂದು ಪದವನ್ನು ರೂಪಿಸುವ ಸಂಖ್ಯೆಗಳು ಮತ್ತು ಚರಾಕ್ಷರಗಳ ಗುಣಲಬ್ಧವನ್ನು ಅಪವರ್ತನಗಳು ಎನ್ನುತ್ತಾರೆ. ಉದಾ: $7x^2$ ಪದದ ಅಪವರ್ತನಗಳು $7, x, x$.
  4.  **ಸಹಗುಣಕ (Coefficient):** ಒಂದು ಪದದಲ್ಲಿ ಚರಾಕ್ಷರದ ಜೊತೆ ಗುಣಿಸಿರುವ ಸಂಖ್ಯಾತ್ಮಕ ಅಪವರ್ತನವನ್ನು ಆ ಪದದ ಸಂಖ್ಯಾ ಸಹಗುಣಕ ಎನ್ನುತ್ತಾರೆ. ಉದಾ: $-4x$ ಪದದಲ್ಲಿ, ಸಹಗುಣಕ $-4$.
  5.  **ಏಕಪದೋಕ್ತಿ (Monomial):** ಒಂದೇ ಒಂದು ಪದವನ್ನು ಹೊಂದಿರುವ ಬೀಜೋಕ್ತಿ. ಉದಾ: $7xy$.
  6.  **ದ್ವಿಪದೋಕ್ತಿ (Binomial):** ಎರಡು ಪದಗಳನ್ನು ಹೊಂದಿರುವ ಬೀಜೋಕ್ತಿ. ಉದಾ: $5y+2$.
  7.  **ತ್ರಿಪದೋಕ್ತಿ (Trinomial):** ಮೂರು ಪದಗಳನ್ನು ಹೊಂದಿರುವ ಬೀಜೋಕ್ತಿ. ಉದಾ: $4p^2 + 5p + 7$.
  8.  **ಬಹುಪದೋಕ್ತಿ (Polynomial):** ಒಂದು ಅಥವಾ ಅದಕ್ಕಿಂತ ಹೆಚ್ಚು ಪದಗಳನ್ನು ಹೊಂದಿರುವ ಬೀಜೋಕ್ತಿ, ಇದರಲ್ಲಿ ಚರಾಕ್ಷರಗಳ ಘಾತಗಳು ಋಣಾತ್ಮಕವಲ್ಲದ ಪೂರ್ಣಾಂಕಗಳಾಗಿರುತ್ತವೆ. ಏಕಪದೋಕ್ತಿ, ದ್ವಿಪದೋಕ್ತಿ ಮತ್ತು ತ್ರಿಪದೋಕ್ತಿಗಳು ಬಹುಪದೋಕ್ತಿಗಳ ವಿಶೇಷ ರೂಪಗಳಾಗಿವೆ.
  9.  **ಸಜಾತಿ ಪದಗಳು (Like Terms):** ಒಂದೇ ರೀತಿಯ ಬೀಜಗಣಿತೀಯ ಅಪವರ್ತನಗಳನ್ನು (ಚರಾಕ್ಷರ ಭಾಗ) ಹೊಂದಿರುವ ಪದಗಳು. ಅವುಗಳ ಸಂಖ್ಯಾ ಸಹಗುಣಕಗಳು ಬೇರೆಯಾಗಿರಬಹುದು. ಉದಾ: $7xy$ ಮತ್ತು $-2xy$.
  10. **ವಿಜಾತಿ ಪದಗಳು (Unlike Terms):** ವಿಭಿನ್ನ ಬೀಜಗಣಿತೀಯ ಅಪವರ್ತನಗಳನ್ನು ಹೊಂದಿರುವ ಪದಗಳು. ಉದಾ: $7xy$ ಮತ್ತು $5yz$.

  [ವಿಶ್ಲೇಷಣೆ]
  - ಈ ಅವಧಾರಣೆ ಏಕೆ ಮುಖ್ಯ: ಬೀಜಗಣಿತದ ಮುಂದಿನ ಎಲ್ಲಾ ಪರಿಕಲ್ಪನೆಗಳಿಗೆ ಇದು ಅಡಿಪಾಯ. ಪದ, ಸಹಗುಣಕ, ಸಜಾತಿ ಪದಗಳನ್ನು ಗುರುತಿಸುವುದು ಸಂಕಲನ, ವ್ಯವಕಲನ ಮತ್ತು ಗುಣಾಕಾರ ಕ್ರಿಯೆಗಳಿಗೆ ಅತ್ಯಗತ್ಯ.
  - Common mistakes: ವಿದ್ಯಾರ್ಥಿಗಳು ಸಜಾತಿ ಮತ್ತು ವಿಜಾತಿ ಪದಗಳನ್ನು ಗುರುತಿಸುವಲ್ಲಿ ತಪ್ಪು ಮಾಡುತ್ತಾರೆ. ಉದಾಹರಣೆಗೆ, $x^2y$ ಮತ್ತು $xy^2$ ನ್ನು ಸಜಾತಿ ಪದಗಳೆಂದು ತಪ್ಪಾಗಿ ಭಾವಿಸಬಹುದು.

  [ಸಂಬಂಧ]
  - ಪೂರ್ವಾಪೇಕ್ಷಿತ ಅವಧಾರಣೆಗಳು: ಸಂಖ್ಯೆಗಳ ಮೇಲಿನ ಮೂಲಭೂತ ಕ್ರಿಯೆಗಳು, ಘಾತಾಂಕಗಳ ನಿಯಮಗಳು.
  - ಮುಂದಿನ ಅವಧಾರಣೆಗಳೊಂದಿಗೆ ಸಂಪರ್ಕ: ಈ ವ್ಯಾಖ್ಯೆಗಳು ಬೀಜೋಕ್ತಿಗಳ ಸಂಕಲನ, ವ್ಯವಕಲನ, ಗುಣಾಕಾರ ಮತ್ತು ಭಾಗಾಕಾರಕ್ಕೆ ಆಧಾರವಾಗಿವೆ.

[ಮೂಲ_ಅವಧಾರಣೆ_2]
concept_id: ALG-EXP-2_concept_2
concept_name: ಬೀಜೋಕ್ತಿಗಳ ಸಂಕಲನ ಮತ್ತು ವ್ಯವಕಲನ (Addition and Subtraction of Algebraic Expressions)
domain: ಬೀಜಗಣಿತ (Algebra)

  [ಸಾರಾಂಶ]
  ಬೀಜೋಕ್ತಿಗಳ ಸಂಕಲನ ಮತ್ತು ವ್ಯವಕಲನವು ಸಜಾತಿ ಪದಗಳನ್ನು ಒಟ್ಟುಗೂಡಿಸುವ ಪ್ರಕ್ರಿಯೆಯಾಗಿದೆ. ಸಂಕಲನದಲ್ಲಿ ಸಜಾತಿ ಪದಗಳ ಸಹಗುಣಕಗಳನ್ನು ಕೂಡಿಸಲಾಗುತ್ತದೆ. ವ್ಯವಕಲನದಲ್ಲಿ, ಕಳೆಯಬೇಕಾದ ಬೀಜೋಕ್ತಿಯ ಪ್ರತಿಯೊಂದು ಪದದ ಚಿಹ್ನೆಯನ್ನು ಬದಲಾಯಿಸಿ ನಂತರ ಕೂಡಿಸಲಾಗುತ್ತದೆ.

  [ಅಂಶಪಟ್ಟಿ / Properties]
  1.  **ಸಂಕಲನದ ನಿಯಮ:** ಕೇವಲ ಸಜಾತಿ ಪದಗಳನ್ನು ಮಾತ್ರ ಕೂಡಿಸಬಹುದು. ಸಜಾತಿ ಪದಗಳನ್ನು ಕೂಡುವಾಗ, ಅವುಗಳ ಸಂಖ್ಯಾ ಸಹಗುಣಕಗಳನ್ನು ಕೂಡಿಸಿ, ಚರಾಕ್ಷರ ಭಾಗವನ್ನು ಹಾಗೆಯೇ ಇಡಲಾಗುತ್ತದೆ. ವಿಜಾತಿ ಪದಗಳನ್ನು ಹಾಗೆಯೇ ಉಳಿಸಿಕೊಳ್ಳಲಾಗುತ್ತದೆ.
  2.  **ವ್ಯವಕಲನದ ನಿಯಮ:** ಒಂದು ಬೀಜೋಕ್ತಿಯನ್ನು ಇನ್ನೊಂದರಿಂದ ಕಳೆಯುವುದು ಎಂದರೆ, ಕಳೆಯಬೇಕಾದ ಬೀಜೋಕ್ತಿಯ ಸಂಕಲನದ ವಿಲೋಮವನ್ನು (additive inverse) ಕೂಡುವುದು. ಅಂದರೆ, ಕಳೆಯಬೇಕಾದ ಬೀಜೋಕ್ತಿಯ ಪ್ರತಿಯೊಂದು ಪದದ ಚಿಹ್ನೆಯನ್ನು (+ ಯನ್ನು - ಗೆ, - ಯನ್ನು + ಗೆ) ಬದಲಾಯಿಸಿ, ನಂತರ ಸಂಕಲನ ಮಾಡಬೇಕು.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  [example_id]
  example_id: "ALG-EXP-2_example_1"
  difficulty: "intermediate"
  problem: "ಕೂಡಿಸಿ: $7xy + 5yz – 3zx$, $4yz + 9zx – 4y$, ಮತ್ತು $-3xz + 5x – 2xy$."
  steps:
    - step: 1
      action: "ಸಜಾತಿ ಪದಗಳು ಒಂದರ ಕೆಳಗೆ ಒಂದು ಬರುವಂತೆ ಬೀಜೋಕ್ತಿಗಳನ್ನು ಅಡ್ಡಸಾಲುಗಳಲ್ಲಿ ಬರೆಯುವುದು."
      justification: "ಸಜಾತಿ ಪದಗಳನ್ನು ಸುಲಭವಾಗಿ ಗುರುತಿಸಿ ಕೂಡಿಸಲು ಈ ವಿಧಾನ ಸಹಕಾರಿ. ಇಲ್ಲಿ $xz$ ಮತ್ತು $zx$ ಒಂದೇ ಆಗಿರುವುದರಿಂದ ಅವು ಸಜಾತಿ ಪದಗಳು."
      calculation: |
        ```
          7xy + 5yz - 3zx
                + 4yz + 9zx - 4y
        - 2xy       - 3zx + 5x
        ```
    - step: 2
      action: "ಪ್ರತಿ ಕಾಲಂನಲ್ಲಿರುವ ಸಜಾತಿ ಪದಗಳನ್ನು ಕೂಡಿಸುವುದು."
      justification: "ಸಜಾತಿ ಪದಗಳ ಸಹಗುಣಕಗಳನ್ನು ಕೂಡಿಸಬೇಕು."
      calculation: |
        - xy ಪದಗಳು: $7xy - 2xy = (7-2)xy = 5xy$
        - yz ಪದಗಳು: $5yz + 4yz = (5+4)yz = 9yz$
        - zx ಪದಗಳು: $-3zx + 9zx - 3zx = (-3+9-3)zx = 3zx$
        - y ಪದ: $-4y$ (ಇದಕ್ಕೆ ಸಜಾತಿ ಪದವಿಲ್ಲ)
        - x ಪದ: $+5x$ (ಇದಕ್ಕೆ ಸಜಾತಿ ಪದವಿಲ್ಲ)
    - step: 3
      action: "ಎಲ್ಲಾ ಪದಗಳನ್ನು ಒಟ್ಟುಗೂಡಿಸಿ ಅಂತಿಮ ಉತ್ತರವನ್ನು ಬರೆಯುವುದು."
      justification: "ಸಂಕಲನದ ಫಲಿತಾಂಶವನ್ನು ಬರೆಯುವುದು."
      calculation: |
        ```
              7xy + 5yz - 3zx
          +         4yz + 9zx - 4y
          +  -2xy       - 3zx + 5x
          -----------------------------
              5xy + 9yz + 3zx + 5x - 4y
        ```
  result: "ಬೀಜೋಕ್ತಿಗಳ ಮೊತ್ತವು $5xy + 9yz + 3zx + 5x - 4y$ ಆಗಿದೆ."
  common_trap: "$-4y$ ಮತ್ತು $5x$ ಪದಗಳಿಗೆ ಸಜಾತಿ ಪದಗಳಿಲ್ಲದ ಕಾರಣ ಅವುಗಳನ್ನು ಹಾಗೆಯೇ ಉತ್ತರದಲ್ಲಿ ಸೇರಿಸಬೇಕು. ಇದನ್ನು ಮರೆಯಬಾರದು."

  steps:
    - step: 1
      action: "ಯಾವುದರಿಂದ ಕಳೆಯಬೇಕೋ ಆ ಬೀಜೋಕ್ತಿಯನ್ನು ಮೇಲೆ ಮತ್ತು ಕಳೆಯಬೇಕಾದ ಬೀಜೋಕ್ತಿಯನ್ನು ಕೆಳಗೆ, ಸಜಾತಿ ಪದಗಳು ಒಂದರ ಕೆಳಗೆ ಬರುವಂತೆ ಬರೆಯುವುದು."
      justification: "ವ್ಯವಕಲನ ಕ್ರಿಯೆಗೆ ಸಿದ್ಧತೆ."
      calculation: |
        ```
          7x² - 4xy + 8y² + 5x - 3y
          5x²       - 4y²      + 6y - 3
        ```
    - step: 2
      action: "ಕಳೆಯಬೇಕಾದ ಬೀಜೋಕ್ತಿಯ (ಕೆಳಗಿನ ಸಾಲು) ಪ್ರತಿಯೊಂದು ಪದದ ಚಿಹ್ನೆಯನ್ನು ಬದಲಾಯಿಸುವುದು."
      justification: "ವ್ಯವಕಲನವು ಸಂಕಲನದ ವಿಲೋಮವನ್ನು ಕೂಡುವುದಕ್ಕೆ ಸಮ. ಆದ್ದರಿಂದ $+5x^2$ എന്നത് $-5x^2$ ಆಗುತ್ತದೆ, $-4y^2$ എന്നത് $+4y^2$ ಆಗುತ್ತದೆ, $+6y$ എന്നത് $-6y$ ಆಗುತ್ತದೆ ಮತ್ತು $-3$ എന്നത് $+3$ ಆಗುತ್ತದೆ."
      calculation: |
        ```
          7x² - 4xy + 8y² + 5x - 3y
          5x²       - 4y²      + 6y - 3
         (-)        (+)        (-)  (+)
        ```
    - step: 3
      action: "ಈಗ, ಬದಲಾದ ಚಿಹ್ನೆಗಳೊಂದಿಗೆ ಪದಗಳನ್ನು ಕೂಡಿಸುವುದು."
      justification: "ಚಿಹ್ನೆ ಬದಲಾಯಿಸಿದ ನಂತರ ಇದು ಸಂಕಲನ ಕ್ರಿಯೆಯಾಗುತ್ತದೆ."
      calculation: |
        - $x^2$ ಪದಗಳು: $7x^2 - 5x^2 = 2x^2$
        - $xy$ ಪದ: $-4xy$ (ಸಜಾತಿ ಪದವಿಲ್ಲ)
        - $y^2$ ಪದಗಳು: $8y^2 + 4y^2 = 12y^2$
        - $x$ ಪದ: $+5x$ (ಸಜಾತಿ ಪದವಿಲ್ಲ)
        - $y$ ಪದಗಳು: $-3y - 6y = -9y$
        - ಸ್ಥಿರಾಂಕಗಳು: $0 + 3 = 3$ (ಮೊದಲ ಬೀಜೋಕ್ತಿಯಲ್ಲಿ ಸ್ಥಿರಾಂಕ 0 ಎಂದು ಭಾವಿಸಬೇಕು)
    - step: 4
      action: "ಅಂತಿಮ ಉತ್ತರವನ್ನು ಬರೆಯುವುದು."
      justification: "ವ್ಯವಕಲನದ ಫಲಿತಾಂಶ."
      calculation: |
        ```
          7x² - 4xy + 8y² + 5x - 3y
        - (5x²       - 4y²      + 6y - 3)
        ------------------------------------
          2x² - 4xy + 12y² + 5x - 9y + 3
        ```
  result: "ವ್ಯತ್ಯಾಸವು $2x^2 - 4xy + 12y^2 + 5x - 9y + 3$ ಆಗಿದೆ."
  common_trap: "ವ್ಯವಕಲನ ಮಾಡುವಾಗ ಕಳೆಯಬೇಕಾದ ಬೀಜೋಕ್ತಿಯ *ಪ್ರತಿಯೊಂದು* ಪದದ ಚಿಹ್ನೆಯನ್ನು ಬದಲಾಯಿಸಬೇಕು. ಕೇವಲ ಮೊದಲ ಪದದ ಚಿಹ್ನೆ ಬದಲಾಯಿಸಿದರೆ ಉತ್ತರ ತಪ್ಪಾಗುತ್ತದೆ."

  [ವಿಶ್ಲೇಷಣೆ]
  - ಈ ಅವಧಾರಣೆ ಏಕೆ ಮುಖ್ಯ: ಬೀಜಗಣಿತದ ಸಮೀಕರಣಗಳನ್ನು ಬಿಡಿಸಲು ಮತ್ತು ಸಂಕೀರ್ಣ ಬೀಜೋಕ್ತಿಗಳನ್ನು ಸರಳೀಕರಿಸಲು ಈ ಕ್ರಿಯೆಗಳು ಅತ್ಯಗತ್ಯ.
  - Boundary conditions: ಕೇವಲ ಸಜಾತಿ ಪದಗಳ ನಡುವೆ ಮಾತ್ರ ಈ ಕ್ರಿಯೆಗಳು ಸಾಧ್ಯ.
  - Common mistakes: ವ್ಯವಕಲನದಲ್ಲಿ ಚಿಹ್ನೆ ಬದಲಾಯಿಸುವುದನ್ನು ಮರೆಯುವುದು ಅಥವಾ ಎಲ್ಲಾ ಪದಗಳ ಚಿಹ್ನೆ ಬದಲಾಯಿಸದಿರುವುದು.

  [ಸಂಬಂಧ]
  - ಪೂರ್ವಾಪೇಕ್ಷಿತ ಅವಧಾರಣೆಗಳು: ಪೂರ್ಣಾಂಕಗಳ ಸಂಕಲನ ಮತ್ತು ವ್ಯವಕಲನ, ಸಜಾತಿ ಮತ್ತು ವಿಜಾತಿ ಪದಗಳನ್ನು ಗುರುತಿಸುವುದು.
  - ಮುಂದಿನ ಅವಧಾರಣೆಗಳೊಂದಿಗೆ ಸಂಪರ್ಕ: ಸಮೀಕರಣಗಳನ್ನು ಬಿಡಿಸುವಾಗ ಪದಗಳನ್ನು ಒಂದು ಕಡೆಯಿಂದ ಇನ್ನೊಂದು ಕಡೆಗೆ ವರ್ಗಾಯಿಸುವುದು ಈ ನಿಯಮಗಳನ್ನೇ ಆಧರಿಸಿದೆ.

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  - gpstr_weightage: high
  - MCQ ಸಾಧ್ಯತೆ: ಎರಡು ಬೀಜೋಕ್ತಿಗಳನ್ನು ನೀಡಿ ಅವುಗಳ ಮೊತ್ತ ಅಥವಾ ವ್ಯತ್ಯಾಸವನ್ನು ಕೇಳಬಹುದು.
  - 2-ಅಂಕ ಪ್ರಶ್ನೆ: ಸಂಕಲನ ಅಥವಾ ವ್ಯವಕಲನದ ಒಂದು ಲೆಕ್ಕವನ್ನು ಬಿಡಿಸಲು ಕೇಳಬಹುದು.
  - 5-ಅಂಕ ಪ್ರಶ್ನೆ: ಸಂಕೀರ್ಣವಾದ ಮೂರು ಅಥವಾ ಹೆಚ್ಚು ಬೀಜೋಕ್ತಿಗಳ ಸಂಕಲನ ಮತ್ತು ವ್ಯವಕಲನವನ್ನು ಒಳಗೊಂಡ ಪ್ರಶ್ನೆ ಬರಬಹುದು.
  - ಪ್ರಮುಖ trap: "A ಯನ್ನು B ಯಿಂದ ಕಳೆಯಿರಿ" (Subtract A from B) ಎಂದರೆ B - A ಮಾಡಬೇಕು, A - B ಅಲ್ಲ. ಈ ವಾಕ್ಯ ರಚನೆಯನ್ನು ಸರಿಯಾಗಿ ಅರ್ಥಮಾಡಿಕೊಳ್ಳಬೇಕು.

concept_name: ಏಕಪದೋಕ್ತಿಗಳ ಗುಣಾಕಾರ (Multiplication of Monomials)
domain: ಬೀಜಗಣಿತ (Algebra)

  [ಸಾರಾಂಶ]
  ಎರಡು ಅಥವಾ ಹೆಚ್ಚು ಏಕಪದೋಕ್ತಿಗಳನ್ನು ಗುಣಿಸುವಾಗ, ಅವುಗಳ ಸಂಖ್ಯಾ ಸಹಗುಣಕಗಳನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ಗುಣಿಸಲಾಗುತ್ತದೆ ಮತ್ತು ಚರಾಕ್ಷರ ಭಾಗಗಳನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ಗುಣಿಸಲಾಗುತ್ತದೆ. ಚರಾಕ್ಷರ ಭಾಗಗಳನ್ನು ಗುಣಿಸುವಾಗ ಘಾತಾಂಕಗಳ ನಿಯಮಗಳನ್ನು ಬಳಸಲಾಗುತ್ತದೆ.

  [ಅಂಶಪಟ್ಟಿ / Properties]
  1.  **ಗುಣಾಕಾರದ ನಿಯಮ:**
      (ಏಕಪದೋಕ್ತಿ 1) $\times$ (ಏಕಪದೋಕ್ತಿ 2) = (ಸಹಗುಣಕ 1 $\times$ ಸಹಗುಣಕ 2) $\times$ (ಚರಾಕ್ಷರ ಭಾಗ 1 $\times$ ಚರಾಕ್ಷರ ಭಾಗ 2)
  2.  **ಘಾತಾಂಕದ ನಿಯಮ:** ಒಂದೇ ಆಧಾರ (base) ಹೊಂದಿರುವ ಘಾತಗಳನ್ನು ಗುಣಿಸುವಾಗ, ಘಾತಸೂಚಿಗಳನ್ನು (exponents) ಕೂಡಿಸಲಾಗುತ್ತದೆ.
      $x^m \times x^n = x^{m+n}$

  ಒಂದು ಬಹುಪದೋಕ್ತಿಯನ್ನು ಏಕಪದೋಕ್ತಿಯಿಂದ ಗುಣಿಸಲು, ವಿತರಣಾ ನಿಯಮವನ್ನು (distributive law) ಬಳಸಲಾಗುತ್ತದೆ. ಅಂದರೆ, ಏಕಪದೋಕ್ತಿಯಿಂದ ಬಹುಪದೋಕ್ತಿಯ ಪ್ರತಿಯೊಂದು ಪದವನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ಗುಣಿಸಿ, ಬಂದ ಗುಣಲಬ್ಧಗಳನ್ನು ಕೂಡಿಸಲಾಗುತ್ತದೆ.

  [ಸೂತ್ರಗಳು]
  - formula_name: "Distributive Law over Addition"
    latex: "a(b+c) = ab + ac"
    condition: "a, b, c ಗಳು ಯಾವುದೇ ಬೀಜೋಕ್ತಿಗಳಾಗಿರಬಹುದು."
    used_in: "ಏಕಪದೋಕ್ತಿಯಿಂದ ಬಹುಪದೋಕ್ತಿಯನ್ನು ಗುಣಿಸಲು."
  - formula_name: "Distributive Law for Polynomials"
    latex: "p \times (q+r+s) = p \times q + p \times r + p \times s"
    condition: "p ಒಂದು ಏಕಪದೋಕ್ತಿ ಮತ್ತು (q+r+s) ಒಂದು ಬಹುಪದೋಕ್ತಿ."

  [example_id]
  example_id: "ALG-EXP-2_example_5"
  difficulty: "intermediate"
  problem: "ಬೀಜೋಕ್ತಿಗಳನ್ನು ಸಂಕ್ಷೇಪಿಸಿ ಮತ್ತು ಸೂಚಿಸಿದ ಪ್ರಕಾರ ಬೀಜೋಕ್ತಿಯ ಬೆಲೆ ಕಂಡುಹಿಡಿಯಿರಿ. (i) $x(x-3)+2$ ಗೆ $x=1$ ಆದಾಗ (ii) $3y(2y-7) - 3(y-4) - 63$ ಗೆ $y=-2$ ಆದಾಗ"
  steps:
    - step: 1
      action: "ಭಾಗ (i) ರ ಬೀಜೋಕ್ತಿಯನ್ನು ಸಂಕ್ಷೇಪಿಸುವುದು."
      justification: "ವಿತರಣಾ ನಿಯಮ ಬಳಸಿ $x$ ನಿಂದ $(x-3)$ ನ್ನು ಗುಣಿಸುವುದು."
      calculation: "$x(x-3)+2 = (x \times x) - (x \times 3) + 2 = x^2 - 3x + 2$"
    - step: 2
      action: "ಸಂಕ್ಷೇಪಿಸಿದ ಬೀಜೋಕ್ತಿಯಲ್ಲಿ $x=1$ ಆದೇಶಿಸುವುದು."
      justification: "x ನ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯಲು."
      calculation: "$(1)^2 - 3(1) + 2 = 1 - 3 + 2 = 3 - 3 = 0$"
    - step: 3
      action: "ಭಾಗ (ii) ರ ಬೀಜೋಕ್ತಿಯನ್ನು ಸಂಕ್ಷೇಪಿಸುವುದು."
      justification: "ವಿತರಣಾ ನಿಯಮವನ್ನು ಎರಡು ಕಡೆ ಬಳಸಿ, ನಂತರ ಸಜಾತಿ ಪದಗಳನ್ನು ಕೂಡಿಸುವುದು."
      calculation: |
        $3y(2y-7) - 3(y-4) - 63$
        $= (3y \times 2y) - (3y \times 7) - (3 \times y) - (3 \times -4) - 63$
        $= 6y^2 - 21y - 3y + 12 - 63$
        $= 6y^2 - (21+3)y + (12-63)$
        $= 6y^2 - 24y - 51$
    - step: 4
      action: "ಸಂಕ್ಷೇಪಿಸಿದ ಬೀಜೋಕ್ತಿಯಲ್ಲಿ $y=-2$ ಆದೇಶಿಸುವುದು."
      justification: "y ನ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯಲು."
      calculation: |
        $6(-2)^2 - 24(-2) - 51$
        $= 6(4) - (-48) - 51$
        $= 24 + 48 - 51$
        $= 72 - 51 = 21$
  result: "(i) 0 (ii) 21"
  common_trap: "ಸಂಖ್ಯೆಗಳನ್ನು ಆದೇಶಿಸುವಾಗ ಋಣ ಸಂಖ್ಯೆಗಳಿಗೆ ಆವರಣ (bracket) ಬಳಸಬೇಕು. ಉದಾ: $y=-2$ ಆದರೆ $y^2 = (-2)^2 = 4$. ಆವರಣವಿಲ್ಲದಿದ್ದರೆ $-2^2 = -4$ ಎಂದು ತಪ್ಪಾಗುವ ಸಾಧ್ಯತೆ ಇದೆ."

  problem: "$3pq(p-q)$ ನ್ನು $2pq(p+q)$ ರಿಂದ ಕಳೆಯಿರಿ."
  steps:
    - step: 1
      action: "ಎರಡೂ ಬೀಜೋಕ್ತಿಗಳನ್ನು ಸಂಕ್ಷೇಪಿಸುವುದು."
      justification: "ವಿತರಣಾ ನಿಯಮ ಬಳಸಿ ಗುಣಾಕಾರ ಮಾಡುವುದು."
      calculation: |
        - $2pq(p+q) = (2pq \times p) + (2pq \times q) = 2p^2q + 2pq^2$
        - $3pq(p-q) = (3pq \times p) - (3pq \times q) = 3p^2q - 3pq^2$
    - step: 2
      action: "ಮೊದಲ ಸಂಕ್ಷೇಪಿತ ಬೀಜೋಕ್ತಿಯಿಂದ ಎರಡನೆಯದನ್ನು ಕಳೆಯುವುದು."
      justification: "ಕಳೆಯಬೇಕಾದ ಬೀಜೋಕ್ತಿಯ ಚಿಹ್ನೆ ಬದಲಾಯಿಸಿ ಕೂಡಿಸುವುದು."
      calculation: |
        ```
          (2p²q + 2pq²) - (3p²q - 3pq²)
        = 2p²q + 2pq² - 3p²q + 3pq²
        ```
    - step: 3
      action: "ಸಜಾತಿ ಪದಗಳನ್ನು ಕೂಡಿಸುವುದು."
      justification: "ಅಂತಿಮ ಉತ್ತರವನ್ನು ಸರಳ ರೂಪದಲ್ಲಿ ಪಡೆಯಲು."
      calculation: |
        $(2p^2q - 3p^2q) + (2pq^2 + 3pq^2) = -p^2q + 5pq^2$
        ಅಥವಾ ಕಾಲಂ ವಿಧಾನದಲ್ಲಿ:
        ```
          2p²q + 2pq²
          3p²q - 3pq²
         (-)    (+)
        -----------------
         -p²q + 5pq²
        ```
  result: "$-p^2q + 5pq^2$"

  [ವಿಶ್ಲೇಷಣೆ]
  - ಈ ಅವಧಾರಣೆ ಏಕೆ ಮುಖ್ಯ: ಸಂಕೀರ್ಣ ಬೀಜೋಕ್ತಿಗಳನ್ನು ಸರಳೀಕರಿಸಲು ವಿತರಣಾ ನಿಯಮದ ಬಳಕೆ ಅತ್ಯಗತ್ಯ.
  - Common mistakes: ಏಕಪದೋಕ್ತಿಯಿಂದ ಬಹುಪದೋಕ್ತಿಯ ಮೊದಲ ಪದವನ್ನು ಮಾತ್ರ ಗುಣಿಸಿ ಉಳಿದವನ್ನು ಬಿಡುವುದು. ಚಿಹ್ನೆಗಳ ಗುಣಾಕಾರದಲ್ಲಿ ತಪ್ಪು ಮಾಡುವುದು.

  [ಸಂಬಂಧ]
  - ಪೂರ್ವಾಪೇಕ್ಷಿತ ಅವಧಾರಣೆಗಳು: ಏಕಪದೋಕ್ತಿಗಳ ಗುಣಾಕಾರ, ವಿತರಣಾ ನಿಯಮ.
  - ಮುಂದಿನ ಅವಧಾರಣೆಗಳೊಂದಿಗೆ ಸಂಪರ್ಕ: ಬಹುಪದೋಕ್ತಿಯಿಂದ ಬಹುಪದೋಕ್ತಿಯ ಗುಣಾಕಾರಕ್ಕೆ ಇದು ನೇರವಾದ ಮುಂದುವರಿಕೆ.

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  - gpstr_weightage: high
  - MCQ ಸಾಧ್ಯತೆ: ಒಂದು ಏಕಪದೋಕ್ತಿ ಮತ್ತು ದ್ವಿಪದೋಕ್ತಿ ನೀಡಿ ಗುಣಲಬ್ಧ ಕೇಳಬಹುದು.
  - 2-ಅಂಕ ಪ್ರಶ್ನೆ: ಸಂಕ್ಷೇಪಿಸಿ ಬೆಲೆ ಕಂಡುಹಿಡಿಯುವ ಲೆಕ್ಕಗಳು (ಉದಾ. 5).
  - 5-ಅಂಕ ಪ್ರಶ್ನೆ: ಸಂಕಲನ/ವ್ಯವಕಲನ ಮತ್ತು ಗುಣಾಕಾರ ಎರಡನ್ನೂ ಒಳಗೊಂಡ ಸಂಕೀರ್ಣ ಸರಳೀಕರಣದ ಪ್ರಶ್ನೆಗಳು.
  - ಪ್ರಮುಖ trap: $-3(y-4)$ ನ್ನು ಗುಣಿಸುವಾಗ $-3 \times y$ ಮತ್ತು $-3 \times (-4)$ ಮಾಡಬೇಕು. ಅಂದರೆ $-3y + 12$. ಅನೇಕರು $-3y - 12$ ಎಂದು ತಪ್ಪಾಗಿ ಬರೆಯುತ್ತಾರೆ.

[ಮೂಲ_ಅವಧಾರಣೆ_5]
concept_id: ALG-EXP-2_concept_5
concept_name: ಬಹುಪದೋಕ್ತಿಯಿಂದ ಬಹುಪದೋಕ್ತಿಯ ಗುಣಾಕಾರ (Multiplication of a Polynomial by a Polynomial)
domain: ಬೀಜಗಣಿತ (Algebra)

  [ಸಾರಾಂಶ]
  ಒಂದು ಬಹುಪದೋಕ್ತಿಯನ್ನು ಇನ್ನೊಂದು ಬಹುಪದೋಕ್ತಿಯಿಂದ ಗುಣಿಸಲು, ಮೊದಲ ಬಹುಪದೋಕ್ತಿಯ ಪ್ರತಿಯೊಂದು ಪದದಿಂದ ಎರಡನೇ ಬಹುಪದೋಕ್ತಿಯ ಪ್ರತಿಯೊಂದು ಪದವನ್ನು ಗುಣಿಸಬೇಕು. ಇದನ್ನು ವಿತರಣಾ ನಿಯಮವನ್ನು ಪುನರಾವರ್ತಿತವಾಗಿ ಬಳಸುವ ಮೂಲಕ ಮಾಡಲಾಗುತ್ತದೆ. ಅಂತಿಮವಾಗಿ, ದೊರೆತ ಗುಣಲಬ್ಧದಲ್ಲಿರುವ ಸಜಾತಿ ಪದಗಳನ್ನು ಕೂಡಿಸಿ ಸರಳೀಕರಿಸಬೇಕು.

  [ಅಂಶಪಟ್ಟಿ / Properties]
  1.  **ದ್ವಿಪದೋಕ್ತಿ $\times$ ದ್ವಿಪದೋಕ್ತಿ:**
      $(a+b)(c+d) = a(c+d) + b(c+d) = ac + ad + bc + bd$
  2.  **ದ್ವಿಪದೋಕ್ತಿ $\times$ ತ್ರಿಪದೋಕ್ತಿ:**
      $(a+b)(c+d+e) = a(c+d+e) + b(c+d+e) = ac+ad+ae+bc+bd+be$
  3.  **ಸಾಮಾನ್ಯ ನಿಯಮ:** ಗುಣಾಕಾರದ ನಂತರ, ಸಜಾತಿ ಪದಗಳನ್ನು ಸಂಯೋಜಿಸುವ ಮೊದಲು ಪದಗಳ ಸಂಖ್ಯೆಯು (ಮೊದಲ ಬಹುಪದೋಕ್ತಿಯ ಪದಗಳ ಸಂಖ್ಯೆ) $\times$ (ಎರಡನೇ ಬಹುಪದೋಕ್ತಿಯ ಪದಗಳ ಸಂಖ್ಯೆ) ಆಗಿರುತ್ತದೆ.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  [example_id]
  example_id: "ALG-EXP-2_example_8"
  difficulty: "intermediate"
  problem: "ಗುಣಿಸಿ: (i) $(x-4)$ ಮತ್ತು $(2x+3)$ (ii) $(x-y)$ ಮತ್ತು $(3x+5y)$"
  steps:
    - step: 1
      action: "ಭಾಗ (i) ನ್ನು ಬಿಡಿಸುವುದು: $(x-4)(2x+3)$."
      justification: "ವಿತರಣಾ ನಿಯಮ ಬಳಸಿ: ಮೊದಲ ದ್ವಿಪದೋಕ್ತಿಯ ಪ್ರತಿ ಪದದಿಂದ ಎರಡನೇ ದ್ವಿಪದೋಕ್ತಿಯನ್ನು ಗುಣಿಸುವುದು."
      calculation: "$(x-4)(2x+3) = x(2x+3) - 4(2x+3)$"
    - step: 2
      action: "ಮತ್ತೊಮ್ಮೆ ವಿತರಣಾ ನಿಯಮ ಬಳಸಿ ಆವರಣಗಳನ್ನು ಬಿಡಿಸುವುದು."
      justification: "ಏಕಪದೋಕ್ತಿಯಿಂದ ದ್ವಿಪದೋಕ್ತಿಯ ಗುಣಾಕಾರ."
      calculation: "$= (x \times 2x) + (x \times 3) - (4 \times 2x) - (4 \times 3) = 2x^2 + 3x - 8x - 12$"
    - step: 3
      action: "ಸಜಾತಿ ಪದಗಳನ್ನು ಕೂಡಿಸಿ ಸರಳೀಕರಿಸುವುದು."
      justification: "ಅಂತಿಮ ಉತ್ತರವನ್ನು ಪಡೆಯಲು."
      calculation: "$= 2x^2 + (3-8)x - 12 = 2x^2 - 5x - 12$"
    - step: 4
      action: "ಭಾಗ (ii) ನ್ನು ಬಿಡಿಸುವುದು: $(x-y)(3x+5y)$."
      justification: "ವಿತರಣಾ ನಿಯಮ ಬಳಸುವುದು."
      calculation: "$(x-y)(3x+5y) = x(3x+5y) - y(3x+5y)$"
    - step: 5
      action: "ಆವರಣಗಳನ್ನು ಬಿಡಿಸುವುದು."
      justification: "ಏಕಪದೋಕ್ತಿಯಿಂದ ದ್ವಿಪದೋಕ್ತಿಯ ಗುಣಾಕಾರ."
      calculation: "$= (x \times 3x) + (x \times 5y) - (y \times 3x) - (y \times 5y) = 3x^2 + 5xy - 3yx - 5y^2$"
    - step: 6
      action: "ಸಜಾತಿ ಪದಗಳನ್ನು ಕೂಡಿಸಿ ಸರಳೀಕರಿಸುವುದು ($xy=yx$)."
      justification: "ಅಂತಿಮ ಉತ್ತರವನ್ನು ಪಡೆಯಲು."
      calculation: "$= 3x^2 + (5-3)xy - 5y^2 = 3x^2 + 2xy - 5y^2$"
  result: "(i) $2x^2 - 5x - 12$ (ii) $3x^2 + 2xy - 5y^2$"
