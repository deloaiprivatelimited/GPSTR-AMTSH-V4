[ಮೂಲ_ಅವಧಾರಣೆ_1]
concept_id: ALG-EQ2-3_concept_1
concept_name: ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳ ವ್ಯಾಖ್ಯೆ ಮತ್ತು ಪ್ರಮಾಣಿತ ರೂಪ (Definition and Standard Form of Linear Equations)
domain: ಬೀಜಗಣಿತ (Algebra)

  [ಸಾರಾಂಶ]
  ಈ ಅವಧಾರಣೆಯು ಒಂದು ಮತ್ತು ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳನ್ನು ಪರಿಚಯಿಸುತ್ತದೆ. ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣದ ಪ್ರಮಾಣಿತ ರೂಪವಾದ `ax + by + c = 0` ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಲಾಗುತ್ತದೆ. ಯಾವುದೇ ಸಮೀಕರಣವನ್ನು ಈ ಪ್ರಮಾಣಿತ ರೂಪಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದು ಮತ್ತು ಅದರಲ್ಲಿನ `a`, `b`, ಮತ್ತು `c` ಗುಣಾಂಕಗಳನ್ನು ಗುರುತಿಸುವುದನ್ನು ಇಲ್ಲಿ ಕಲಿಯಲಾಗುತ್ತದೆ.

  [ವ್ಯಾಖ್ಯೆ]
  1.  **ಒಂದು ಚರಾಕ್ಷರವಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣ (Linear Equation in One Variable):** `ax + b = 0` ರೂಪದಲ್ಲಿ ಬರೆಯಬಹುದಾದ ಸಮೀಕರಣ, ಇಲ್ಲಿ `a` ಮತ್ತು `b` ಗಳು ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳಾಗಿದ್ದು `a ≠ 0` ಆಗಿರುತ್ತದೆ. ಇದಕ್ಕೆ ಒಂದೇ ಒಂದು (ಅನನ್ಯ) ಪರಿಹಾರವಿರುತ್ತದೆ.
  2.  **ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣ (Linear Equation in Two Variables):** `ax + by + c = 0` ರೂಪದಲ್ಲಿ ಬರೆಯಬಹುದಾದ ಯಾವುದೇ ಸಮೀಕರಣವನ್ನು ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣ ಎನ್ನುತ್ತೇವೆ. ಇಲ್ಲಿ `a`, `b` ಮತ್ತು `c` ಗಳು ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳಾಗಿದ್ದು, `a` ಮತ್ತು `b` ಎರಡೂ ಏಕಕಾಲದಲ್ಲಿ ಸೊನ್ನೆಯಾಗಿರುವುದಿಲ್ಲ (`a` and `b` are not both zero).

  [ಅಂಶಪಟ್ಟಿ / Properties]
  ಒಂದು ರೇಖಾತ್ಮಕ ಸಮೀಕರಣವನ್ನು ಬಿಡಿಸುವಾಗ, ಅದರ ಪರಿಹಾರದ ಮೇಲೆ ಈ ಕೆಳಗಿನ ಕ್ರಿಯೆಗಳು ಯಾವುದೇ ಪರಿಣಾಮ ಬೀರುವುದಿಲ್ಲ:
  1.  ಸಮೀಕರಣದ ಎರಡೂ ಬದಿಗಳಿಗೆ ಒಂದೇ ಸಮಾನ ಸಂಖ್ಯೆಯನ್ನು ಕೂಡಿಸುವುದು ಅಥವಾ ಕಳೆಯುವುದು.
  2.  ಸಮೀಕರಣದ ಎರಡೂ ಬದಿಗಳನ್ನು ಒಂದೇ ಶೂನ್ಯವಲ್ಲದ (non-zero) ಸಂಖ್ಯೆಯಿಂದ ಗುಣಿಸುವುದು ಅಥವಾ ಭಾಗಿಸುವುದು.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  **ಉದಾಹರಣೆ 1: ಸಮೀಕರಣಗಳನ್ನು ಪ್ರಮಾಣಿತ ರೂಪಕ್ಕೆ ಪರಿವರ್ತಿಸುವುದು**
  example_id: ALG-EQ2-3_example_1
  difficulty: basic
  problem: ಕೆಳಗಿನ ಪ್ರತಿಯೊಂದು ಸಮೀಕರಣವನ್ನು `ax + by + c = 0` ರೂಪದಲ್ಲಿ ಬರೆಯಿರಿ ಮತ್ತು ಪ್ರತಿಸಂದರ್ಭದಲ್ಲಿ `a, b` ಮತ್ತು `c` ಗಳ ಬೆಲೆಗಳನ್ನು ಬರೆಯಿರಿ.
  (i) `2x + 3y = 4.37`
  (ii) `x - 4 = \sqrt{3}y`
  (iii) `4 = 5x - 3y`
  (iv) `2x = y`
  steps:
    - step: 1
      action: (i) `2x + 3y = 4.37` ಸಮೀಕರಣವನ್ನು ಪರಿಗಣಿಸಿ.
      justification: `ax + by + c = 0` ರೂಪಕ್ಕೆ ತರಲು, ಬಲಭಾಗದಲ್ಲಿರುವ ಪದವನ್ನು ಎಡಭಾಗಕ್ಕೆ ಸ್ಥಳಾಂತರಿಸಬೇಕು.
      calculation: |
        `2x + 3y = 4.37`
        `2x + 3y - 4.37 = 0`
        ಈಗ ಇದನ್ನು `ax + by + c = 0` ಜೊತೆ ಹೋಲಿಸಿದಾಗ,
        `a = 2`, `b = 3`, `c = -4.37`
    - step: 2
      action: (ii) `x - 4 = \sqrt{3}y` ಸಮೀಕರಣವನ್ನು ಪರಿಗಣಿಸಿ.
      justification: `y` ಪದವನ್ನು ಎಡಭಾಗಕ್ಕೆ ತಂದು ಪ್ರಮಾಣಿತ ರೂಪದಲ್ಲಿ ಬರೆಯುವುದು.
      calculation: |
        `x - 4 = \sqrt{3}y`
        `x - \sqrt{3}y - 4 = 0`
        ಈಗ ಇದನ್ನು `ax + by + c = 0` ಜೊತೆ ಹೋಲಿಸಿದಾಗ,
        `a = 1`, `b = -\sqrt{3}`, `c = -4`
    - step: 3
      action: (iii) `4 = 5x - 3y` ಸಮೀಕರಣವನ್ನು ಪರಿಗಣಿಸಿ.
      justification: ಎಡಭಾಗದಲ್ಲಿರುವ ಸ್ಥಿರಾಂಕವನ್ನು ಬಲಭಾಗಕ್ಕೆ ತಂದು, ಸಮೀಕರಣವನ್ನು `0` ಗೆ ಸಮೀಕರಿಸುವುದು.
      calculation: |
        `4 = 5x - 3y`
        `5x - 3y - 4 = 0`
        ಈಗ ಇದನ್ನು `ax + by + c = 0` ಜೊತೆ ಹೋಲಿಸಿದಾಗ,
        `a = 5`, `b = -3`, `c = -4`
        (ಗಮನಿಸಿ: ಇದನ್ನು `-5x + 3y + 4 = 0` ಎಂದೂ ಬರೆಯಬಹುದು, ಆಗ `a = -5, b = 3, c = 4` ಆಗುತ್ತದೆ.)
    - step: 4
      action: (iv) `2x = y` ಸಮೀಕರಣವನ್ನು ಪರಿಗಣಿಸಿ.
      justification: `y` ಪದವನ್ನು ಎಡಭಾಗಕ್ಕೆ ತಂದು, ಸ್ಥಿರಾಂಕ ಪದವನ್ನು `0` ಎಂದು ಪರಿಗಣಿಸುವುದು.
      calculation: |
        `2x = y`
        `2x - y = 0`
        `2x - 1y + 0 = 0`
        ಈಗ ಇದನ್ನು `ax + by + c = 0` ಜೊತೆ ಹೋಲಿಸಿದಾಗ,
        `a = 2`, `b = -1`, `c = 0`
  result: ಪ್ರತಿ ಸಮೀಕರಣವನ್ನು `ax + by + c = 0` ರೂಪದಲ್ಲಿ ಬರೆದು `a, b, c` ಬೆಲೆಗಳನ್ನು ಕಂಡುಹಿಡಿಯಲಾಗಿದೆ.

  **ಉದಾಹರಣೆ 2: ಒಂದು ಚರಾಕ್ಷರದ ಸಮೀಕರಣವನ್ನು ಎರಡು ಚರಾಕ್ಷರದ ಸಮೀಕರಣವಾಗಿ ಬರೆಯುವುದು**
  example_id: ALG-EQ2-3_example_2
  difficulty: basic
  problem: ಕೆಳಗಿನ ಪ್ರತಿಯೊಂದನ್ನೂ ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ಒಂದು ಸಮೀಕರಣವನ್ನಾಗಿ ಬರೆಯಿರಿ.
  (i) `x = -5`
  (ii) `y = 2`
  (iii) `2x = 3`
  (iv) `5y = 2`
  steps:
    - step: 1
      action: (i) `x = -5` ಸಮೀಕರಣವನ್ನು ಪರಿಗಣಿಸಿ.
      justification: `y` ಚರಾಕ್ಷರ ಇಲ್ಲದಿರುವುದರಿಂದ, ಅದರ ಗುಣಾಂಕವನ್ನು `0` ಎಂದು ತೆಗೆದುಕೊಂಡು `0y` ಪದವನ್ನು ಸೇರಿಸುವುದು.
      calculation: |
        `x = -5`
        `x + 5 = 0`
        `1x + 0y + 5 = 0`
    - step: 2
      action: (ii) `y = 2` ಸಮೀಕರಣವನ್ನು ಪರಿಗಣಿಸಿ.
      justification: `x` ಚರಾಕ್ಷರ ಇಲ್ಲದಿರುವುದರಿಂದ, ಅದರ ಗುಣಾಂಕವನ್ನು `0` ಎಂದು ತೆಗೆದುಕೊಂಡು `0x` ಪದವನ್ನು ಸೇರಿಸುವುದು.
      calculation: |
        `y = 2`
        `y - 2 = 0`
        `0x + 1y - 2 = 0`
    - step: 3
      action: (iii) `2x = 3` ಸಮೀಕರಣವನ್ನು ಪರಿಗಣಿಸಿ.
      justification: `0y` ಪದವನ್ನು ಸೇರಿಸಿ ಪ್ರಮಾಣಿತ ರೂಪಕ್ಕೆ ತರುವುದು.
      calculation: |
        `2x = 3`
        `2x - 3 = 0`
        `2x + 0y - 3 = 0`
    - step: 4
      action: (iv) `5y = 2` ಸಮೀಕರಣವನ್ನು ಪರಿಗಣಿಸಿ.
      justification: `0x` ಪದವನ್ನು ಸೇರಿಸಿ ಪ್ರಮಾಣಿತ ರೂಪಕ್ಕೆ ತರುವುದು.
      calculation: |
        `5y = 2`
        `5y - 2 = 0`
        `0x + 5y - 2 = 0`
  result: ಒಂದು ಚರಾಕ್ಷರವಿರುವ ಸಮೀಕರಣಗಳನ್ನು, ಕಾಣೆಯಾದ ಚರಾಕ್ಷರದ ಗುಣಾಂಕವನ್ನು ಸೊನ್ನೆ ಎಂದು ಪರಿಗಣಿಸಿ ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ಸಮೀಕರಣಗಳಾಗಿ ಬರೆಯಲಾಗಿದೆ.

  [ದೃಶ್ಯ_ಸಹಾಯ]
  type: number_line
  description: `2x + 5 = 0` ಸಮೀಕರಣದ ಪರಿಹಾರವಾದ `x = -5/2` ಅನ್ನು ಸಂಖ್ಯಾರೇಖೆಯ ಮೇಲೆ ಗುರುತಿಸಲಾಗಿದೆ.
  labels: ಸಂಖ್ಯಾರೇಖೆಯು -4 ರಿಂದ 4 ರವರೆಗಿನ ಪೂರ್ಣಾಂಕಗಳನ್ನು ಹೊಂದಿದೆ. -3 ಮತ್ತು -2 ರ ನಡುವೆ `-5/2` (-2.5) ಬಿಂದುವನ್ನು "ಮೂಲ" (Root) ಎಂದು ಗುರುತಿಸಲಾಗಿದೆ.
  mathematical_significance: ಒಂದು ಚರಾಕ್ಷರವಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣದ ಪರಿಹಾರವು ಸಂಖ್ಯಾರೇಖೆಯ ಮೇಲಿನ ಒಂದು ಅನನ್ಯ (unique) ಬಿಂದುವಾಗಿರುತ್ತದೆ ಎಂಬುದನ್ನು ಇದು ದೃಶ್ಯೀಕರಿಸುತ್ತದೆ.

  [ವಿಶ್ಲೇಷಣೆ]
  - `ax + by + c = 0` ವ್ಯಾಖ್ಯೆಯಲ್ಲಿ "a ಮತ್ತು b ಎರಡೂ ಸೊನ್ನೆ ಅಲ್ಲದಿರುವ" ಎಂಬ ಷರತ್ತು ಅತ್ಯಂತ ಮುಖ್ಯ. ಒಂದು ವೇಳೆ `a=0` ಮತ್ತು `b=0` ಎರಡೂ ಆದರೆ, ಸಮೀಕರಣವು `c=0` ಆಗುತ್ತದೆ. ಇದು ಚರಾಕ್ಷರಗಳನ್ನೇ ಹೊಂದಿರದ ಒಂದು ಹೇಳಿಕೆಯಾಗುತ್ತದೆಯೇ ಹೊರತು ಸಮೀಕರಣವಾಗುವುದಿಲ್ಲ.
  - ಯಾವುದೇ ಸಮೀಕರಣವನ್ನು ಪ್ರಮಾಣಿತ ರೂಪಕ್ಕೆ ತರುವಾಗ, ಎಲ್ಲಾ ಪದಗಳನ್ನು ಸಮೀಕರಣದ ಒಂದು ಬದಿಗೆ (ಸಾಮಾನ್ಯವಾಗಿ ಎಡಬದಿಗೆ) ತಂದು, ಇನ್ನೊಂದು ಬದಿಯನ್ನು ಸೊನ್ನೆಗೆ ಸಮೀಕರಿಸಲಾಗುತ್ತದೆ. ಈ ಪ್ರಕ್ರಿಯೆಯಲ್ಲಿ ಪದಗಳನ್ನು ಒಂದು ಬದಿಯಿಂದ ಇನ್ನೊಂದು ಬದಿಗೆ ಸಾಗಿಸುವಾಗ ಅವುಗಳ ಚಿಹ್ನೆಗಳು ಬದಲಾಗುತ್ತವೆ.

  [ಸಂಬಂಧ]
  - ಪೂರ್ವಾಪೇಕ್ಷಿತ ಅವಧಾರಣೆಗಳು: ಬೀಜಗಣಿತದ ಮೂಲಭೂತ ಪರಿಕಲ್ಪನೆಗಳು, ಚರಾಕ್ಷರಗಳು, ಸ್ಥಿರಾಂಕಗಳು, ಮತ್ತು ಒಂದು ಚರಾಕ್ಷರವಿರುವ ಸಮೀಕರಣಗಳನ್ನು ಬಿಡಿಸುವುದು.
  - ಮುಂದಿನ ಅವಧಾರಣೆಗಳೊಂದಿಗೆ ಸಂಪರ್ಕ: ಈ ಪ್ರಮಾಣಿತ ರೂಪವು ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳ ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯಲು, ಅವುಗಳ ನಕ್ಷೆಗಳನ್ನು ರಚಿಸಲು (Graphing) ಮತ್ತು ಎರಡು ಸಮೀಕರಣಗಳ ವ್ಯವಸ್ಥೆಯನ್ನು (System of linear equations) ಬಿಡಿಸಲು ಅಡಿಪಾಯವಾಗಿದೆ.

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  - gpstr_weightage: medium
  - MCQ ಸಾಧ್ಯತೆ: ಒಂದು ಸಮೀಕರಣವನ್ನು `ax + by + c = 0` ರೂಪಕ್ಕೆ ಪರಿವರ್ತಿಸಿ `a, b, c` ಗಳ ಬೆಲೆಗಳನ್ನು ಗುರುತಿಸಲು ಕೇಳಬಹುದು. ಉದಾಹರಣೆಗೆ, `2x = 5y - 3` ಸಮೀಕರಣದಲ್ಲಿ `c` ಯ ಬೆಲೆ ಏನು? (ಸರಿಯಾದ ಉತ್ತರ: 3).
  - 2-ಅಂಕ ಪ್ರಶ್ನೆ: ಒಂದು ವಾಕ್ಯದ ಸಮಸ್ಯೆಯನ್ನು (word problem) ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣವಾಗಿ ನಿರೂಪಿಸಲು ಕೇಳಬಹುದು.
  - ಪ್ರಮುಖ trap: ಪ್ರಮಾಣಿತ ರೂಪಕ್ಕೆ ಪರಿವರ್ತಿಸುವಾಗ ಸ್ಥಿರಾಂಕ `c` ಯ ಚಿಹ್ನೆಯನ್ನು ತಪ್ಪಾಗಿ ಗುರುತಿಸುವುದು. ಉದಾಹರಣೆಗೆ, `2x + 3y = 4.37` ಅನ್ನು `2x + 3y - 4.37 = 0` ಎಂದು ಬರೆದಾಗ, `c` ಯ ಬೆಲೆ `-4.37` ಆಗುತ್ತದೆ, `4.37` ಅಲ್ಲ.

[ಮೂಲ_ಅವಧಾರಣೆ_2]
concept_id: ALG-EQ2-3_concept_2
concept_name: ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳ ಪರಿಹಾರಗಳು (Solutions of Linear Equations)
domain: ಬೀಜಗಣಿತ (Algebra)

  [ಸಾರಾಂಶ]
  ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣದ 'ಪರಿಹಾರ' ಎಂದರೇನು ಎಂಬುದನ್ನು ಈ ಭಾಗವು ವಿವರಿಸುತ್ತದೆ. ಒಂದು ಚರಾಕ್ಷರದ ಸಮೀಕರಣಕ್ಕಿರುವಂತೆ ಇದಕ್ಕೆ ಒಂದೇ ಪರಿಹಾರವಿರುವುದಿಲ್ಲ, ಬದಲಾಗಿ ಅಪರಿಮಿತ ಸಂಖ್ಯೆಯ ಪರಿಹಾರಗಳಿರುತ್ತವೆ ಎಂಬ ಪ್ರಮುಖ ಅಂಶವನ್ನು ಸ್ಥಾಪಿಸಲಾಗುತ್ತದೆ. ಈ ಅಪರಿಮಿತ ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವ ವಿಧಾನವನ್ನು ಉದಾಹರಣೆಗಳ ಮೂಲಕ ವಿವರಿಸಲಾಗಿದೆ.

  [ವ್ಯಾಖ್ಯೆ]
  1.  **ರೇಖಾತ್ಮಕ ಸಮೀಕರಣದ ಪರಿಹಾರ (Solution of a Linear Equation):** `ax + by + c = 0` ಸಮೀಕರಣದಲ್ಲಿ, `x` ಮತ್ತು `y` ಚರಾಕ್ಷರಗಳ ಯಾವ ಬೆಲೆಗಳ ಜೋಡಿಯು (`x, y`) ಸಮೀಕರಣವನ್ನು ಸತ್ಯವಾಗಿಸುತ್ತದೆಯೋ (satisfies the equation), ಆ ಕ್ರಮಬದ್ಧ ಜೋಡಿಯನ್ನು (ordered pair) ಸಮೀಕರಣದ ಒಂದು ಪರಿಹಾರ ಎನ್ನುತ್ತೇವೆ.

  [ಅಂಶಪಟ್ಟಿ / Properties]
  1.  **ಅಪರಿಮಿತ ಪರಿಹಾರಗಳು (Infinite Solutions):** ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ಪ್ರತಿಯೊಂದು ರೇಖಾತ್ಮಕ ಸಮೀಕರಣವು ಅಪರಿಮಿತ ಸಂಖ್ಯೆಯ ಪರಿಹಾರಗಳನ್ನು ಹೊಂದಿರುತ್ತದೆ.
  2.  **ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವ ವಿಧಾನ (Method to find solutions):**
      *   ಚರಾಕ್ಷರಗಳಲ್ಲಿ ಒಂದಕ್ಕೆ (ಉದಾಹರಣೆಗೆ `x`) ಇಷ್ಟಬಂದ ಒಂದು ವಾಸ್ತವ ಸಂಖ್ಯೆಯನ್ನು ಆದೇಶಿಸಿ.
      *   ಆಗ ಸಮೀಕರಣವು ಇನ್ನೊಂದು ಚರಾಕ್ಷರದಲ್ಲಿ (ಉದಾಹರಣೆಗೆ `y`) ಒಂದು ರೇಖಾತ್ಮಕ ಸಮೀಕರಣವಾಗುತ್ತದೆ. ಅದನ್ನು ಬಿಡಿಸಿ `y` ಯ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
      *   ಈ `(x, y)` ಜೋಡಿಯು ಒಂದು ಪರಿಹಾರವಾಗಿರುತ್ತದೆ. ಈ ಪ್ರಕ್ರಿಯೆಯನ್ನು `x` ನ ಬೇರೆ ಬೇರೆ ಬೆಲೆಗಳಿಗೆ ಪುನರಾವರ್ತಿಸಿ ಅನೇಕ ಪರಿಹಾರಗಳನ್ನು ಪಡೆಯಬಹುದು.
  3.  **ಸುಲಭ ವಿಧಾನ (Easy Method):**
      *   `x = 0` ಎಂದು ಆದೇಶಿಸಿ `y` ಯ ಅನುರೂಪ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
      *   `y = 0` ಎಂದು ಆದೇಶಿಸಿ `x` ಯ ಅನುರೂಪ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯುವುದು. ಈ ಎರಡು ಪರಿಹಾರಗಳು ನಕ್ಷೆ ರಚಿಸಲು ಬಹಳ ಉಪಯುಕ್ತ.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  **ಉದಾಹರಣೆ 3: ಒಂದು ಸಮೀಕರಣಕ್ಕೆ 4 ವಿಭಿನ್ನ ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು**
  example_id: ALG-EQ2-3_example_3
  difficulty: intermediate
  problem: `x + 2y = 6` ಎಂಬ ಸಮೀಕರಣಕ್ಕೆ 4 ವಿಭಿನ್ನ ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
  steps:
    - step: 1
      action: ಪರಿಹಾರ 1 ಅನ್ನು ಕಂಡುಹಿಡಿಯಲು `x = 2` ಎಂದು ಆದೇಶಿಸೋಣ.
      justification: `x` ಗೆ ಒಂದು ಮೌಲ್ಯವನ್ನು ಆಯ್ದುಕೊಂಡು `y` ಗಾಗಿ ಸಮೀಕರಣವನ್ನು ಬಿಡಿಸುವುದು.
      calculation: |
        `x + 2y = 6`
        `2 + 2y = 6`
        `2y = 6 - 2`
        `2y = 4`
        `y = 2`
        ಆದ್ದರಿಂದ, `(2, 2)` ಒಂದು ಪರಿಹಾರವಾಗಿದೆ.
    - step: 2
      action: ಪರಿಹಾರ 2 ಅನ್ನು ಕಂಡುಹಿಡಿಯಲು `x = 0` ಎಂದು ಆದೇಶಿಸೋಣ.
      justification: ಇದು `y`-ಅಕ್ಷವನ್ನು ರೇಖೆಯು ಛೇದಿಸುವ ಬಿಂದುವನ್ನು ನೀಡುತ್ತದೆ.
      calculation: |
        `x + 2y = 6`
        `0 + 2y = 6`
        `2y = 6`
        `y = 3`
        ಆದ್ದರಿಂದ, `(0, 3)` ಇನ್ನೊಂದು ಪರಿಹಾರವಾಗಿದೆ.
    - step: 3
      action: ಪರಿಹಾರ 3 ಅನ್ನು ಕಂಡುಹಿಡಿಯಲು `y = 0` ಎಂದು ಆದೇಶಿಸೋಣ.
      justification: ಇದು `x`-ಅಕ್ಷವನ್ನು ರೇಖೆಯು ಛೇದಿಸುವ ಬಿಂದುವನ್ನು ನೀಡುತ್ತದೆ.
      calculation: |
        `x + 2y = 6`
        `x + 2(0) = 6`
        `x + 0 = 6`
        `x = 6`
        ಆದ್ದರಿಂದ, `(6, 0)` ಮತ್ತೊಂದು ಪರಿಹಾರವಾಗಿದೆ.
    - step: 4
      action: ಪರಿಹಾರ 4 ಅನ್ನು ಕಂಡುಹಿಡಿಯಲು `y = 1` ಎಂದು ಆದೇಶಿಸೋಣ.
      justification: `y` ಗೆ ಇನ್ನೊಂದು ಮೌಲ್ಯವನ್ನು ಆಯ್ದುಕೊಂಡು `x` ಗಾಗಿ ಬಿಡಿಸುವುದು.
      calculation: |
        `x + 2y = 6`
        `x + 2(1) = 6`
        `x + 2 = 6`
        `x = 6 - 2`
        `x = 4`
        ಆದ್ದರಿಂದ, `(4, 1)` ನಾಲ್ಕನೇ ಪರಿಹಾರವಾಗಿದೆ.
  result: `x + 2y = 6` ಸಮೀಕರಣದ ನಾಲ್ಕು ವಿಭಿನ್ನ ಪರಿಹಾರಗಳು `(2, 2)`, `(0, 3)`, `(6, 0)` ಮತ್ತು `(4, 1)`.

  **ಉದಾಹರಣೆ 4: ವಿವಿಧ ಸಮೀಕರಣಗಳಿಗೆ ಎರಡೆರಡು ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು**
  example_id: ALG-EQ2-3_example_4
  difficulty: intermediate
  problem: ಕೆಳಗಿನ ಪ್ರತಿಯೊಂದು ಸಮೀಕರಣಕ್ಕೂ ಎರಡೆರಡು ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
  (i) `4x + 3y = 12`
  (ii) `2x + 5y = 0`
  (iii) `3y + 4 = 0`
  steps:
    - step: 1
      action: (i) `4x + 3y = 12` ಸಮೀಕರಣಕ್ಕೆ ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
      justification: `x=0` ಮತ್ತು `y=0` ಎಂದು ಆದೇಶಿಸುವ ಸುಲಭ ವಿಧಾನವನ್ನು ಬಳಸುವುದು.
      calculation: |
        `x = 0` ಆದಾಗ:
        `4(0) + 3y = 12`
        `3y = 12`
        `y = 4`
        ಒಂದು ಪರಿಹಾರ: `(0, 4)`

        `y = 0` ಆದಾಗ:
        `4x + 3(0) = 12`
        `4x = 12`
        `x = 3`
        ಎರಡನೇ ಪರಿಹಾರ: `(3, 0)`
    - step: 2
      action: (ii) `2x + 5y = 0` ಸಮೀಕರಣಕ್ಕೆ ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
      justification: `x=0` ಎಂದು ಆದೇಶಿಸಿ ಮೊದಲ ಪರಿಹಾರವನ್ನು ಕಂಡುಹಿಡಿಯುವುದು. ನಂತರ `x` ಗೆ ಬೇರೊಂದು ಬೆಲೆ ಆದೇಶಿಸಿ ಎರಡನೇ ಪರಿಹಾರವನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
      calculation: |
        `x = 0` ಆದಾಗ:
        `2(0) + 5y = 0`
        `5y = 0`
        `y = 0`
        ಒಂದು ಪರಿಹಾರ: `(0, 0)`

        `x = 1` ಆದಾಗ:
        `2(1) + 5y = 0`
        `2 + 5y = 0`
        `5y = -2`
        `y = -2/5`
        ಎರಡನೇ ಪರಿಹಾರ: `(1, -2/5)`
    - step: 3
      action: (iii) `3y + 4 = 0` ಸಮೀಕರಣಕ್ಕೆ ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
      justification: ಈ ಸಮೀಕರಣವನ್ನು `0x + 3y + 4 = 0` ಎಂದು ಬರೆಯಬಹುದು. ಇಲ್ಲಿ `y` ಯ ಬೆಲೆ `x` ಮೇಲೆ ಅವಲಂಬಿತವಾಗಿಲ್ಲ.
      calculation: |
        `3y + 4 = 0`
        `3y = -4`
        `y = -4/3`
        `y` ಯ ಬೆಲೆಯು `x` ನ ಯಾವುದೇ ಬೆಲೆಗೆ ಸ್ಥಿರವಾಗಿರುತ್ತದೆ.
        `x = 0` ಆದಾಗ, `y = -4/3`. ಒಂದು ಪರಿಹಾರ: `(0, -4/3)`
        `x = 1` ಆದಾಗ, `y = -4/3`. ಎರಡನೇ ಪರಿಹಾರ: `(1, -4/3)`
  result: ಪ್ರತಿ ಸಮೀಕರಣಕ್ಕೂ ಎರಡೆರಡು ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯಲಾಗಿದೆ.

  [ವಿಶ್ಲೇಷಣೆ]
  - ಒಂದು ಚರಾಕ್ಷರದ ಸಮೀಕರಣಕ್ಕೆ ಒಂದು ಅನನ್ಯ ಪರಿಹಾರವಿದ್ದರೆ, ಎರಡು ಚರಾಕ್ಷರಗಳ ಸಮೀಕರಣಕ್ಕೆ ಅಪರಿಮಿತ ಪರಿಹಾರಗಳಿರುತ್ತವೆ ಎಂಬುದು ಒಂದು ಪ್ರಮುಖ ಪರಿಕಲ್ಪನಾತ್ಮಕ ಜಿಗಿತವಾಗಿದೆ.
  - ಈ ಅಪರಿಮಿತ ಪರಿಹಾರಗಳ ಗಣವು ಕಾರ್ಟೀಸಿಯನ್ ಸಮತಲದಲ್ಲಿ (Cartesian Plane) ಒಂದು ಸರಳ ರೇಖೆಯನ್ನು ರೂಪಿಸುತ್ತದೆ. ಪ್ರತಿಯೊಂದು ಪರಿಹಾರ `(x, y)` ಆ ರೇಖೆಯ ಮೇಲಿನ ಒಂದು ಬಿಂದುವಾಗಿರುತ್ತದೆ.
  - `ax + by = 0` ರೂಪದ ಸಮೀಕರಣಗಳು ಯಾವಾಗಲೂ ಮೂಲಬಿಂದು `(0, 0)` ಮೂಲಕ ಹಾದು ಹೋಗುತ್ತವೆ.
  - `ax + c = 0` ಅಥವಾ `by + c = 0` ರೂಪದ ಸಮೀಕರಣಗಳು ಕ್ರಮವಾಗಿ `y`-ಅಕ್ಷಕ್ಕೆ ಸಮಾಂತರವಾಗಿರುವ ಲಂಬ ರೇಖೆ (vertical line) ಮತ್ತು `x`-ಅಕ್ಷಕ್ಕೆ ಸಮಾಂತರವಾಗಿರುವ ಅಡ್ಡ ರೇಖೆಯನ್ನು (horizontal line) ಪ್ರತಿನಿಧಿಸುತ್ತವೆ.

  [ಸಂಬಂಧ]
  - ಪೂರ್ವಾಪೇಕ್ಷಿತ ಅವಧಾರಣೆಗಳು: ರೇಖಾತ್ಮಕ ಸಮೀಕರಣದ ಪ್ರಮಾಣಿತ ರೂಪ, ಬೀಜಗಣಿತದ ಪದಗಳನ್ನು ಆದೇಶಿಸುವುದು ಮತ್ತು ಸಮೀಕರಣ ಬಿಡಿಸುವುದು.
  - ಮುಂದಿನ ಅವಧಾರಣೆಗಳೊಂದಿಗೆ ಸಂಪರ್ಕ: ಈ ಪರಿಹಾರಗಳನ್ನು ಕಾರ್ಟೀಸಿಯನ್ ಸಮತಲದಲ್ಲಿ ಬಿಂದುಗಳಾಗಿ ಗುರುತಿಸಿ, ರೇಖಾತ್ಮಕ ಸಮೀಕರಣದ ನಕ್ಷೆ (Graph) ರಚಿಸುವುದು ಮುಂದಿನ ಹಂತವಾಗಿದೆ.

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  - gpstr_weightage: high
  - MCQ ಸಾಧ್ಯತೆ: ಒಂದು ಬಿಂದು `(p, q)` ವನ್ನು ಕೊಟ್ಟು, ಅದು ನೀಡಿರುವ ಸಮೀಕರಣದ ಪರಿಹಾರ ಹೌದೋ ಅಲ್ಲವೋ ಎಂದು ಪರೀಕ್ಷಿಸಲು ಕೇಳಬಹುದು. ಅಥವಾ `2x + 3y = k` ಸಮೀಕರಣಕ್ಕೆ `(2, 1)` ಒಂದು ಪರಿಹಾರವಾದರೆ `k` ಯ ಬೆಲೆ ಕಂಡುಹಿಡಿಯಿರಿ. (ಉತ್ತರ: `k = 2(2) + 3(1) = 7`).
  - 2-ಅಂಕ ಪ್ರಶ್ನೆ: ಒಂದು ಸಮೀಕರಣಕ್ಕೆ 2 ಅಥವಾ 4 ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿಯಲು ಕೇಳುವುದು.
  - 5-ಅಂಕ ಪ್ರಶ್ನೆ: ಸಾಮಾನ್ಯವಾಗಿ ನಕ್ಷೆ ರಚನೆಯೊಂದಿಗೆ ಸೇರಿಸಿ ಕೇಳಲಾಗುತ್ತದೆ. (ಪರಿಹಾರಗಳನ್ನು ಕಂಡುಹಿಡಿದು ನಕ್ಷೆ ರಚಿಸಿ).
  - ಪ್ರಮುಖ trap: ಪರಿಹಾರ `(x, y)` ಅನ್ನು ಆದೇಶಿಸುವಾಗ `x` ಮತ್ತು `y` ಬೆಲೆಗಳನ್ನು ಅದಲು ಬದಲು ಮಾಡುವುದು. ಉದಾಹರಣೆಗೆ, `(2, 3)` ಅನ್ನು `x+y=5` ಕ್ಕೆ ಆದೇಶಿಸುವಾಗ `x=3, y=2` ಎಂದು ತಪ್ಪಾಗಿ ತೆಗೆದುಕೊಳ್ಳುವುದು.
