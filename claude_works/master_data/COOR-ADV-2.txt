[ಮೂಲ_ಅವಧಾರಣೆ_1]
concept_id: COOR-ADV-2_concept_1
concept_name: ದೂರ ಸೂತ್ರ (Distance Formula)
domain: ನಿರ್ದೇಶಾಂಕ ರೇಖಾಗಣಿತ

[ಸಾರಾಂಶ]
ನಿರ್ದೇಶಾಂಕ ರೇಖಾಗಣಿತದಲ್ಲಿ, ಎರಡು ಬಿಂದುಗಳ ನಡುವಿನ ನೇರ ಅಂತರವನ್ನು ಕಂಡುಹಿಡಿಯಲು ದೂರ ಸೂತ್ರವನ್ನು ಬಳಸಲಾಗುತ್ತದೆ. ಈ ಸೂತ್ರವು ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯದ ಒಂದು ನೇರ ಅನ್ವಯವಾಗಿದೆ. ಸಮತಲದಲ್ಲಿ ಯಾವುದೇ ಎರಡು ಬಿಂದುಗಳ ನಿರ್ದೇಶಾಂಕಗಳು ತಿಳಿದಿದ್ದರೆ, ಅವುಗಳ ನಡುವಿನ ದೂರವನ್ನು ಈ ಸೂತ್ರದ ಸಹಾಯದಿಂದ ನಿಖರವಾಗಿ ಲೆಕ್ಕಾಚಾರ ಮಾಡಬಹುದು.

[ವ್ಯಾಖ್ಯೆ]
1.  **ನಿರ್ದೇಶಾಂಕ ಅಕ್ಷಗಳು (Coordinate Axes):** ಒಂದು ಸಮತಲದಲ್ಲಿ ಬಿಂದುವಿನ ಸ್ಥಾನವನ್ನು ಗುರುತಿಸಲು ಬಳಸುವ ಎರಡು ಪರಸ್ಪರ ಲಂಬವಾಗಿರುವ ಸರಳರೇಖೆಗಳು. ಇವುಗಳನ್ನು x-ಅಕ್ಷ (ಕ್ಷಿತಿಜ ಅಕ್ಷ) ಮತ್ತು y-ಅಕ್ಷ (ಲಂಬ ಅಕ್ಷ) ಎಂದು ಕರೆಯಲಾಗುತ್ತದೆ.
2.  **x-ನಿರ್ದೇಶಾಂಕ (x-coordinate):** ಒಂದು ಬಿಂದುವಿಗೆ y-ಅಕ್ಷದಿಂದ ಇರುವ ದೂರ. ಇದನ್ನು 'ಕ್ಷಿತಿಜ ದೂರ' ಅಥವಾ 'ಅಬ್ಸಿಸ್ಸಾ' (abscissa) ಎಂದೂ ಕರೆಯುತ್ತಾರೆ.
3.  **y-ನಿರ್ದೇಶಾಂಕ (y-coordinate):** ಒಂದು ಬಿಂದುವಿಗೆ x-ಅಕ್ಷದಿಂದ ಇರುವ ದೂರ. ಇದನ್ನು 'ಲಂಬ ದೂರ' ಅಥವಾ 'ಆರ್ಡಿನೇಟ್' (ordinate) ಎಂದೂ ಕರೆಯುತ್ತಾರೆ.
4.  **ಮೂಲಬಿಂದು (Origin):** x-ಅಕ್ಷ ಮತ್ತು y-ಅಕ್ಷಗಳು ಛೇದಿಸುವ ಬಿಂದು. ಇದರ ನಿರ್ದೇಶಾಂಕಗಳು (0, 0).

[ಸೂತ್ರಗಳು]
**1. ದೂರ ಸೂತ್ರ (Distance Formula)**
ಎರಡು ಬಿಂದುಗಳಾದ $P(x_1, y_1)$ ಮತ್ತು $Q(x_2, y_2)$ ನಡುವಿನ ದೂರವನ್ನು ಕಂಡುಹಿಡಿಯುವ ಸೂತ್ರ.

*   **formula_name:** Distance Formula
*   **latex:** `d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}`
*   **condition:** $P$ ಮತ್ತು $Q$ ಸಮತಲದಲ್ಲಿರುವ ಯಾವುದೇ ಎರಡು ಬಿಂದುಗಳಾಗಿರಬೇಕು.
*   **exception:** ಇಲ್ಲ.
*   **derived_from:** ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯ.
*   **used_in:** ಎರಡು ಬಿಂದುಗಳ ನಡುವಿನ ಅಂತರವನ್ನು ಕಂಡುಹಿಡಿಯಲು, ರೇಖಾಖಂಡದ ಉದ್ದವನ್ನು ಅಳೆಯಲು, ಮತ್ತು ಜ್ಯಾಮಿತೀಯ ಆಕೃತಿಗಳ ಗುಣಲಕ್ಷಣಗಳನ್ನು ಸಾಧಿಸಲು.
*   **derivation:**
    *   **step 1:** $P(x_1, y_1)$ ಮತ್ತು $Q(x_2, y_2)$ ಬಿಂದುಗಳನ್ನು ಗುರುತಿಸಿ. x-ಅಕ್ಷಕ್ಕೆ $PR$ ಮತ್ತು $QS$ ಲಂಬಗಳನ್ನು ಎಳೆಯಿರಿ. $QS$ ಗೆ $PT$ ಲಂಬವನ್ನು ಎಳೆಯಿರಿ.
    *   **step 2:** ಇದರಿಂದ ಲಂಬಕೋನ ತ್ರಿಭುಜ $PTQ$ ಉಂಟಾಗುತ್ತದೆ.
    *   **step 3:** $PT$ ಯ ಉದ್ದವು x-ನಿರ್ದೇಶಾಂಕಗಳ ವ್ಯತ್ಯಾಸಕ್ಕೆ ಸಮ: $PT = RS = OS - OR = x_2 - x_1$.
    *   **step 4:** $QT$ ಯ ಉದ್ದವು y-ನಿರ್ದೇಶಾಂಕಗಳ ವ್ಯತ್ಯಾಸಕ್ಕೆ ಸಮ: $QT = QS - TS = y_2 - y_1$.
    *   **step 5:** $\triangle PTQ$ ನಲ್ಲಿ ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯವನ್ನು ಅನ್ವಯಿಸಿದಾಗ: $PQ^2 = PT^2 + QT^2$.
    *   **result:** $PQ^2 = (x_2 - x_1)^2 + (y_2 - y_1)^2 \implies PQ = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$.

**2. ಮೂಲಬಿಂದುವಿನಿಂದ ದೂರ (Distance from Origin)**
ಮೂಲಬಿಂದು $(0, 0)$ ಮತ್ತು ಯಾವುದೇ ಬಿಂದು $P(x, y)$ ನಡುವಿನ ದೂರ.

*   **formula_name:** Distance from Origin
*   **latex:** `OP = \sqrt{x^2 + y^2}`
*   **condition:** ಒಂದು ಬಿಂದು ಮೂಲಬಿಂದುವಾಗಿರಬೇಕು.
*   **derived_from:** ದೂರ ಸೂತ್ರದಲ್ಲಿ $(x_1, y_1) = (0, 0)$ ಮತ್ತು $(x_2, y_2) = (x, y)$ ಎಂದು ಆದೇಶಿಸಿದಾಗ.

[ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
**ಉದಾಹರಣೆ 1**
*   **example_id:** COOR-ADV-2_example_1
*   **difficulty:** intermediate
*   **problem:** (3, 2), (-2, -3) ಮತ್ತು (2, 3) ಬಿಂದುಗಳಿಂದ ತ್ರಿಭುಜ ಉಂಟಾಗುತ್ತದೆಯೇ? ಹೌದು ಎಂದಾದರೆ, ಉಂಟಾದ ತ್ರಿಭುಜದ ವಿಧವನ್ನು ಹೆಸರಿಸಿ.
*   **steps:**
    *   **step 1:** ದತ್ತ ಬಿಂದುಗಳನ್ನು P(3, 2), Q(-2, -3) ಮತ್ತು R(2, 3) ಎಂದು ಹೆಸರಿಸೋಣ. ದೂರ ಸೂತ್ರವನ್ನು ಬಳಸಿ ಬಾಹುಗಳ ಉದ್ದವನ್ನು ಕಂಡುಹಿಡಿಯೋಣ.
        *   **justification:** ತ್ರಿಭುಜದ ಅಸ್ತಿತ್ವ ಮತ್ತು ವಿಧವನ್ನು ನಿರ್ಧರಿಸಲು ಬಾಹುಗಳ ಉದ್ದಗಳು ಬೇಕು.
        *   **calculation:**
            $PQ = \sqrt{(-2 - 3)^2 + (-3 - 2)^2} = \sqrt{(-5)^2 + (-5)^2} = \sqrt{25 + 25} = \sqrt{50}$
            $QR = \sqrt{(2 - (-2))^2 + (3 - (-3))^2} = \sqrt{(4)^2 + (6)^2} = \sqrt{16 + 36} = \sqrt{52}$
            $PR = \sqrt{(2 - 3)^2 + (3 - 2)^2} = \sqrt{(-1)^2 + (1)^2} = \sqrt{1 + 1} = \sqrt{2}$
    *   **step 2:** ತ್ರಿಭುಜದ ಅಸಮಾನತೆಯ ನಿಯಮವನ್ನು ಪರಿಶೀಲಿಸೋಣ (ಯಾವುದೇ ಎರಡು ಬಾಹುಗಳ ಮೊತ್ತವು ಮೂರನೇ ಬಾಹಿಗಿಂತ ಹೆಚ್ಚಿರಬೇಕು).
        *   **justification:** ಈ ನಿಯಮ ಪಾಲನೆಯಾದರೆ ಮಾತ್ರ ತ್ರಿಭುಜ ರಚನೆ ಸಾಧ್ಯ.
        *   **calculation:** $PQ+PR = \sqrt{50} + \sqrt{2} \approx 7.07 + 1.41 = 8.48 > \sqrt{52} \approx 7.21$. ನಿಯಮ ಪಾಲನೆಯಾಗಿದೆ. ಆದ್ದರಿಂದ ತ್ರಿಭುಜ ಉಂಟಾಗುತ್ತದೆ.
    *   **step 3:** ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯದ ವಿಲೋಮವನ್ನು ಬಳಸಿ ತ್ರಿಭುಜದ ವಿಧವನ್ನು ಪರೀಕ್ಷಿಸೋಣ.
        *   **justification:** $a^2 + b^2 = c^2$ ಆಗಿದ್ದರೆ, ಅದು ಲಂಬಕೋನ ತ್ರಿಭುಜ.
        *   **calculation:**
            $PQ^2 = (\sqrt{50})^2 = 50$
            $PR^2 = (\sqrt{2})^2 = 2$
            $QR^2 = (\sqrt{52})^2 = 52$
            $PQ^2 + PR^2 = 50 + 2 = 52 = QR^2$.
*   **result:** $PQ^2 + PR^2 = QR^2$ ಆಗಿರುವುದರಿಂದ, ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯದ ವಿಲೋಮದ ಪ್ರಕಾರ, PQR ಒಂದು ಲಂಬಕೋನ ತ್ರಿಭುಜ ಮತ್ತು ಕೋನ $\angle P = 90^\circ$.

**ಉದಾಹರಣೆ 2**
*   **example_id:** COOR-ADV-2_example_2
*   **difficulty:** intermediate
*   **problem:** (1, 7), (4, 2), (-1, -1) ಮತ್ತು (-4, 4) ಬಿಂದುಗಳು ಚೌಕದ ಶೃಂಗಗಳು ಎಂದು ತೋರಿಸಿ.
*   **steps:**
    *   **step 1:** ದತ್ತ ಬಿಂದುಗಳನ್ನು A(1, 7), B(4, 2), C(-1, -1) ಮತ್ತು D(-4, 4) ಎಂದು ಹೆಸರಿಸೋಣ. ಚೌಕದ ಗುಣಲಕ್ಷಣಗಳನ್ನು (ಎಲ್ಲಾ ಬಾಹುಗಳು ಸಮ ಮತ್ತು ಕರ್ಣಗಳು ಸಮ) ಪರೀಕ್ಷಿಸಲು ದೂರ ಸೂತ್ರವನ್ನು ಬಳಸೋಣ.
    *   **step 2:** ನಾಲ್ಕು ಬಾಹುಗಳ ಉದ್ದವನ್ನು ಕಂಡುಹಿಡಿಯೋಣ.
        *   **calculation:**
            $AB = \sqrt{(4-1)^2 + (2-7)^2} = \sqrt{3^2 + (-5)^2} = \sqrt{9+25} = \sqrt{34}$
            $BC = \sqrt{(-1-4)^2 + (-1-2)^2} = \sqrt{(-5)^2 + (-3)^2} = \sqrt{25+9} = \sqrt{34}$
            $CD = \sqrt{(-4-(-1))^2 + (4-(-1))^2} = \sqrt{(-3)^2 + 5^2} = \sqrt{9+25} = \sqrt{34}$
            $DA = \sqrt{(1-(-4))^2 + (7-4)^2} = \sqrt{5^2 + 3^2} = \sqrt{25+9} = \sqrt{34}$
    *   **step 3:** ಎಲ್ಲಾ ನಾಲ್ಕು ಬಾಹುಗಳು ಸಮವಾಗಿವೆ: $AB = BC = CD = DA = \sqrt{34}$.
    *   **step 4:** ಎರಡು ಕರ್ಣಗಳ ಉದ್ದವನ್ನು ಕಂಡುಹಿಡಿಯೋಣ.
        *   **calculation:**
            $AC = \sqrt{(-1-1)^2 + (-1-7)^2} = \sqrt{(-2)^2 + (-8)^2} = \sqrt{4+64} = \sqrt{68}$
            $BD = \sqrt{(-4-4)^2 + (4-2)^2} = \sqrt{(-8)^2 + 2^2} = \sqrt{64+4} = \sqrt{68}$
    *   **step 5:** ಕರ್ಣಗಳು ಸಮವಾಗಿವೆ: $AC = BD = \sqrt{68}$.
*   **result:** ಚತುರ್ಭುಜ ABCD ಯಲ್ಲಿ ಎಲ್ಲಾ ನಾಲ್ಕು ಬಾಹುಗಳು ಸಮವಾಗಿವೆ ಮತ್ತು ಕರ್ಣಗಳು ಕೂಡ ಸಮವಾಗಿವೆ. ಆದ್ದರಿಂದ, ದತ್ತ ಬಿಂದುಗಳು ಒಂದು ಚೌಕದ ಶೃಂಗಗಳಾಗಿವೆ.

**ಉದಾಹರಣೆ 3**
*   **example_id:** COOR-ADV-2_example_3
*   **difficulty:** basic
*   **problem:** A(3, 1), B(6, 4) ಮತ್ತು C(8, 6) ಬಿಂದುಗಳು ಒಂದೇ ಸರಳರೇಖೆಯಲ್ಲಿವೆಯೇ (ಸರಳರೇಖಾಗತವೇ) ಎಂದು ಪರೀಕ್ಷಿಸಿ.
*   **steps:**
    *   **step 1:** ಬಿಂದುಗಳು ಸರಳರೇಖಾಗತವಾಗಿದ್ದರೆ, ಎರಡು ಚಿಕ್ಕ ದೂರಗಳ ಮೊತ್ತವು ಅತಿ ದೊಡ್ಡ ದೂರಕ್ಕೆ ಸಮನಾಗಿರಬೇಕು. ಅಂದರೆ, $AB + BC = AC$. ದೂರ ಸೂತ್ರವನ್ನು ಬಳಸಿ AB, BC, ಮತ್ತು AC ಯನ್ನು ಕಂಡುಹಿಡಿಯೋಣ.
    *   **step 2:** ದೂರಗಳನ್ನು ಲೆಕ್ಕ ಹಾಕಿ.
        *   **calculation:**
            $AB = \sqrt{(6-3)^2 + (4-1)^2} = \sqrt{3^2 + 3^2} = \sqrt{9+9} = \sqrt{18} = 3\sqrt{2}$
            $BC = \sqrt{(8-6)^2 + (6-4)^2} = \sqrt{2^2 + 2^2} = \sqrt{4+4} = \sqrt{8} = 2\sqrt{2}$
            $AC = \sqrt{(8-3)^2 + (6-1)^2} = \sqrt{5^2 + 5^2} = \sqrt{25+25} = \sqrt{50} = 5\sqrt{2}$
    *   **step 3:** ದೂರಗಳ ಮೊತ್ತವನ್ನು ಹೋಲಿಸಿ.
        *   **calculation:** $AB + BC = 3\sqrt{2} + 2\sqrt{2} = 5\sqrt{2}$
    *   **step 4:** $AB + BC = AC$ ($5\sqrt{2} = 5\sqrt{2}$) ಎಂದು ನಾವು ನೋಡಬಹುದು.
*   **result:** $AB + BC = AC$ ಆಗಿರುವುದರಿಂದ, A, B, ಮತ್ತು C ಬಿಂದುಗಳು ಸರಳರೇಖಾಗತವಾಗಿವೆ.

**ಉದಾಹರಣೆ 4**
*   **example_id:** COOR-ADV-2_example_4
*   **difficulty:** intermediate
*   **problem:** (x, y) ಬಿಂದುವು (7, 1) ಮತ್ತು (3, 5) ಬಿಂದುಗಳಿಂದ ಸಮಾನದೂರದಲ್ಲಿದ್ದರೆ, x ಮತ್ತು y ಗಳ ನಡುವೆ ಒಂದು ಸಂಬಂಧವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
*   **steps:**
    *   **step 1:** ದತ್ತ ಬಿಂದುಗಳನ್ನು P(x, y), A(7, 1) ಮತ್ತು B(3, 5) ಎಂದು ಹೆಸರಿಸೋಣ. P ಬಿಂದುವು A ಮತ್ತು B ಗಳಿಂದ ಸಮಾನದೂರದಲ್ಲಿದೆ ಎಂದು ನೀಡಲಾಗಿದೆ. ಅಂದರೆ, $PA = PB$.
    *   **step 2:** ಲೆಕ್ಕಾಚಾರವನ್ನು ಸುಲಭಗೊಳಿಸಲು, ನಾವು $PA^2 = PB^2$ ಎಂದು ತೆಗೆದುಕೊಳ್ಳಬಹುದು. ದೂರ ಸೂತ್ರವನ್ನು ಅನ್ವಯಿಸಿ.
        *   **justification:** ವರ್ಗ ಮಾಡುವುದರಿಂದ ವರ್ಗಮೂಲ (square root) ಚಿಹ್ನೆಯನ್ನು ತೆಗೆದುಹಾಕಬಹುದು.
        *   **calculation:** $(x - 7)^2 + (y - 1)^2 = (x - 3)^2 + (y - 5)^2$
    *   **step 3:** ಸಮೀಕರಣವನ್ನು ವಿಸ್ತರಿಸಿ ಮತ್ತು ಸರಳೀಕರಿಸಿ.
        *   **calculation:**
            $(x^2 - 14x + 49) + (y^2 - 2y + 1) = (x^2 - 6x + 9) + (y^2 - 10y + 25)$
            $x^2 - 14x + y^2 - 2y + 50 = x^2 - 6x + y^2 - 10y + 34$
    *   **step 4:** x ಮತ್ತು y ಪದಗಳನ್ನು ಒಂದು ಕಡೆ ಮತ್ತು ಸ್ಥಿರಾಂಕಗಳನ್ನು ಇನ್ನೊಂದು ಕಡೆಗೆ ತನ್ನಿ.
        *   **calculation:**
            $-14x + 6x - 2y + 10y = 34 - 50$
            $-8x + 8y = -16$
    *   **step 5:** ಸಮೀಕರಣವನ್ನು -8 ರಿಂದ ಭಾಗಿಸಿ.
        *   **calculation:** $x - y = 2$
*   **result:** x ಮತ್ತು y ನಡುವಿನ ಅಪೇಕ್ಷಿತ ಸಂಬಂಧ $x - y = 2$ ಆಗಿದೆ. ಇದು AB ರೇಖಾಖಂಡದ ಲಂಬ ಸಮದ್ವಿಭಾಜಕದ (perpendicular bisector) ಸಮೀಕರಣವಾಗಿದೆ.

    *   **step 2:** P ಬಿಂದುವು A(6, 5) ಮತ್ತು B(-4, 3) ಗಳಿಂದ ಸಮಾನ ದೂರದಲ್ಲಿದೆ. ಅಂದರೆ, $PA = PB$ ಅಥವಾ $PA^2 = PB^2$.

            $36 + y^2 - 10y + 25 = 16 + y^2 - 6y + 9$
            $61 - 10y = 25 - 6y$

        *   **calculation:**

*   **result:** ಅಪೇಕ್ಷಿತ ಬಿಂದು P(0, 9) ಆಗಿದೆ.

[ವಿಶ್ಲೇಷಣೆ]
*   ದೂರ ಸೂತ್ರವು ನಿರ್ದೇಶಾಂಕ ರೇಖಾಗಣಿತದ ಮೂಲಭೂತ ಸಾಧನವಾಗಿದೆ. ಇದು ಬೀಜಗಣಿತ ಮತ್ತು ರೇಖಾಗಣಿತವನ್ನು ಸಂಪರ್ಕಿಸುತ್ತದೆ.
*   ದೂರವು ಯಾವಾಗಲೂ ಧನಾತ್ಮಕ (non-negative)ವಾಗಿರುತ್ತದೆ. ಸೂತ್ರದಲ್ಲಿ ವರ್ಗಮೂಲದ ಧನಾತ್ಮಕ ಮೌಲ್ಯವನ್ನು ಮಾತ್ರ ತೆಗೆದುಕೊಳ್ಳಲಾಗುತ್ತದೆ.
*   $(x_2 - x_1)^2$ ಮತ್ತು $(x_1 - x_2)^2$ ಒಂದೇ ಆಗಿರುವುದರಿಂದ, ಬಿಂದುಗಳನ್ನು ಯಾವುದೇ ಕ್ರಮದಲ್ಲಿ ತೆಗೆದುಕೊಂಡರೂ ಅಂತಿಮ ಉತ್ತರ ಒಂದೇ ಆಗಿರುತ್ತದೆ.
*   **Common Mistake:** ವಿದ್ಯಾರ್ಥಿಗಳು ಸಾಮಾನ್ಯವಾಗಿ ಲೆಕ್ಕಾಚಾರದಲ್ಲಿ, ವಿಶೇಷವಾಗಿ ಋಣಾತ್ಮಕ ನಿರ್ದೇಶಾಂಕಗಳೊಂದಿಗೆ, ಚಿಹ್ನೆಗಳಲ್ಲಿ ತಪ್ಪು ಮಾಡುತ್ತಾರೆ. ಉದಾಹರಣೆಗೆ, $(x_2 - (-x_1))$ ಅನ್ನು $(x_2 - x_1)$ ಎಂದು ತಪ್ಪಾಗಿ ಬರೆಯುವುದು.

[ಸಂಬಂಧ]
*   **ಪೂರ್ವಾಪೇಕ್ಷಿತ ಅವಧಾರಣೆಗಳು:** ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯ, ನೈಜ ಸಂಖ್ಯೆಗಳ ಮೇಲಿನ ಕಾರ್ಯಾಚರಣೆಗಳು, ಬೀಜಗಣಿತದ ವಿಸ್ತರಣೆಗಳು.
*   **ಮುಂದಿನ ಅವಧಾರಣೆಗಳೊಂದಿಗೆ ಸಂಪರ್ಕ:** ಈ ಸೂತ್ರವನ್ನು ಬಳಸಿ ವೃತ್ತದ ಸಮೀಕರಣ, ಲಂಬ ಸಮದ್ವಿಭಾಜಕದ ಸಮೀಕರಣ, ಮತ್ತು ವಿವಿಧ ಜ್ಯಾಮಿತೀಯ ಆಕೃತಿಗಳ ಗುಣಲಕ್ಷಣಗಳನ್ನು ಸಾಧಿಸಲಾಗುತ್ತದೆ. ಇದು ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರದಂತಹ ಮುಂದಿನ ಪರಿಕಲ್ಪನೆಗಳಿಗೆ ಅಡಿಪಾಯವಾಗಿದೆ.

[ಮೂಲ_ಅವಧಾರಣೆ_2]
concept_id: COOR-ADV-2_concept_2
concept_name: ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರ (Section Formula)
domain: ನಿರ್ದೇಶಾಂಕ ರೇಖಾಗಣಿತ

[ಸಾರಾಂಶ]
ಎರಡು ನಿರ್ದಿಷ್ಟ ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು, ಒಂದು ನಿರ್ದಿಷ್ಟ ಅನುಪಾತದಲ್ಲಿ ಆಂತರಿಕವಾಗಿ ವಿಭಾಗಿಸುವ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಕಂಡುಹಿಡಿಯಲು ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು ಬಳಸಲಾಗುತ್ತದೆ. ಈ ಸೂತ್ರವು ಸಮರೂಪಿ ತ್ರಿಭುಜಗಳ ಗುಣಲಕ್ಷಣಗಳನ್ನು ಆಧರಿಸಿದೆ.

[ಸೂತ್ರಗಳು]
**1. ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರ (Section Formula)**
$A(x_1, y_1)$ ಮತ್ತು $B(x_2, y_2)$ ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು $m_1:m_2$ ಅನುಪಾತದಲ್ಲಿ ಆಂತರಿಕವಾಗಿ ವಿಭಾಗಿಸುವ $P(x, y)$ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು.

*   **formula_name:** Section Formula (Internal Division)
*   **latex:** `P(x, y) = \left( \frac{m_1x_2 + m_2x_1}{m_1 + m_2}, \frac{m_1y_2 + m_2y_1}{m_1 + m_2} \right)`
*   **condition:** ಬಿಂದು P ಯು ರೇಖಾಖಂಡ AB ಯ ಮೇಲೆ (ಆಂತರಿಕವಾಗಿ) ಇರಬೇಕು. $m_1, m_2 > 0$.
*   **derived_from:** ಸಮರೂಪಿ ತ್ರಿಭುಜಗಳ ಗುಣಲಕ್ಷಣಗಳು (Similarity of triangles).
*   **derivation:**
    *   **step 1:** $A(x_1, y_1)$, $B(x_2, y_2)$ ಮತ್ತು $P(x, y)$ ಬಿಂದುಗಳಿಂದ x-ಅಕ್ಷಕ್ಕೆ ಲಂಬಗಳನ್ನು ಎಳೆಯಿರಿ. $A$ ಮತ್ತು $P$ ಯಿಂದ $x$-ಅಕ್ಷಕ್ಕೆ ಸಮಾಂತರವಾಗಿ ರೇಖೆಗಳನ್ನು ಎಳೆಯಿರಿ.
    *   **step 2:** ಇದರಿಂದ $\triangle PAQ$ ಮತ್ತು $\triangle BPC$ ಎಂಬ ಎರಡು ಸಮರೂಪಿ ತ್ರಿಭುಜಗಳು ಉಂಟಾಗುತ್ತವೆ ($\triangle PAQ \sim \triangle BPC$).
    *   **step 3:** ಸಮರೂಪಿ ತ್ರಿಭುಜಗಳ ಅನುರೂಪ ಬಾಹುಗಳು ಸಮಾನುಪಾತದಲ್ಲಿರುತ್ತವೆ: $\frac{PA}{BP} = \frac{AQ}{PC} = \frac{PQ}{BC}$.
    *   **step 4:** ದೂರಗಳನ್ನು ನಿರ್ದೇಶಾಂಕಗಳ ರೂಪದಲ್ಲಿ ಬರೆಯಿರಿ: $\frac{m_1}{m_2} = \frac{x - x_1}{x_2 - x} = \frac{y - y_1}{y_2 - y}$.
    *   **step 5:** x-ನಿರ್ದೇಶಾಂಕಕ್ಕಾಗಿ ಸಮೀಕರಣವನ್ನು ಬಿಡಿಸಿ: $m_1(x_2 - x) = m_2(x - x_1) \implies m_1x_2 - m_1x = m_2x - m_2x_1 \implies x(m_1+m_2) = m_1x_2 + m_2x_1$.
    *   **result (x):** $x = \frac{m_1x_2 + m_2x_1}{m_1 + m_2}$.
    *   **step 6:** y-ನಿರ್ದೇಶಾಂಕಕ್ಕಾಗಿ ಸಮೀಕರಣವನ್ನು ಬಿಡಿಸಿ: $m_1(y_2 - y) = m_2(y - y_1) \implies m_1y_2 - m_1y = m_2y - m_2y_1 \implies y(m_1+m_2) = m_1y_2 + m_2y_1$.
    *   **result (y):** $y = \frac{m_1y_2 + m_2y_1}{m_1 + m_2}$.

**2. ಮಧ್ಯಬಿಂದು ಸೂತ್ರ (Midpoint Formula)**
ರೇಖಾಖಂಡವನ್ನು 1:1 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ ಬಿಂದು (ಮಧ್ಯಬಿಂದು).

*   **formula_name:** Midpoint Formula
*   **latex:** `P(x, y) = \left( \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2} \right)`
*   **condition:** P ಯು AB ಯ ನಿಖರ ಮಧ್ಯಬಿಂದುವಾಗಿರಬೇಕು.
*   **derived_from:** ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರದಲ್ಲಿ $m_1 = 1$ ಮತ್ತು $m_2 = 1$ ಎಂದು ಆದೇಶಿಸಿದಾಗ.

[ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
**ಉದಾಹರಣೆ 6**
*   **example_id:** COOR-ADV-2_example_6
*   **difficulty:** basic
*   **problem:** (4, -3) ಮತ್ತು (8, 5) ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು ಆಂತರಿಕವಾಗಿ 3:1 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
*   **steps:**
    *   **step 1:** ದತ್ತಾಂಶವನ್ನು ಗುರುತಿಸಿ: $(x_1, y_1) = (4, -3)$, $(x_2, y_2) = (8, 5)$, $m_1 = 3$, $m_2 = 1$.
    *   **step 2:** ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು ಬಳಸಿ x-ನಿರ್ದೇಶಾಂಕವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
        *   **calculation:** $x = \frac{m_1x_2 + m_2x_1}{m_1 + m_2} = \frac{3(8) + 1(4)}{3 + 1} = \frac{24 + 4}{4} = \frac{28}{4} = 7$.
    *   **step 3:** ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು ಬಳಸಿ y-ನಿರ್ದೇಶಾಂಕವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
        *   **calculation:** $y = \frac{m_1y_2 + m_2y_1}{m_1 + m_2} = \frac{3(5) + 1(-3)}{3 + 1} = \frac{15 - 3}{4} = \frac{12}{4} = 3$.
*   **result:** ಅಪೇಕ್ಷಿತ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು (7, 3).

*   **problem:** A(-6, 10) ಮತ್ತು B(3, -8) ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು (-4, 6) ಬಿಂದುವು ಯಾವ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆ?
*   **steps:**
    *   **step 1:** ವಿಭಾಗಿಸುವ ಬಿಂದು P(-4, 6) ಮತ್ತು ದತ್ತ ಬಿಂದುಗಳು A(-6, 10), B(3, -8). ಅನುಪಾತವು $m_1:m_2$ ಆಗಿರಲಿ.
    *   **step 2:** ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು x-ನಿರ್ದೇಶಾಂಕಕ್ಕೆ ಅನ್ವಯಿಸಿ.
        *   **calculation:** $-4 = \frac{m_1(3) + m_2(-6)}{m_1 + m_2} = \frac{3m_1 - 6m_2}{m_1 + m_2}$.
    *   **step 3:** ಸಮೀಕರಣವನ್ನು $m_1:m_2$ ಅನುಪಾತಕ್ಕಾಗಿ ಬಿಡಿಸಿ.
        *   **calculation:**
            $-4(m_1 + m_2) = 3m_1 - 6m_2$
            $-4m_1 - 4m_2 = 3m_1 - 6m_2$
            $6m_2 - 4m_2 = 3m_1 + 4m_1$
            $2m_2 = 7m_1$
            $\frac{m_1}{m_2} = \frac{2}{7}$.
*   **result:** ಅಪೇಕ್ಷಿತ ಅನುಪಾತ 2:7 ಆಗಿದೆ. (y-ನಿರ್ದೇಶಾಂಕವನ್ನು ಬಳಸಿ ಇದೇ ಉತ್ತರವನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಬಹುದು).

**ಉದಾಹರಣೆ 8**
*   **example_id:** COOR-ADV-2_example_8
*   **difficulty:** intermediate
*   **problem:** A(2, -2) ಮತ್ತು B(-7, 4) ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡದ ತ್ರೈಭಾಜಕ ಬಿಂದುಗಳ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
*   **steps:**
    *   **step 1:** ತ್ರೈಭಾಜಕ ಬಿಂದುಗಳು P ಮತ್ತು Q ಆಗಿರಲಿ. P ಬಿಂದುವು AB ಯನ್ನು 1:2 ಅನುಪಾತದಲ್ಲಿ ಮತ್ತು Q ಬಿಂದುವು 2:1 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತವೆ.
    *   **step 2:** P ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ (ಅನುಪಾತ 1:2).
        *   **calculation:**
            $x_P = \frac{1(-7) + 2(2)}{1+2} = \frac{-7+4}{3} = \frac{-3}{3} = -1$
            $y_P = \frac{1(4) + 2(-2)}{1+2} = \frac{4-4}{3} = \frac{0}{3} = 0$
            ಆದ್ದರಿಂದ P = (-1, 0).
    *   **step 3:** Q ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ (ಅನುಪಾತ 2:1).
        *   **calculation:**
            $x_Q = \frac{2(-7) + 1(2)}{2+1} = \frac{-14+2}{3} = \frac{-12}{3} = -4$
            $y_Q = \frac{2(4) + 1(-2)}{2+1} = \frac{8-2}{3} = \frac{6}{3} = 2$
            ಆದ್ದರಿಂದ Q = (-4, 2).
*   **result:** ತ್ರೈಭಾಜಕ ಬಿಂದುಗಳು (-1, 0) ಮತ್ತು (-4, 2).

**ಉದಾಹರಣೆ 9**
*   **example_id:** COOR-ADV-2_example_9
*   **difficulty:** intermediate
*   **problem:** (5, -6) ಮತ್ತು (-1, -4) ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು y-ಅಕ್ಷವು ಯಾವ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆ? ಛೇದಕ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
*   **steps:**
    *   **step 1:** ಅನುಪಾತವು k:1 ಆಗಿರಲಿ. y-ಅಕ್ಷದ ಮೇಲಿನ ಯಾವುದೇ ಬಿಂದುವಿನ x-ನಿರ್ದೇಶಾಂಕ 0 ಆಗಿರುತ್ತದೆ.
    *   **step 2:** ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು ಬಳಸಿ ವಿಭಾಗಿಸುವ ಬಿಂದುವಿನ x-ನಿರ್ದೇಶಾಂಕವನ್ನು 0 ಗೆ ಸಮೀಕರಿಸಿ.
        *   **calculation:** $x = \frac{k(-1) + 1(5)}{k+1} = \frac{-k+5}{k+1} = 0$.
    *   **step 3:** k ಗಾಗಿ ಸಮೀಕರಣವನ್ನು ಬಿಡಿಸಿ.
        *   **calculation:** $-k + 5 = 0 \implies k = 5$.
    *   **step 4:** ಅನುಪಾತವು 5:1 ಆಗಿದೆ. ಈಗ ಛೇದಕ ಬಿಂದುವಿನ y-ನಿರ್ದೇಶಾಂಕವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
        *   **calculation:** $y = \frac{5(-4) + 1(-6)}{5+1} = \frac{-20-6}{6} = \frac{-26}{6} = -\frac{13}{3}$.
*   **result:** y-ಅಕ್ಷವು ರೇಖಾಖಂಡವನ್ನು 5:1 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆ ಮತ್ತು ಛೇದಕ ಬಿಂದುವು $(0, -13/3)$ ಆಗಿದೆ.

**ಉದಾಹರಣೆ 10**
*   **example_id:** COOR-ADV-2_example_10
*   **difficulty:** advanced
*   **problem:** A(6, 1), B(8, 2), C(9, 4) ಮತ್ತು D(p, 3) ಬಿಂದುಗಳು ಸಮಾಂತರ ಚತುರ್ಭುಜದ ಅನುಕ್ರಮ ಶೃಂಗಗಳಾದರೆ, p ಯ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
*   **steps:**
    *   **step 1:** ಸಮಾಂತರ ಚತುರ್ಭುಜದ ಕರ್ಣಗಳು ಪರಸ್ಪರ ಅರ್ಧಿಸುತ್ತವೆ. ಅಂದರೆ, ಎರಡೂ ಕರ್ಣಗಳ ಮಧ್ಯಬಿಂದು ಒಂದೇ ಆಗಿರುತ್ತದೆ.
    *   **step 2:** ಕರ್ಣ AC ಯ ಮಧ್ಯಬಿಂದುವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
        *   **calculation:** $M_{AC} = \left( \frac{6+9}{2}, \frac{1+4}{2} \right) = \left( \frac{15}{2}, \frac{5}{2} \right)$.
    *   **step 3:** ಕರ್ಣ BD ಯ ಮಧ್ಯಬಿಂದುವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
        *   **calculation:** $M_{BD} = \left( \frac{8+p}{2}, \frac{2+3}{2} \right) = \left( \frac{8+p}{2}, \frac{5}{2} \right)$.
    *   **step 4:** ಎರಡೂ ಮಧ್ಯಬಿಂದುಗಳು ಒಂದೇ ಆಗಿರುವುದರಿಂದ, ಅವುಗಳ x-ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಸಮೀಕರಿಸಿ.
        *   **calculation:** $\frac{15}{2} = \frac{8+p}{2}$.
    *   **step 5:** p ಗಾಗಿ ಸಮೀಕರಣವನ್ನು ಬಿಡಿಸಿ.
        *   **calculation:** $15 = 8 + p \implies p = 15 - 8 = 7$.
*   **result:** p ಯ ಬೆಲೆ 7 ಆಗಿದೆ.

[ದೃಶ್ಯ_ಸಹಾಯ]
*   **type:** diagram
*   **description:** ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರದ ನಿರೂಪಣೆಗೆ ಬಳಸಿದ ಸಾಮಾನ್ಯ ಚಿತ್ರ. A, B ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡದ ಮೇಲೆ P ಬಿಂದುವಿದೆ. ಸಮರೂಪಿ ತ್ರಿಭುಜಗಳನ್ನು ರಚಿಸಲಾಗಿದೆ.
*   **labels:** A(x₁, y₁), B(x₂, y₂), P(x, y), m₁, m₂.
*   **mathematical_significance:** ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರದ ನಿಷ್ಪತ್ತಿಗೆ (derivation) ಜ್ಯಾಮಿತೀಯ ಆಧಾರವನ್ನು ಒದಗಿಸುತ್ತದೆ. (ಚಿತ್ರ 7.10)

[ವಿಶ್ಲೇಷಣೆ]
*   ಈ ಸೂತ್ರವು ಆಂತರಿಕ ವಿಭಾಗಕ್ಕೆ (internal division) ಮಾತ್ರ ಅನ್ವಯಿಸುತ್ತದೆ. ಬಾಹ್ಯ ವಿಭಾಗದ (external division) ಸೂತ್ರವು ಬೇರೆಯಾಗಿದ್ದು, ಅದನ್ನು ಉನ್ನತ ತರಗತಿಗಳಲ್ಲಿ ಕಲಿಯಲಾಗುತ್ತದೆ.
*   ಅನುಪಾತವನ್ನು ಕಂಡುಹಿಡಿಯುವಾಗ, k:1 ಅನುಪಾತವನ್ನು ಬಳಸುವುದು ಲೆಕ್ಕಾಚಾರವನ್ನು ಸರಳಗೊಳಿಸುತ್ತದೆ.
*   **Common Mistake:** ವಿದ್ಯಾರ್ಥಿಗಳು $m_1$ ಮತ್ತು $m_2$ ಅನ್ನು $x_1, x_2$ ಮತ್ತು $y_1, y_2$ ಜೊತೆ ತಪ್ಪಾಗಿ ಜೋಡಿಸುತ್ತಾರೆ. ಸರಿಯಾದ ಜೋಡಣೆ: $m_1$ ಜೊತೆ $(x_2, y_2)$ ಮತ್ತು $m_2$ ಜೊತೆ $(x_1, y_1)$.

[ಸಂಬಂಧ]
*   **ಪೂರ್ವಾಪೇಕ್ಷಿತ ಅವಧಾರಣೆಗಳು:** ಸಮರೂಪಿ ತ್ರಿಭುಜಗಳು, ರೇಖೀಯ ಸಮೀಕರಣಗಳು, ಅನುಪಾತ ಮತ್ತು ಸಮಾನುಪಾತ.
*   **ಮುಂದಿನ ಅವಧಾರಣೆಗಳೊಂದಿಗೆ ಸಂಪರ್ಕ:** ತ್ರಿಭುಜದ ಗುರುತ್ವ ಕೇಂದ್ರ (centroid), ವೃತ್ತಗಳ ಸ್ಪರ್ಶಕಗಳು ಮತ್ತು ಇತರ ಉನ್ನತ ಮಟ್ಟದ ರೇಖಾಗಣಿತದ ಪರಿಕಲ್ಪನೆಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಈ ಸೂತ್ರವು ಅವಶ್ಯಕ.

[ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
*   **gpstr_weightage:** high
*   **MCQ ಸಾಧ್ಯತೆ:** ಮಧ್ಯಬಿಂದುವನ್ನು ಕಂಡುಹಿಡಿಯುವುದು, ನಿರ್ದಿಷ್ಟ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ ಬಿಂದುವನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
*   **2-ಅಂಕ ಪ್ರಶ್ನೆ:** ಅನುಪಾತವನ್ನು ಕಂಡುಹಿಡಿಯುವುದು, ತ್ರೈಭಾಜಕ ಬಿಂದುಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
*   **5-ಅಂಕ ಪ್ರಶ್ನೆ:** ಸಮಾಂತರ ಚತುರ್ಭುಜ, ವಜ್ರಾಕೃತಿಗಳ ಶೃಂಗಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು ಅಥವಾ ಗುಣಲಕ್ಷಣಗಳನ್ನು ಸಾಧಿಸುವುದು.
*   **ಪ್ರಮುಖ trap:** $m_1, m_2$ ಮತ್ತು ನಿರ್ದೇಶಾಂಕಗಳ ತಪ್ಪಾದ ಜೋಡಣೆ. ಲೆಕ್ಕಾಚಾರದಲ್ಲಿ ಚಿಹ್ನೆಗಳ ತಪ್ಪು.
