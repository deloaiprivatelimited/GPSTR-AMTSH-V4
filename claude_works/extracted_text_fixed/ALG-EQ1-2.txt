# ಅಧ್ಯಾಯ 2. ಒಂದು ಚರಾಕ್ಷರವುಳ್ಳ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳು

## 2.1 ಪೀಠಿಕೆ

ಬೀಜಗಣಿತದ ಹಲವು ಪದೋಕ್ತಿಗಳು ಮತ್ತು ಸಮೀಕರಣಗಳನ್ನು ನೀವು ಈಗಾಗಲೇ ಹಿಂದಿನ ತರಗತಿಗಳಲ್ಲಿ ಅಭ್ಯಸಿಸಿದ್ದೀರಿ.

ಈ ರೀತಿ ಚರ್ಚಿಸಿದ ಬೀಜಪದೋಕ್ತಿಗಳ ಕೆಲವು ಉದಾಹರಣೆಗಳು ಕೆಳಕಂಡಂತಿವೆ.
$5x, 2x – 3, 3x + y, 2xy + 5, xyz + x + y + z, x² + 1, y + y²$

ಇದೇ ರೀತಿ ನಾವು ಚರ್ಚಿಸಿದ ಸಮೀಕರಣಗಳು ಕೆಳಕಂಡಂತಿವೆ.
$5x = 25, 2x-3=9, 2y+\frac{5}{2}=\frac{37}{2}, 6z+10=-2$

ಸಮೀಕರಣಗಳಲ್ಲಿ ಮಾತ್ರ ಸಮತೆಯ ಚಿಹ್ನೆ (=)ಯನ್ನು ಬಳಸುತ್ತೇವೆಯೇ ವಿನಃ ಪದೋಕ್ತಿಗಳಲ್ಲಿ ಬಳಸುವುದಿಲ್ಲ.

ಮೇಲೆ ತಿಳಿಸಿದ ಬೀಜಪದೋಕ್ತಿಗಳಲ್ಲಿ ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚಿನ ಚರಾಕ್ಷರಗಳಿಂದ ಕೂಡಿದ ಪದೋಕ್ತಿಗಳೂ ಇವೆ. ಉದಾಹರಣೆಗೆ $2xy + 5$ರಲ್ಲಿ ಎರಡು ಚರಾಕ್ಷರಗಳಿವೆ.

ಅಲ್ಲದೆ, ಈ ರೀತಿ ಬಳಸುವ ಪದೋಕ್ತಿಗಳು ರೇಖಾತ್ಮಕ. ಅಂದರೆ ಅಂತಹ ಪದೋಕ್ತಿಗಳಲ್ಲಿರುವ ಚರಾಕ್ಷರದ ಅತಿ ಹೆಚ್ಚಿನ ಘಾತವು ಒಂದು. ಇಂತಹ ರೇಖಾತ್ಮಕ ಪದೋಕ್ತಿಗಳ ಕೆಲವು ಉದಾಹರಣೆಗಳು ಕೆಳಕಂಡಂತಿವೆ.
$2x, 2x + 1, 3y – 7, 12 – 5z, \frac{5}{4}(x-4)+10$

ಕೆಳಕಂಡ ರೀತಿಯ ಪದೋಕ್ತಿಗಳು ರೇಖಾತ್ಮಕವಲ್ಲ.
$x² + 1, y + y², 1 + z + z^2 + z³$

ಇವುಗಳಲ್ಲಿರುವ ಚರಾಕ್ಷರಗಳು ಒಂದಕ್ಕಿಂತ ಹೆಚ್ಚಿನ ಘಾತಗಳಲ್ಲಿವೆ.

ಇಲ್ಲಿ ನಾವು ಚರ್ಚಿಸುವ ಸಮೀಕರಣಗಳಲ್ಲಿ ಒಂದೇ ಒಂದು ಚರಾಕ್ಷರದ ರೇಖಾತ್ಮಕ ಬೀಜಪದೋಕ್ತಿಗಳನ್ನು ಮಾತ್ರ ಪರಿಗಣಿಸುತ್ತೇವೆ. ಅಂತಹ ಸಮೀಕರಣಗಳಿಗೆ ಒಂದು ಚರಾಕ್ಷರವುಳ್ಳ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳೆಂದು ಹೆಸರು. ಈಗಾಗಲೇ ಅಭ್ಯಸಿಸಿದ ಸರಳ ಸಮೀಕರಣಗಳೆಲ್ಲವೂ ಈ ರೀತಿಯವುಗಳಾಗಿವೆ.

ನಾವು ಇಲ್ಲಿಯವರೆಗೆ ತಿಳಿದಿದ್ದನ್ನು ಸಂಕ್ಷಿಪ್ತವಾಗಿ ಪುನರಾವರ್ತಿಸೋಣ.

(a) ಬೀಜಗಣಿತೀಯ ಸಮೀಕರಣವೆಂದರೆ ಚರಾಕ್ಷರಗಳನ್ನೊಳಗೊಂಡ ಪದೋಕ್ತಿಗಳ ಸಮತೆಯನ್ನು ತಿಳಿಸುವ ಒಂದು ಉಕ್ತಿ. ಅದರಲ್ಲಿ ಸಮತೆಯ ಚಿಹ್ನೆಯಿದೆ. ಈ ಚಿಹ್ನೆಯ ಎಡದಲ್ಲಿರುವುದು ಸಮೀಕರಣದ ಎಡಭಾಗ ಮತ್ತು ಬಲದಲ್ಲಿರುವುದು ಸಮೀಕರಣದ ಬಲಭಾಗ.

(b) ಒಂದು ಸಮೀಕರಣದಲ್ಲಿರುವ ಚರಾಕ್ಷರಗಳಿಗೆ ಕೆಲವು ಬೆಲೆಗಳನ್ನು ಕೊಟ್ಟರೆ ಮಾತ್ರ ಸಮೀಕರಣದ ಎಡಭಾಗ ಮತ್ತು ಬಲಭಾಗಗಳಲ್ಲಿರುವ ಪದೋಕ್ತಿಗಳು ಸಮವಾಗಿರುತ್ತವೆ. ಅಂತಹ ಬೆಲೆಗಳಿಗೆ ಸಮೀಕರಣದ ಪರಿಹಾರಗಳೆಂದು ಹೆಸರು.

--- NOTE ---
$2x - 3 = 7$ ಸಮೀಕರಣದಲ್ಲಿ $x = 5$ ಹಾಕಿದರೆ ಎಡಭಾಗ = $2 \times 5 – 3 = 7$ = ಬಲಭಾಗ. ಆದ್ದರಿಂದ ಈ ಸಮೀಕರಣಕ್ಕೆ $x = 5$ ಪರಿಹಾರವಾಗುತ್ತದೆ. ಆದರೆ $x = 10$ ಹಾಕಿದರೆ ಎಡಭಾಗ $2 \times 10 - 3 = 17$ ಆಗುವುದು. ಆದರೆ ಇದು ಬಲಭಾಗಕ್ಕೆ ಸಮನಾಗಿಲ್ಲ. ಆದ್ದರಿಂದ $x = 10$ ಈ ಸಮೀಕರಣಕ್ಕೆ ಪರಿಹಾರವಲ್ಲ.
--- END NOTE ---

(c) ಒಂದು ಸಮೀಕರಣದ ಪರಿಹಾರವನ್ನು ಕಂಡುಹಿಡಿಯುವುದು ಹೇಗೆಂದು ನೋಡೋಣ. ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆಗಳು ಸಮವಾಗಿದೆಯೆಂದು ಊಹಿಸಿಕೊಳ್ಳುತ್ತೇವೆ. ಅನಂತರ ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆ ಸಮತೆಯನ್ನು ಉಳಿಸಿಕೊಳ್ಳಲು ಎಡಗಡೆ ಮಾಡುವ ಅಂಕಗಣಿತದ ಮೂಲಕ್ರಿಯೆಯನ್ನೇ ಬಲಗಡೆಯಲ್ಲಿಯೂ ಮಾಡುತ್ತೇವೆ. ಈ ರೀತಿ ಕೆಲವು ಸಲ ಮಾಡಿದಾಗ ಪರಿಹಾರ ಸಿಗುತ್ತದೆ.

## 2.2 ಎರಡೂ ಕಡೆ ಚರಾಕ್ಷರವಿರುವ ಸಮೀಕರಣಗಳನ್ನು ಬಿಡಿಸುವ ವಿಧಾನ

ಒಂದು ಸಮೀಕರಣವು ಎರಡು ಪದೋಕ್ತಿಗಳ ಬೆಲೆಗಳ ಸಮತೆಯನ್ನು ಹೇಳುತ್ತದೆ. $2x-3 = 7$ ಸಮೀಕರಣದಲ್ಲಿ ಇರುವ ಎರಡು ಪದೋಕ್ತಿಗಳೆಂದರೆ $2x-3$ ಮತ್ತು $7$. ನಾವು ಇದುವರೆಗೆ ಪರಿಗಣಿಸಿದ ಸಮೀಕರಣಗಳ ಬಲಗಡೆ ಕೇವಲ ಸಂಖ್ಯೆಗಳಿದ್ದವು. ಆದರೆ ಇದು ಯಾವಾಗಲೂ ಹೀಗೆಯೇ ಇರಬೇಕಾಗಿಲ್ಲ. ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆ ಇರುವ ಪದೋಕ್ತಿಗಳಲ್ಲಿ ಚರಾಕ್ಷರಗಳು ಇರಬಹುದು. ಉದಾಹರಣೆಗೆ $2x-3 = x+2$, ಈ ಸಮೀಕರಣದಲ್ಲಿ ಎರಡೂ ಕಡೆ ಇರುವ ಪದೋಕ್ತಿಗಳಲ್ಲಿ ಚರಾಕ್ಷರವಿದೆ. ಎಡಗಡೆ ಇರುವ ಪದೋಕ್ತಿ $2x-3$ ಮತ್ತು ಬಲಗಡೆ ಇರುವ ಪದೋಕ್ತಿ $x+2$.

ಈಗ ಈ ರೀತಿ ಎರಡೂ ಕಡೆ ಚರಾಕ್ಷರವಿರುವ ಸಮೀಕರಣಗಳನ್ನು ಬಿಡಿಸುವ ವಿಧಾನಗಳನ್ನು ಚರ್ಚಿಸೋಣ.

--- EXAMPLE START ---
Source: Class 8, Example 1
Problem: ಬಿಡಿಸಿರಿ $2x - 3 = x + 2$
Solution:
ಪರಿಹಾರ : $2x - 3 = x + 2$
$2x = x + 2 + 3$
$2x - x = x + 5 - x$ (ಎರಡೂ ಕಡೆಯಿಂದ x ಕಳೆದಿದೆ)
$x = 5$
--- EXAMPLE END ---

--- NOTE ---
ಇಲ್ಲಿ ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆಯಿಂದ ಚರಾಕ್ಷರವಿರುವ ಪದವನ್ನು ಕಳೆದಿದ್ದೇವೆ. ಕೇವಲ ಸಂಖ್ಯೆಯನ್ನಲ್ಲ. ಏಕೆಂದರೆ ಚರಾಕ್ಷರಗಳು ಸಹ ಸಂಖ್ಯೆಗಳೇ. ಅಲ್ಲದೆ, ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆಯಿಂದ x ನ್ನು ಕಳೆಯುವುದೂ ಮತ್ತು x ನ್ನು ಎಡಗಡೆಗೆ ವರ್ಗಾಯಿಸುವುದೂ ಒಂದೇ ಎಂಬುದನ್ನು ಗಮನಿಸಿ.
--- END NOTE ---

--- EXAMPLE START ---
Source: Class 8, Example 2
Problem: ಬಿಡಿಸಿರಿ $5x + \frac{7}{2} = \frac{3}{2}x - 14$
Solution:
ಪರಿಹಾರ : ಈ ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆ 2 ರಿಂದ ಗುಣಿಸಿದರೆ
$$2 \times (5x + \frac{7}{2}) = 2 \times (\frac{3}{2}x - 14)$$
$$(2 \times 5x) + (2 \times \frac{7}{2}) = (2 \times \frac{3}{2}x) - (2 \times 14)$$
$$10x + 7 = 3x - 28$$
3x ನ್ನು ಎಡಭಾಗಕ್ಕೆ ತಂದರೆ
$$10x - 3x + 7 = -28$$
$$7x + 7 = -28$$
$$7x = -28 - 7$$
$$7x = -35$$
$$x = \frac{-35}{7}$$
ಅಥವಾ $x = -5$
--- EXAMPLE END ---

## 2.3 ಸಮೀಕರಣಗಳನ್ನು ಸುಲಭ ರೂಪಕ್ಕೆ ತರುವುದು.

--- EXAMPLE START ---
Source: Class 8, Example 3
Problem: ಬಿಡಿಸಿರಿ $\frac{6x + 1}{3} + 1 = \frac{x-3}{6}$
Solution:
ಪರಿಹಾರ : ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆ 6 ರಿಂದ ಗುಣಿಸಿದರೆ

--- NOTE ---
6 ರಿಂದ ಗುಣಿಸಿದ್ದು ಏಕೆ? ಅದು ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆಯ ಛೇದಗಳ ಲ.ಸಾ.ಅ.
--- END NOTE ---

$$\frac{6(6x + 1)}{3} + 6 \times 1 = \frac{6(x-3)}{6}$$
$$2(6x + 1) + 6 = x - 3$$
$$12x + 2 + 6 = x - 3$$
$$12x + 8 = x - 3$$
$$12x - x + 8 = -3$$
$$11x + 8 = -3$$
$$11x = -3 – 8$$
$$11x = -11$$
$$x = -1$$
ಈಗ ಉತ್ತರವನ್ನು ಪರೀಕ್ಷಿಸೋಣ.
ಸಮೀಕರಣದ ಎಡಭಾಗ = $\frac{6(-1) + 1}{3} + 1 = \frac{-6+1}{3} + 1 = \frac{-5}{3} + 1 = \frac{-2}{3}$
ಸಮೀಕರಣದ ಬಲಭಾಗ = $\frac{(-1)-3}{6} = \frac{-4}{6} = \frac{-2}{3}$
ಆದ್ದರಿಂದ ಎರಡೂ ಕಡೆಗಳು ಸಮವಾಗಿವೆ.
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 8, Example 4
Problem: ಬಿಡಿಸಿರಿ $5x – 2 (2x – 7) = 2 (3x - 1) + \frac{7}{2}$
Solution:
ಪರಿಹಾರ : ಆವರಣಗಳನ್ನು ಬಿಡಿಸಿದರೆ ಸಮೀಕರಣದ ಎಡಭಾಗ
$$= 5x - 4x + 14 = x + 14$$
ಸಮೀಕರಣದ ಬಲಭಾಗ $6x - 2 + \frac{7}{2} = 6x - \frac{4}{2} + \frac{7}{2} = 6x + \frac{3}{2}$
ಈಗ ಸಮೀಕರಣವು $x + 14 = 6x + \frac{3}{2}$
$$14 = 6x - x + \frac{3}{2}$$
$$14 = 5x + \frac{3}{2}$$
$$14 - \frac{3}{2} = 5x$$
$$\frac{28-3}{2} = 5x$$
$$\frac{25}{2} = 5x$$
$$x = \frac{25}{2} \times \frac{1}{5} = \frac{5 \times 5}{2 \times 5} = \frac{5}{2}$$

--- NOTE ---
(A thought bubble is shown)
ಕೊಟ್ಟ ಸಮೀಕರಣವನ್ನು ಸುಲಭ ರೂಪಕ್ಕೆ ತಂದ ವಿಧಾನವನ್ನು ಗಮನಿಸಿ
--- END NOTE ---

ಆದ್ದರಿಂದ ಸಮೀಕರಣದ ಪರಿಹಾರವು $x = \frac{5}{2}$.
ಈಗ ಈ ಬೆಲೆಯನ್ನು ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆ ಹಾಕಿ ಪರೀಕ್ಷಿಸೋಣ.
ಸಮೀಕರಣದ ಎಡಭಾಗ = $5(\frac{5}{2}) - 2(2 \times \frac{5}{2} - 7)$
$$= \frac{25}{2} - 2(5-7) = \frac{25}{2} - 2(-2)$$
$$= \frac{25}{2} + 4 = \frac{25+8}{2} = \frac{33}{2}$$
ಸಮೀಕರಣದ ಬಲಭಾಗ = $2(3 \times \frac{5}{2} - 1) + \frac{7}{2}$
$$= 2(\frac{15}{2} - 1) + \frac{7}{2} = 2(\frac{15-2}{2}) + \frac{7}{2} = 2(\frac{13}{2}) + \frac{7}{2}$$
$$= 13 + \frac{7}{2} = \frac{26+7}{2} = \frac{33}{2}$$
$\therefore$ ಸಮೀಕರಣದ ಎಡಭಾಗ = ಬಲಭಾಗ
--- EXAMPLE END ---

### ಅಭ್ಯಾಸ 2.2

ಕೆಳಗಿನ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳನ್ನು ಬಿಡಿಸಿರಿ.
1. $\frac{x}{2} - \frac{1}{5} = \frac{x}{3} + \frac{1}{4}$
2. $\frac{n}{2} - \frac{3n}{4} + \frac{5n}{6} = 21$
3. $x + 7 - \frac{8x}{3} = \frac{17}{6} - \frac{5x}{2}$
4. $\frac{x-5}{3} = \frac{x-3}{5}$
5. $\frac{3t-2}{4} - \frac{2t+3}{3} = \frac{2}{3} - t$
6. $m - \frac{m-1}{2} = 1 - \frac{m-2}{3}$

ಕೆಳಗಿನ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳನ್ನು ಸಂಕ್ಷೇಪಿಸಿ ಬಿಡಿಸಿರಿ.
7. $3(t-3) = 5(2t + 1)$
8. $15(y – 4) −2 (y −9) + 5(y + 6) = 0$
9. $3(5z-7) - 2(9z – 11) = 4(8z – 13) – 17$
10. $0.25(4f-3) = 0.05(10f-9)$

### ಸಾರಾಂಶ

1. ಚರಾಕ್ಷರಗಳನ್ನೊಳಗೊಂಡ ಸಮತೆಯೇ ಬೀಜಗಣಿತೀಯ ಸಮೀಕರಣ. ಸಮ ಚಿಹ್ನೆಯ ಒಂದು ಕಡೆ ಇರುವ ಪದೋಕ್ತಿಯ ಬೆಲೆಯು ಇನ್ನೊಂದು ಕಡೆಯ ಪದೋಕ್ತಿಯ ಬೆಲೆಗೆ ಸಮನಾಗಿರುವುದೆಂದು ಅದು ತಿಳಿಸುತ್ತದೆ.
2. 6 ಮತ್ತು 7 ನೇ ತರಗತಿಗಳಲ್ಲಿ ನಾವು ಅಭ್ಯಸಿಸಿದ ಸಮೀಕರಣಗಳು ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳು. ಅವುಗಳಲ್ಲಿ ಕೇವಲ ಒಂದೇ ಒಂದು ಚರಾಕ್ಷರವಿರುವ ಪದೋಕ್ತಿಗಳಿದ್ದು ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳಾಗಿರುವುವು. ಅಂದರೆ ಅವುಗಳಲ್ಲಿನ ಚರಾಕ್ಷರದ ಅತ್ಯಂತ ಹೆಚ್ಚಿನ ಘಾತ 1 ಆಗಿರುವುದು.
3. ರೇಖಾತ್ಮಕ ಸಮೀಕರಣದ ಎರಡೂ ಕಡೆ ರೇಖಾತ್ಮಕ ಪದೋಕ್ತಿಗಳಿರಬಹುದು. 6 ಮತ್ತು 7ನೇ ತರಗತಿಯಲ್ಲಿ ನೀವು ಅಭ್ಯಸಿಸಿದ ಸಮೀಕರಣಗಳಲ್ಲಿ ಒಂದು ಕಡೆ ಕೇವಲ ಒಂದು ಸಂಖ್ಯೆ ಇತ್ತು.
4. ಸಂಖ್ಯೆಗಳಂತೆಯೇ ಚರಾಕ್ಷರಗಳನ್ನು ಸಹ ಒಂದು ಕಡೆಯಿಂದ ಇನ್ನೊಂದು ಕಡೆಗೆ ವರ್ಗಾಯಿಸಬಹುದು.
5. ಸಮೀಕರಣಗಳನ್ನು ಬಿಡಿಸುವುದಕ್ಕೆ ಮುನ್ನ ಅದರಲ್ಲಿರುವ ಪದೋಕ್ತಿಗಳನ್ನು ಕೆಲವು ಸಲ ನಾವು ಎಂದಿನ ವಿಧಾನಗಳಲ್ಲಿಯೇ ಸುಲಭೀಕರಿಸಬೇಕಾಗುತ್ತದೆ. ಕೆಲವು ಸಮೀಕರಣಗಳು ಆರಂಭದಲ್ಲಿ ರೇಖಾತ್ಮಕವಾಗಿ ಕಾಣದಿದ್ದರೂ ಅವುಗಳ ಎರಡೂ ಕಡೆಗಳನ್ನು ಸೂಕ್ತ ಪದೋಕ್ತಿಯಿಂದ ಗುಣಿಸುವುದರ ಮೂಲಕ ರೇಖಾತ್ಮಕವನ್ನಾಗಿ ರೂಪಾಂತರ ಮಾಡಬಹುದು.
6. ವಿವಿಧ ಅನ್ವಯಗಳಲ್ಲಿ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳ ಉಪಯೋಗ ಪಡೆಯಬಹುದು. ಸಂಖ್ಯೆಗಳು, ಆಕಾರಗಳ ಸುತ್ತಳತೆಗಳು, ಹಣದ ನೋಟುಗಳನ್ನೊಳಗೊಂಡ ವಿವಿಧ ಸಮಸ್ಯೆಗಳನ್ನು ಬಿಡಿಸಲು ನಾವು ಈ ಉಪಯೋಗವನ್ನು ಪಡೆಯಬಹುದು.
