# ನಿರ್ದೇಶಾಂಕ ರೇಖಾಗಣಿತ

## 7.1 ಪೀಠಿಕೆ
ಒಂದು ಸಮತಲದ ಮೇಲೆ ಒಂದು ಬಿಂದುವನ್ನು ಗುರುತಿಸಲು, ನಮಗೆ ಒಂದು ಜೊತೆ ನಿರ್ದೇಶಾಂಕ ಅಕ್ಷಗಳು ಬೇಕಾಗುತ್ತವೆ ಎಂಬುದನ್ನು ನೀವು IX ತರಗತಿಯಲ್ಲಿ ಅಧ್ಯಯನ ಮಾಡಿರುವಿರಿ. y - ಅಕ್ಷದಿಂದ ಬಿಂದುವಿಗಿರುವ ದೂರವನ್ನು ಅದರ x – ನಿರ್ದೇಶಾಂಕ ಅಥವಾ ಕ್ಷಿತಿಜ ದೂರ ಎನ್ನುತ್ತೇವೆ. x – ಅಕ್ಷದಿಂದ ಬಿಂದುವಿಗಿರುವ ದೂರವನ್ನು ಅದರ y – ನಿರ್ದೇಶಾಂಕ ಅಥವಾ ಲಂಬದೂರ ಎನ್ನುತ್ತೇವೆ. x – ಅಕ್ಷದ ಮೇಲಿರುವ ಒಂದು ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು $(x, 0)$ ರೂಪದಲ್ಲಿರುತ್ತವೆ ಹಾಗೂ y – ಅಕ್ಷದ ಮೇಲಿರುವ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು $(0, y)$ ರೂಪದಲ್ಲಿರುತ್ತವೆ.

ಇಲ್ಲಿ ನಿಮಗೊಂದು ಆಟವಿದೆ. ಗ್ರಾಫ್ ಹಾಳೆಯಲ್ಲಿ ಒಂದು ಜೊತೆ ಪರಸ್ಪರ ಲಂಬವಾಗಿರುವ ಅಕ್ಷಗಳನ್ನು ಎಳೆಯಿರಿ. ಈಗ ಈ ಮುಂದಿನ ಬಿಂದುಗಳನ್ನು ಗುರುತಿಸಿ, ಸೂಚನೆಯಂತೆ ಅವುಗಳನ್ನು ಸೇರಿಸುತ್ತಾ ಹೋಗಿ, A (4, 8) ಬಿಂದುವನ್ನು B(3, 9) ಕ್ಕೆ, C(3, 8)ಕ್ಕೆ, D(1, 6)ಕ್ಕೆ, E(1, 5)ಕ್ಕೆ, F (3, 3)ಕ್ಕೆ, G (6, 3)ಕ್ಕೆ, H (8, 5)ಕ್ಕೆ, I (8, 6)ಕ್ಕೆ, J (6, 8)ಕ್ಕೆ, K (6, 9)ಕ್ಕೆ, L (5, 8)ಕ್ಕೆ, A ಗೆ ಸೇರಿಸಿ. ಆ ಬಳಿಕ, P (3.5, 7), Q (3, 6) ಮತ್ತು R (4, 6) ಬಿಂದುಗಳನ್ನು ಒಂದು ತ್ರಿಭುಜ ಉಂಟಾಗುವಂತೆ ಸೇರಿಸಿ. X (5.5, 7), Y (5, 6) ಮತ್ತು Z (6, 6) ಬಿಂದುಗಳನ್ನೂ ಕೂಡಾ ಒಂದು ತ್ರಿಭುಜ ಉಂಟಾಗುವಂತೆ ಸೇರಿಸಿ. ಈಗ ತ್ರಿಭುಜ ಉಂಟಾಗುವಂತೆ S (4, 5), T (4.5, 4) ಮತ್ತು U (5, 5) ಬಿಂದುಗಳಿಗೆ ಸೇರಿಸಿ ಹಾಗೂ U ನ್ನು (9, 5) ಮತ್ತು (9, 6) ಬಿಂದುಗಳಿಗೆ ಸೇರಿಸಿ. ನಿಮಗೆ ಯಾವ ಚಿತ್ರ ದೊರೆಯಿತು?

$ax + by + c = 0$ ರೂಪದಲ್ಲಿರುವ (a, b ಗಳು ಏಕಕಾಲಕ್ಕೆ ಸೊನ್ನೆಗಳಲ್ಲ), ಎರಡು ಚರಾಕ್ಷರಗಳಿರುವ ರೇಖಾತ್ಮಕ ಸಮೀಕರಣಗಳು, ನಕ್ಷಾಕ್ರಮದಲ್ಲಿ ಪ್ರತಿನಿಧಿಸುವಾಗ ಸರಳರೇಖೆಯನ್ನುಂಟು ಮಾಡುತ್ತವೆ ಎಂಬುದನ್ನು ನೀವು ನೋಡಿದ್ದೀರಿ. ಅಲ್ಲದೆ, ಅಧ್ಯಾಯ 2 ರಲ್ಲಿ $y = ax^2 + bx + c$ $(a \neq 0)$ ಇದರ ನಕ್ಷೆಯು ಒಂದು ಪರವಲಯ ಎಂಬುದನ್ನು ಕೂಡಾ ನೀವು ನೋಡಲಿರುವಿರಿ. ವಾಸ್ತವದಲ್ಲಿ, ರೇಖಾಗಣಿತದ ಆಕೃತಿಗಳ ಅಧ್ಯಯನಕ್ಕೆ ಒಂದು ಬೀಜಗಣಿತೀಯ ಸಾಧನವಾಗಿ, ನಿರ್ದೇಶಾಂಕ ರೇಖಾಗಣಿತವು ಅಭಿವೃದ್ಧಿ ಹೊಂದಿತು. ಬೀಜಗಣಿತವನ್ನುಪಯೋಗಿಸಿ, ರೇಖಾಗಣಿತವನ್ನು ಅಭ್ಯಾಸಮಾಡಲು ಮತ್ತು ರೇಖಾಗಣಿತದ ಸಹಾಯದಿಂದ ಬೀಜಗಣಿತವನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಇದು ನಮಗೆ ಸಹಾಯ ಮಾಡುತ್ತದೆ. ಇದರಿಂದಾಗಿ, ಭೌತಶಾಸ್ತ್ರ, ಇಂಜಿನಿಯರಿಂಗ್, ಸಮುದ್ರಯಾನ, ಭೂಕಂಪ ಶಾಸ್ತ್ರ ಮತ್ತು ಕಲೆಗಳಂತಹ ಅನೇಕ ಕ್ಷೇತ್ರಗಳಲ್ಲಿ ನಿರ್ದೇಶಾಂಕ ರೇಖಾಗಣಿತವು ವಿಪುಲವಾಗಿ ಅನ್ವಯಿಸಲ್ಪಡುತ್ತದೆ.

ಈ ಅಧ್ಯಾಯದಲ್ಲಿ, ಎರಡು ಬಿಂದುಗಳ ನಿರ್ದೇಶಾಂಕಗಳು ತಿಳಿದಿದ್ದಾಗ, ಆ ಬಿಂದುಗಳ ನಡುವಿನ ದೂರವನ್ನು ಹೇಗೆ ಕಂಡುಹಿಡಿಯಬುದೆಂಬುದನ್ನು ನೀವು ಅಧ್ಯಯನ ಮಾಡುವಿರಿ. ಎರಡು ದತ್ತ ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು ಒಂದು ದತ್ತ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಹೇಗೆ ಕಂಡುಹಿಡಿಯಬಹುದು ಎಂಬುದನ್ನೂ ಕೂಡಾ ನೀವು ಅಭ್ಯಾಸ ಮಾಡುವಿರಿ.

## 7.2 ದೂರಸೂತ್ರ
ಕೆಳಗಿನ ಸಂದರ್ಭವನ್ನು ಪರಿಗಣಿಸೋಣ:

B ಎಂಬ ನಗರವು, A ಎಂಬ ನಗರದಿಂದ 36km ಪೂರ್ವದಲ್ಲಿ ಮತ್ತು 15km ಉತ್ತರದಲ್ಲಿದೆ. ವಾಸ್ತವವಾಗಿ ಅಳತೆ ಮಾಡದೆ, A ನಗರದಿಂದ B ನಗರಕ್ಕಿರುವ ದೂರವನ್ನು ಹೇಗೆ ಕಂಡುಹಿಡಿಯುವಿರಿ? ನಾವು ನೋಡೋಣ. ಈ ಸಂದರ್ಭವನ್ನು ಚಿತ್ರ 7.1 ರಲ್ಲಿ ತೋರಿಸಿರುವಂತೆ, ನಕ್ಷೆಯ ಮೂಲಕ ಪ್ರತಿನಿಧಿಸಬಹುದು. ಈ ದೂರವನ್ನು ಲೆಕ್ಕ ಮಾಡಲು ನೀವು ಪೈಥಾಗೊರಸ್‌ನ ಪ್ರಮೇಯವನ್ನು ಉಪಯೋಗಿಸಿಕೊಳ್ಳಬಹುದು.

ಈಗ, ಎರಡು ಬಿಂದುಗಳು x - ಅಕ್ಷದ ಮೇಲೆ ಇವೆಯೆಂದುಕೊಳ್ಳೋಣ. ಅವುಗಳ ನಡುವಿನ ದೂರವನ್ನು ನಾವು ಕಂಡುಹಿಡಿಯಬಲ್ಲೆವೆ? ಉದಾಹರಣೆಗೆ, ಚಿತ್ರ 7.2 ರಲ್ಲಿ ತೋರಿಸಿರುವಂತೆ, A (4, 0) ಮತ್ತು B (6, 0) ಬಿಂದುಗಳನ್ನು ಪರಿಗಣಿಸಿ. A ಮತ್ತು B ಬಿಂದುಗಳು x - ಅಕ್ಷದ ಮೇಲಿವೆ.

ಚಿತ್ರದಿಂದ, OA = 4 ಮಾನಗಳು ಮತ್ತು OB = 6 ಮಾನಗಳು ಎಂಬುದನ್ನು ನೀವು ನೋಡಬಹುದು. ಆದ್ದರಿಂದ, A ಯಿಂದ B ಗಿರುವ ದೂರ, ಅಂದರೆ, $AB = OB - OA = 6 - 4 = 2$ ಮಾನಗಳು.

ಹೀಗೆ, ಎರಡು ಬಿಂದುಗಳು x - ಅಕ್ಷದ ಮೇಲಿದ್ದರೆ, ಅವುಗಳ ನಡುವಿನ ದೂರವನ್ನು ನಾವು ಸುಲಭವಾಗಿ ಕಂಡುಹಿಡಿಯಬಹುದು.

ಈಗ, y - ಅಕ್ಷದ ಮೇಲಿರುವ ಎರಡು ಬಿಂದುಗಳನ್ನು ನಾವು ತೆಗೆದುಕೊಳ್ಳೋಣ. ಅವುಗಳ ನಡುವಿನ ದೂರವನ್ನು ಕಂಡುಹಿಡಿಯಲು ಸಾಧ್ಯವೇ? C (0, 3) ಮತ್ತು D (0, 8) ಬಿಂದುಗಳು y - ಅಕ್ಷದ ಮೇಲಿದ್ದರೆ, ಮೊದಲಿನ ರೀತಿಯಲ್ಲೇ ನಾವು ಕಂಡುಹಿಡಿಯಬಹುದು. ಎಂದರೆ, $CD = 8 - 3 = 5$ ಮಾನಗಳು (ಚಿತ್ರ 7.2 ನ್ನು ನೋಡಿ.)

ಮುಂದೆ A ಯಿಂದ C ಗಿರುವ ದೂರವನ್ನು (ಚಿತ್ರ 7.2 ರಲ್ಲಿ) ಕಂಡುಹಿಡಿಯಲು ನಿಮಗೆ ಸಾಧ್ಯವೇ? OA = 4 ಮಾನಗಳು ಮತ್ತು OC = 3 ಮಾನಗಳು ಆಗಿರುವುದರಿಂದ, C ಯಿಂದ A ಗಿರುವ ದೂರ ಅಂದರೆ $AC = \sqrt{3^2 + 4^2} = 5$ ಮಾನಗಳು. ಅದೇ ರೀತಿ, D ಯಿಂದ B ಗಿರುವ ದೂರ $BD = \sqrt{(6-0)^2 + (0-8)^2} = \sqrt{36+64} = \sqrt{100} = 10$ ಮಾನಗಳು ಎಂದು ನೀವು ಕಂಡುಹಿಡಿಯಬಹುದು.

ಈಗ ನಿರ್ದೇಶಾಂಕ ಅಕ್ಷಗಳ ಮೇಲೆ ಇಲ್ಲದ ಎರಡು ಬಿಂದುಗಳನ್ನು ಪರಿಗಣಿಸಿದರೆ, ಅವುಗಳ ನಡುವಿನ ದೂರವನ್ನು ಕಂಡುಹಿಡಿಯಲು ಸಾಧ್ಯವೇ? ಹೌದು. ಇದಕ್ಕಾಗಿ ನಾವು ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯವನ್ನು ಉಪಯೋಗಿಸಿಕೊಳ್ಳುತ್ತೇವೆ. ಒಂದು ಉದಾಹರಣೆಯನ್ನು ನೋಡೋಣ.

ಚಿತ್ರ 7.3 ರಲ್ಲಿ, P (4, 6) ಮತ್ತು Q (6, 8) ಬಿಂದುಗಳು ಒಂದನೇ ಚತುರ್ಥಕದಲ್ಲಿವೆ. ಅವುಗಳ ನಡುವಿನ ದೂರವನ್ನು ಕಂಡುಹಿಡಿಯಲು ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯವನ್ನು ಹೇಗೆ ಉಪಯೋಗಿಸಬಹುದು? P ಮತ್ತು Q ಗಳಿಂದ x- ಅಕ್ಷಕ್ಕೆ ಕ್ರಮವಾಗಿ PR ಮತ್ತು QS ಎಂಬ ಲಂಬಗಳನ್ನು ಎಳೆಯೋಣ. PT ಯನ್ನು QS ಗೆ ಲಂಬವಾಗಿ ಎಳೆಯೋಣ. ಆಗ R ಮತ್ತು S ಗಳ ನಿರ್ದೇಶಾಂಕಗಳು ಕ್ರಮವಾಗಿ (4, 0) ಮತ್ತು (6, 0). ಆದ್ದರಿಂದ RS = 2 ಮಾನಗಳು, QS = 8 ಮಾನಗಳು ಮತ್ತು TS = PR = 6 ಮಾನಗಳು.

ಆದ್ದರಿಂದ, $QT = QS - TS = 8 - 6 = 2$ ಮಾನಗಳು ಮತ್ತು $PT = RS = 2$ ಮಾನಗಳು.

ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯವನ್ನುಪಯೋಗಿಸಿದಾಗ,
$$PQ^2 = PT^2 + QT^2$$
$$= 2^2 + 2^2 = 8$$
$$PQ = \sqrt{8} = 2\sqrt{2} \text{ ಮಾನಗಳು}$$

ಬೇರೆ ಬೇರೆ ಚತುರ್ಥಕಗಳಲ್ಲಿರುವ ಎರಡು ಬಿಂದುಗಳ ನಡುವಿನ ದೂರವನ್ನು ಹೇಗೆ ಕಂಡುಹಿಡಿಯಬಹುದು?

P (6, 4) ಮತ್ತು Q (-5, -3) ಎಂಬ ಬಿಂದುಗಳನ್ನು ಪರಿಗಣಿಸಿ (ಚಿತ್ರ 7.4 ನ್ನು ನೋಡಿ). x - ಅಕ್ಷಕ್ಕೆ ಲಂಬವಾಗಿ QS ನ್ನು ಎಳೆಯಿರಿ. QS ನ್ನು ವಿಸ್ತರಿಸಿ. PT ⊥ QS ನ್ನು ಎಳೆಯಿರಿ. ಇದು y - ಅಕ್ಷವನ್ನು R ನಲ್ಲಿ ಛೇದಿಸಲಿ.

ಆಗ PT = 11 ಮಾನಗಳು ಮತ್ತು QT = 7 ಮಾನಗಳು. (ಏಕೆ)?
$\triangle PTQ$ ನಲ್ಲಿ, ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯವನ್ನು ಬಳಸಿದಾಗ $PQ = \sqrt{11^2 + 7^2} = \sqrt{121+49} = \sqrt{170}$ ಮಾನಗಳು.

P $(x_1, y_1)$ ಮತ್ತು Q $(x_2, y_2)$ ಎಂಬ ಯಾವುದಾದರೂ ಎರಡು ಬಿಂದುಗಳ ನಡುವಿನ ದೂರವನ್ನು ನಾವೀಗ ಕಂಡುಹಿಡಿಯೋಣ.
x - ಅಕ್ಷಕ್ಕೆ PR ಮತ್ತು QS ಎಂಬ ಲಂಬಗಳನ್ನು ಎಳೆಯಿರಿ. PT ⊥ QS ಎಳೆಯಿರಿ. (ಚಿತ್ರ 7.5 ನ್ನು ನೋಡಿ) ಆಗ, $OR = x_1$, $OS = x_2$ ಆದ್ದರಿಂದ, $RS = x_2 - x_1 = PT$. ಅಲ್ಲದೆ, $SQ = y_2$, $ST = PR = y_1$ ಆದ್ದರಿಂದ, $QT = y_2 - y_1$.
$\triangle PTQ$ ನಲ್ಲಿ ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯವನ್ನು ಅನ್ವಯಿಸಿದಾಗ,
$$PQ^2 = PT^2 + QT^2$$
$$= (x_2 - x_1)^2 + (y_2 - y_1)^2$$
ಆದ್ದರಿಂದ,
$$PQ = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

ದೂರವು ಎಂದಿಗೂ ಋಣಾತ್ಮಕವಲ್ಲದಿರುವುದರಿಂದ ನಾವು ಧನಾತ್ಮಕ ವರ್ಗಮೂಲವನ್ನು ಮಾತ್ರ ತೆಗೆದುಕೊಳ್ಳುತ್ತೇವೆ ಎಂಬುದನ್ನು ಗಮನಿಸಿ. ಆದ್ದರಿಂದ, P $(x_1, y_1)$ ಮತ್ತು Q $(x_2, y_2)$ ಬಿಂದುಗಳ ನಡುವಿನ ದೂರವು,
$$PQ = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$
ಇದನ್ನು 'ದೂರಸೂತ್ರ' ಎನ್ನುತ್ತೇವೆ

Source: Class 10, Example 1 (page 112)
Problem: (3, 2), (-2, -3) ಮತ್ತು (2, 3) ಬಿಂದುಗಳಿಂದ ತ್ರಿಭುಜ ಉಂಟಾಗುತ್ತದೆಯೆ? ಹೌದು ಎಂದಾದರೆ, ಉಂಟಾದ ತ್ರಿಭುಜದ ವಿಧವನ್ನು ಹೆಸರಿಸಿ.
Solution:
ಪರಿಹಾರ: PQ, QR ಮತ್ತು PR ದೂರಗಳನ್ನು ಕಂಡುಹಿಡಿಯಲು ದೂರಸೂತ್ರವನ್ನು ಅನ್ವಯಿಸೋಣ. ಇಲ್ಲಿ P (3, 2), Q (-2, -3) ಮತ್ತು R (2, 3) ದತ್ತ ಬಿಂದುಗಳು.
$$PQ = \sqrt{(3 - (-2))^2 + (2 - (-3))^2} = \sqrt{5^2 + 5^2} = \sqrt{50} = 7.07 \text{ (ಅಂದಾಜು)}$$
$$QR = \sqrt{(-2-2)^2 + (-3-3)^2} = \sqrt{(-4)^2 + (-6)^2} = \sqrt{16+36} = \sqrt{52} = 7.21 \text{ (ಅಂದಾಜು)}$$
$$PR = \sqrt{(3 - 2)^2 + (2 - 3)^2} = \sqrt{1^2 + (-1)^2} = \sqrt{2} = 1.41 \text{ (ಅಂದಾಜು)}$$
ಇವುಗಳಲ್ಲಿ ಯಾವುದೇ ಎರಡು ದೂರಗಳ ಮೊತ್ತವು, ಮೂರನೇ ದೂರಕ್ಕಿಂತ ಹೆಚ್ಚಾಗಿರುವುದರಿಂದ, P, Q ಮತ್ತು R ಬಿಂದುಗಳು ತ್ರಿಭುಜವನ್ನುಂಟುಮಾಡುತ್ತವೆ.
ಅಲ್ಲದೆ, $PQ^2 + PR^2 = (\sqrt{50})^2 + (\sqrt{2})^2 = 50 + 2 = 52 = (\sqrt{52})^2 = QR^2$. ಪೈಥಾಗೊರಸ್ ಪ್ರಮೇಯದ ವಿಲೋಮದ ಪ್ರಕಾರ $\angle P = 90^\circ$ ಎಂದು ತಿಳಿಯಬಹುದು. ಆದ್ದರಿಂದ PQR ಒಂದು ಲಂಬಕೋನ ತ್ರಿಭುಜ.

## 7.3 ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರ
ವಿಭಾಗ 7.2ರ ಸಂದರ್ಭವನ್ನು ನೆನಪಿಸಿಕೊಳ್ಳೋಣ. ಒಂದು ದೂರವಾಣಿ ಸಂಸ್ಥೆಯು A ಮತ್ತು B ಗಳ ನಡುವೆ P ಬಿಂದುವಿನಲ್ಲಿ ಒಂದು ಪ್ರಸಾರ ಗೋಪುರ (ಟವರ್)ವನ್ನು ಹಾಕಲು ಬಯಸುತ್ತದೆ. ಟವರ್‌ಗೆ B ಯಿಂದ ಇರುವ ದೂರವು A ಯಿಂದ ಇರುವ ದೂರದ ಎರಡು ಪಟ್ಟು ಆಗಿರಬೇಕು. P ಯು AB ಯ ಮೇಲಿದ್ದರೆ, ಅದು AB ಯನ್ನು 1:2 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆ. (ಚಿತ್ರ 7.9 ನ್ನು ನೋಡಿರಿ.) ನಾವು A ಯನ್ನು ಮೂಲಬಿಂದು 'O' ಎಂದು ಮತ್ತು 1 km ನ್ನು ಎರಡೂ ಅಕ್ಷಗಳ ಮೇಲೆ 1 ಮಾನವೆಂದು ತೆಗೆದುಕೊಂಡರೆ, B ಯ ನಿರ್ದೇಶಾಂಕಗಳು (36, 15) ಎಂದಾಗುತ್ತವೆ. ಟವರ್‌ನ ಸ್ಥಾನವನ್ನು ಗುರುತಿಸಲು P ಯ ನಿರ್ದೇಶಾಂಕಗಳು ನಮಗೆ ಗೊತ್ತಿರಬೇಕು. ಈ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ನಾವು ಕಂಡುಹಿಡಿಯುವುದು ಹೇಗೆ?

--- FIGURE ---
Description: A diagram illustrating the setup for the section formula problem. A right-angled triangle is formed with the origin A, point B(36, 15), and a point E on the x-axis. Point P(x,y) lies on the hypotenuse AB.
Labels: Point A is at the origin O. Point B is at (36, 15). Point P(x, y) is on the line segment AB. A vertical line from B meets the x-axis at E. A vertical line from P meets the x-axis at D. A horizontal line from P meets BE at C. The figure is labeled "ಚಿತ್ರ 7.9".
Mathematical significance: This sets up a specific problem to introduce the concept and derivation of the section formula.
--- END FIGURE ---

P ಯ ನಿರ್ದೇಶಾಂಕಗಳು (x, y) ಆಗಿರಲಿ. $\triangle APD$ ಮತ್ತು $\triangle ABE$ ಗಳು ಸಮರೂಪಿಗಳಾಗಿವೆ. (ಚಿತ್ರ 7.9 ರಲ್ಲಿ PD || BE).
ಆದ್ದರಿಂದ, $\frac{AP}{AB} = \frac{AD}{AE} = \frac{PD}{BE}$
$\frac{1}{3} = \frac{x}{36} = \frac{y}{15}$
ಈ ಸಮೀಕರಣಗಳಿಂದ ಸಿಗುವುದೆಂದರೆ, x = 12 ಮತ್ತು y = 5.
P (12, 5) ಎಂಬುದು ಟವರ್‌ನ ಸ್ಥಾನ.

ಮೇಲಿನ ಉದಾಹರಣೆಯಿಂದ ಒಂದು ಸಾಮಾನ್ಯ ಸೂತ್ರವನ್ನು ಕಂಡುಹಿಡಿಯೋಣ.
A $(x_1, y_1)$ ಮತ್ತು B $(x_2, y_2)$ ಎಂಬ ಯಾವುದೇ ಎರಡು ಬಿಂದುಗಳನ್ನು ಪರಿಗಣಿಸಿ ಮತ್ತು P (x, y) ಯು AB ಯನ್ನು ಆಂತರಿಕವಾಗಿ $m_1 : m_2$ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆಂದು ಭಾವಿಸಿ. ಅಂದರೆ $\frac{PA}{PB} = \frac{m_1}{m_2}$ (ಚಿತ್ರ 7.10 ನ್ನು ನೋಡಿರಿ.)

--- FIGURE ---
Description: A general diagram for deriving the section formula.
Labels: The x-axis and y-axis are shown. Point A is $(x_1, y_1)$, Point B is $(x_2, y_2)$. Point P(x, y) lies on the segment AB. The ratio of division is shown as $m_1$ (for AP) and $m_2$ (for PB). Perpendiculars AR, PS, BT are drawn to the x-axis. Lines AQ and PC are drawn parallel to the x-axis, forming two similar right-angled triangles, $\triangle APQ$ and $\triangle BPC$. The figure is labeled "ಚಿತ್ರ 7.10".
Mathematical significance: This figure illustrates the geometric construction used to derive the section formula.
--- END FIGURE ---

x - ಅಕ್ಷಕ್ಕೆ AR, PS ಮತ್ತು BT ಲಂಬಗಳನ್ನು ಎಳೆಯಿರಿ. x- ಅಕ್ಷಕ್ಕೆ ಸಮಾಂತರವಾಗಿ AQ ಮತ್ತು PC ಗಳನ್ನು ಎಳೆಯಿರಿ. ಆಗ ಕೋನ ಕೋನ ಸಮರೂಪತೆಯ ನಿರ್ಧಾರಕ ಗುಣದಿಂದ
$$\triangle PAQ \sim \triangle BPC$$
ಆದ್ದರಿಂದ,
$$\frac{PA}{BP} = \frac{AQ}{PC} = \frac{PQ}{BC} \quad (1)$$
ಈಗ,
$AQ = RS = OS - OR = x - x_1$
$PC = ST = OT - OS = x_2 - x$
$PQ = PS - QS = PS - AR = y - y_1$
$BC = BT - CT = BT - PS = y_2 - y$

ಈ ಬೆಲೆಗಳನ್ನು (1) ರಲ್ಲಿ ಆದೇಶಿಸಿದಾಗ,
$$\frac{m_1}{m_2} = \frac{x - x_1}{x_2 - x} \text{ ಮತ್ತು } \frac{m_1}{m_2} = \frac{y - y_1}{y_2 - y}$$
$$\frac{m_1}{m_2} = \frac{x - x_1}{x_2 - x} \implies x = \frac{m_1x_2 + m_2x_1}{m_1 + m_2} \text{ ಎಂದಾಗುತ್ತದೆ.}$$
ಅದೇ ರೀತಿ,
$$\frac{m_1}{m_2} = \frac{y - y_1}{y_2 - y} \implies y = \frac{m_1y_2 + m_2y_1}{m_1 + m_2} \text{ ಎಂದಾಗುತ್ತದೆ.}$$
ಆದ್ದರಿಂದ A $(x_1, y_1)$ ಮತ್ತು B $(x_2, y_2)$ ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು ಆಂತರಿಕವಾಗಿ, $m_1 : m_2$ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ P (x, y) ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು.
$$ \left( \frac{m_1x_2 + m_2x_1}{m_1 + m_2}, \frac{m_1y_2 + m_2y_1}{m_1 + m_2} \right) \quad (2) $$
ಇದನ್ನು ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರ ಎನ್ನುತ್ತೇವೆ.

A, P ಮತ್ತು B ಗಳಿಂದ y – ಅಕ್ಷಕ್ಕೆ ಲಂಬಗಳನ್ನು ಎಳೆದು ಆ ಬಳಿಕ ಈ ಮೇಲಿನಂತೆ ಮುಂದುವರಿದರೆ ಕೂಡಾ ಸೂತ್ರವನ್ನು ವ್ಯುತ್ಪತ್ತಿ ಮಾಡಬಹುದು.

P ಯು AB ಯನ್ನು ವಿಭಾಗಿಸುವ ಅನುಪಾತ k: 1 ಆದರೆ P ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು
$$ \left( \frac{kx_2 + x_1}{k+1}, \frac{ky_2 + y_1}{k+1} \right) $$

ವಿಶೇಷ ಪ್ರಕರಣ: ಒಂದು ರೇಖಾಖಂಡದ ಮಧ್ಯಬಿಂದುವು ಆ ರೇಖಾಖಂಡವನ್ನು 1 : 1 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆ, ಆದ್ದರಿಂದ A $(x_1, y_1)$ ಮತ್ತು B $(x_2, y_2)$ ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡದ ಮಧ್ಯಬಿಂದು Pಯ ನಿರ್ದೇಶಾಂಕಗಳು,
$$ \left( \frac{1.x_2 + 1.x_1}{1+1}, \frac{1.y_2 + 1.y_1}{1+1} \right) = \left( \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2} \right) $$

ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು ಆಧರಿಸಿ ಕೆಲವೊಂದು ಉದಾಹರಣೆಗಳನ್ನು ಬಿಡಿಸೋಣ.

--- EXAMPLE START ---
Source: Class 10, Example 6 (page 117)
Problem: (4, -3) ಮತ್ತು (8, 5) ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು ಆಂತರಿಕವಾಗಿ 3 : 1 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
Solution:
ಪರಿಹಾರ: P (x, y)ಯು ಅಪೇಕ್ಷಿತ ಬಿಂದುವಾಗಿರಲಿ. ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು ಉಪಯೋಗಿಸಿದಾಗ
$$x = \frac{3(8) + 1(4)}{3+1} = \frac{24+4}{4} = \frac{28}{4} = 7, \quad y = \frac{3(5) + 1(-3)}{3+1} = \frac{15-3}{4} = \frac{12}{4} = 3 \text{ ಎಂದಾಗುತ್ತದೆ.}$$
ಆದ್ದರಿಂದ ಅಪೇಕ್ಷಿತ ಬಿಂದು (7, 3).
--- EXAMPLE END ---

--- EXAMPLE START ---
Source: Class 10, Example 7 (page 117)
Problem: A (-6, 10) ಮತ್ತು B (3, -8) ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು (-4, 6) ಬಿಂದುವು ಯಾವ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆ?
Solution:
ಪರಿಹಾರ: (-4, 6) AB ಯನ್ನು ಆಂತರಿಕವಾಗಿ $m_1 : m_2$ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸಲಿ. ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು ಉಪಯೋಗಿಸಿದಾಗ
$$(-4, 6) = \left( \frac{3m_1 - 6m_2}{m_1 + m_2}, \frac{-8m_1 + 10m_2}{m_1 + m_2} \right) \quad (1)$$
(x, y) = (a, b) ಆದರೆ, x= a, y = b ಎಂಬುದನ್ನು ನೆನಪಿಸಿಕೊಳ್ಳಿ
ಆದ್ದರಿಂದ,
$$-4 = \frac{3m_1 - 6m_2}{m_1 + m_2}$$
ಈಗ,
$$-4(m_1 + m_2) = 3m_1 - 6m_2$$
$$-4m_1 - 4m_2 = 3m_1 - 6m_2$$
ಅಂದರೆ, $2m_2 = 7m_1$
ಅಂದರೆ, $m_1 : m_2 = 2 : 7$
ಅನುಪಾತವು y - ನಿರ್ದೇಶಾಂಕಕ್ಕೂ ಹೊಂದಾಣಿಕೆಯಾಗುತ್ತದೆ ಎಂಬುದನ್ನು ನೀವು ಪರಿಶೀಲಿಸ ಬೇಕಾಗುತ್ತದೆ.
ಈಗ
$$\frac{-8m_1 + 10m_2}{m_1 + m_2} = \frac{-8(\frac{m_1}{m_2}) + 10}{(\frac{m_1}{m_2}) + 1} \text{ (ಎಲ್ಲಾ ಪದಗಳಿಗೂ } m_2 \text{ನಿಂದ ಭಾಗಿಸಿದಾಗ)}$$
$$= \frac{-8(\frac{2}{7}) + 10}{\frac{2}{7} + 1} = \frac{\frac{-16+70}{7}}{\frac{2+7}{7}} = \frac{54}{9} = 6$$
ಆದ್ದರಿಂದ (-4, 6) ಬಿಂದುವು. A (-6, 10) ಮತ್ತು B (3, -8) ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು 2 : 7 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆ.

ಪರ್ಯಾಯವಾಗಿ: $m_1 : m_2$ ಅನುಪಾತವನ್ನು $\frac{m_1}{m_2} : 1$ ಅಥವಾ k : 1 ಎಂದು ಕೂಡಾ ಬರೆಯಬಹುದು.
(-4, 6) AB ಯನ್ನು ಆಂತರಿಕವಾಗಿ k: 1 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆ ಎಂದುಕೊಳ್ಳೋಣ. ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು ಉಪಯೋಗಿಸಿದಾಗ
$$(-4, 6) = \left( \frac{k(3) + 1(-6)}{k+1}, \frac{k(-8) + 1(10)}{k+1} \right) = \left( \frac{3k - 6}{k+1}, \frac{-8k + 10}{k+1} \right) \quad (2)$$
ಆದ್ದರಿಂದ,
$$-4 = \frac{3k - 6}{k+1}$$
ಅಂದರೆ, $-4k - 4 = 3k - 6$
ಅಂದರೆ, $2 = 7k \implies k = \frac{2}{7}$
ಅಂದರೆ, k : 1 = 2 : 7. y ನಿರ್ದೇಶಾಂಕಕ್ಕೆ ಕೂಡಾ ನೀವಿದನ್ನು ಪರಿಶೀಲಿಸಬಹುದು.
ಆದ್ದರಿಂದ (-4, 6) ಬಿಂದುವು, A(-6, 10) ಮತ್ತು B(3, -8) ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು 2 : 7 ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುತ್ತದೆ.
--- EXAMPLE END ---

ಸೂಚನೆ: ಈ ಅನುಪಾತವನ್ನು ಮತ್ತೊಂದು ವಿಧಾನದಲ್ಲಿ ಕಂಡುಹಿಡಿಯಲು, ನೀವು PA ಮತ್ತು PB ಗಳ ದೂರಗಳನ್ನು ಲೆಕ್ಕ ಹಾಕಬೇಕು ಮತ್ತು ಅವುಗಳ ಅನುಪಾತವನ್ನು ಕಂಡುಹಿಡಿಯಬೇಕು. ಆದರೆ ಇಲ್ಲಿ A, P ಮತ್ತು B ಗಳು ಸರಳರೇಖಾಗತ ಎಂದು ತಿಳಿದಿರಬೇಕಾಗುತ್ತದೆ.

--- EXAMPLE START ---
Source: Class 10, Example 8 (page 119)
Problem: A (2, -2) ಮತ್ತು B (-7, 4) ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡದ ತ್ರೈಭಾಜಕ ಬಿಂದುಗಳ (ಅಂದರೆ, ಮೂರು ಸಮ ಭಾಗಗಳನ್ನು ಮಾಡುವ ಬಿಂದುಗಳು) ನಿರ್ದೇಶಾಂಕಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.

$$ \left( \frac{1(-7) + 2(2)}{1+2}, \frac{1(4) + 2(-2)}{1+2} \right) = \left( \frac{-7+4}{3}, \frac{4-4}{3} \right) = \left( \frac{-3}{3}, \frac{0}{3} \right) \text{ ಅಂದರೆ, } (-1, 0). $$

## 7.4 ಸಾರಾಂಶ
ಈ ಅಧ್ಯಾಯದಲ್ಲಿ, ನೀವು ಕೆಳಗಿನ ಅಂಶಗಳನ್ನು ಕಲಿತಿರುವಿರಿ.
1. P$(x_1, y_1)$ ಮತ್ತು Q $(x_2, y_2)$ ಗಳ ನಡುವಿನ ದೂರವು $\sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$
2. P(x, y) ಬಿಂದುವಿನಿಂದ ಮೂಲಬಿಂದುವಿಗಿರುವ ದೂರವು $\sqrt{x^2 + y^2}$
3. A$(x_1, y_1)$ ಮತ್ತು B $(x_2, y_2)$ ಬಿಂದುಗಳನ್ನು ಆಂತರಿಕವಾಗಿ $m_1 : m_2$ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ P (x, y) ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು
$$ \left( \frac{m_1x_2 + m_2x_1}{m_1 + m_2}, \frac{m_1y_2 + m_2y_1}{m_1 + m_2} \right) $$
4. P $(x_1, y_1)$ ಮತ್ತು Q $(x_2, y_2)$ ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡದ ಮಧ್ಯಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು $(\frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2})$

--- NOTE ---
**ಓದುಗರಿಗೆ ಸೂಚನೆ**

ವಿಭಾಗ 7.3 ರಲ್ಲಿ A$(x_1, y_1)$ ಮತ್ತು B $(x_2, y_2)$ ಎಂಬ ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು ಆಂತರಿಕವಾಗಿ $m_1 : m_2$ ಎಂಬ ಅನುಪಾತದಲ್ಲಿ ವಿಭಾಗಿಸುವ P (x, y) ಎಂಬ ಬಿಂದುವಿನ ನಿರ್ದೇಶಾಂಕಗಳು ಈ ಕೆಳಗಿನಂತಿವೆ:
$$ x = \frac{m_1x_2 + m_2x_1}{m_1 + m_2}, \quad y = \frac{m_1y_2 + m_2y_1}{m_1 + m_2} $$
ಎಂಬುದಾಗಿ ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರದಲ್ಲಿ ಚರ್ಚಿಸಿದ್ದೇವೆ.
ಇಲ್ಲಿ PA : PB = $m_1 : m_2$ ಎಂಬುದನ್ನು ಗಮನಿಸಿ.
ಆದಾಗ್ಯೂ, P ಯು A ಮತ್ತು B ಗಳ ನಡುವೆ ಇಲ್ಲದೆ AB ರೇಖಾಖಂಡದ ಹೊರಗೆ PA : PB = $m_1 : m_2$ ಆಗುವಂತೆ AB ಸರಳ ರೇಖೆಯ ಮೇಲೆ ಇದ್ದರೆ, A ಮತ್ತು B ಬಿಂದುಗಳನ್ನು ಸೇರಿಸುವ ರೇಖಾಖಂಡವನ್ನು P ಯು ಬಾಹ್ಯವಾಗಿ ವಿಭಾಗಿಸುತ್ತದೆ ಎಂದು ನಾವು ಹೇಳುತ್ತೇವೆ. ಇಂತಹ ಪ್ರಕರಣಕ್ಕೆ ಭಾಗ ಪ್ರಮಾಣ ಸೂತ್ರವನ್ನು ನೀವು ಉನ್ನತ ತರಗತಿಗಳಲ್ಲಿ ಕಲಿಯುತ್ತೀರಿ.
