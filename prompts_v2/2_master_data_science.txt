ಈ ಹಂತ ಬಹಳ ಮುಖ್ಯ — PDF ನಲ್ಲಿ ಇರುವ ಆದರೆ extracted text ನಲ್ಲಿ
ಕಳೆದುಹೋಗಿರುವ ವಿಷಯವನ್ನು ಸೇರಿಸಬೇಕು:

✔ ಯಾವುದಾದರೂ definition, formula, diagram ಕಳೆದಿದ್ದರೆ → ಸೇರಿಸಿ
✔ ಚಿತ್ರ/ರೇಖಾಚಿತ್ರ ವಿವರಣೆ text ನಲ್ಲಿ ಕಡಿಮೆಯಾಗಿದ್ದರೆ → PDF ನಿಂದ ಪೂರ್ಣ ವಿವರಣೆ ಸೇರಿಸಿ
✔ Table / flowchart ಕಳೆದಿದ್ದರೆ → PDF ನಿಂದ ಪುನಃ ಸೃಷ್ಟಿಸಿ
✔ Chemical equations, scientific data → PDF ನಿಂದ ಪರಿಶೀಲಿಸಿ

CHAPTER-WISE CONTENT:
Input is organized by chapter. Handle this by:
* ಸರಳದಿಂದ ಕಠಿಣಕ್ಕೆ ಕ್ರಮ ಇಟ್ಟುಕೊಳ್ಳಬೇಕು (foundational → advanced)
* ಒಂದೇ ಕಲ್ಪನೆಯ ಪುನರಾವರ್ತನೆ ತಪ್ಪಿಸಬೇಕು
* Repeated definitions, laws → ONE consolidated explanation

CONTENT FILTERING:
ಕೆಳಗಿನವನ್ನು IGNORE ಮಾಡಬೇಕು:
✗ "ಪಾಠದ ಪರಿಚಯ" ಭಾಗ
✗ "Learning Outcomes / ಸಾಮರ್ಥ್ಯಗಳು"
✗ Trivial "ತಿಳಿದಿರಲಿ / Did You Know" boxes (unless scientifically important)

ಕೇವಲ ಇವನ್ನು ತೆಗೆದುಕೊಳ್ಳಬೇಕು:
✔ ಎಲ್ಲಾ Definitions (ವ್ಯಾಖ್ಯೆಗಳು)
✔ ಎಲ್ಲಾ Scientific Laws and Principles (ನಿಯಮಗಳು ಮತ್ತು ತತ್ವಗಳು)
✔ ಎಲ್ಲಾ Formulas/Equations — with derivation if textbook shows it
✔ ಎಲ್ಲಾ Chemical Equations (balanced, with states)
✔ ಎಲ್ಲಾ Properties and Classifications
✔ ಎಲ್ಲಾ Worked Examples — COMPLETE step-by-step
✔ ಎಲ್ಲಾ Experimental Activities that teach a concept or derive a result
✔ ಎಲ್ಲಾ Diagrams/Figures — detailed text description
✔ ಎಲ್ಲಾ Special Cases, Exceptions, Boundary Conditions
✔ Common mistakes students make

* basic       → direct formula substitution, 1–2 steps
* intermediate → requires choosing the right formula, 3–4 steps
