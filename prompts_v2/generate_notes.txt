   - Each theorem in a card:
     • Theorem name
     • Statement in a quote-style block
     • Proof steps (if available) in a collapsible/expandable section
     • Converse (if any) as a secondary note
     • Common error in a small warning tag

E. PROPERTIES
   - Clean numbered list with brief explanations

F. WORKED EXAMPLES

     • Solution as numbered steps, each with:

       - LaTeX calculation (display mode)
       - Justification in small muted text
     • Final answer in a green success box
     • "⚠️ ಎಚ್ಚರ:" common trap in an orange warning box (if present)

     • GPSTR weightage badge
